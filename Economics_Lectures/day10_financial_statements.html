CORE CONCEPTS

재무제표 읽기

CONCEPT 01
재무제표 3가지 표의 논리적 흐름

기업의 재무상황은 세 개의 표로 완전히 설명됩니다. 각 표는 다른 각도에서 기업의 건강도를 보여줍니다.

손익계산서 (Income Statement)
= 일정 기간(분기/연도)의 수입과 지출
수입(Revenue) - 지출(Expense) = 순이익(Net Income)
// 기업이 "얼마나 벌었는가"를 측정

재무상태표 (Balance Sheet)
= 특정 시점(분기말/연도말)의 자산, 부채, 자본
자산(Asset) = 부채(Liability) + 자본(Equity)
// 기업의 "현재 자산 상태"를 스냅샷으로 표시

현금흐름표 (Cash Flow Statement)
= 현금의 실제 움직임 (회계와 다름!)
영업CF + 투자CF + 금융CF = 현금 순변화
// "실제로 현금이 남았는가"를 측정
// 중요: 이익이 많아도 현금이 부족할 수 있음!
CONCEPT 02
삼성전자 2023년 재무분석

삼성전자의 실제 재무제표 데이터(2023년, IFRS 기준)입니다. 메모리 칩 시장 불황이 반영되어 있습니다.

손익계산서 (2023년)
항목금액 (조원)전년대비
매출액 (Revenue)74.0-18%
영업이익 (Operating Income)6.5-88% 급락!
영업이익율8.8%-
순이익 (Net Income)4.8-94%
EPS (주당순이익)2,935원-93%
재무상태표 (2023년말)
항목금액 (조원)
총 자산 (Total Assets)402.3
├─ 현금 및 현금성자산37.2
├─ 단기투자증권12.5
├─ 고정자산 (공장, 장비)172.1
총 부채 (Total Liabilities)152.4
├─ 단기차입금18.5
└─ 장기차입금25.3
자본금 (Shareholders' Equity)249.9

분석: 부채비율 = 부채/자본 = 152.4/249.9 = 61% (건강한 수준). 다만 영업이익 급락으로 부채 상환 능력 악화 중.

CONCEPT 03
현금흐름 분석: 이익과 현금의 괴리

회계상 '이익'과 실제 '현금'은 다릅니다. 현금흐름이 음수면 아무리 이익을 내도 위험합니다.

삼성전자 2023년 현금흐름:
영업현금흐름 (Operating CF)
= 순이익(4.8조) - 감가상각(12조) + 기타조정
= +18.5조원 (양호)

투자현금흐름 (Investing CF)
= -공장 신축비 -12조, -장비 구입비 -8조
= -28.3조원 (과도한 투자)

금융현금흐름 (Financing CF)
= 배당금 지급 -8.5조 + 차입금 증가 +3.2조
= -5.3조원

// 순 현금 흐름 = +18.5 - 28.3 - 5.3 = -15.1조원
// 즉, 2023년 삼성은 보유 현금을 15조 소소비 (미래 수익성 회복 베팅)

의미: 영업이익이 엄청 줄었지만(4.8조) 감가상각 등의 비현금 비용 때문에 실제 현금흐름은 양호(18.5조). 따라서 단기적 현금 위기는 없으나, 투자비 과다(28조) + 배당금(8조)으로 현금을 소진 중입니다.

심화 학습

감가상각이 현금흐름을 늘리는 이유

감가상각의 역설: 회계상 '비용'이지만, 실제 현금은 나가지 않습니다. 따라서 순이익에 다시 더해서 계산합니다.

예시: 공장 건설
Year 0: 공장 짓기 → 현금 100억 지출
Year 1~10: 감가상각 10억/년 × 10년

회계상 이익:
= 수입 - 감가상각 - 기타지출
= 150 - 10 - 20 = 120억 (이익)

실제 현금흐름:
= 수입 - (감가상각 제외 지출)
= 150 - 20 = 130억 (현금)
// 감가상각 10억은 "종이상의 비용" → 현금은 나가지 않음

삼성의 경우: 2023년 순이익 4.8조인데 감가상각 12조를 더하면 영업현금흐름이 18.5조가 됩니다. 따라서 "순이익이 4조인데 어떻게 산다고?" → 답: 감가상각 덕분.

투자자 관점: 현금흐름이 좋은 기업을 찾으세요. 이익만 크면 함정일 수 있습니다.

부채비율(Debt-to-Equity) 건강 기준선

부채비율 = 부채 ÷ 자본금

삼성 2023: 152.4조 ÷ 249.9조 = 61%

업종별 건강한 부채비율:
제조업 (반도체, 자동차): 50~80%
금융업: 200~300% (정상범위)
유틸리티 (전력, 가스): 60~100%

해석:
부채비율 50% 이하 → 매우 건전
부채비율 50~100% → 보통 (정상)
부채비율 100% 이상 → 위험 신호

삼성은 61%로 건강한 수준이지만, 2023년처럼 이익이 급락하면 부채 상환 능력이 악화됩니다. 따라서 부채비율도 중요하지만 이자 보상 능력(Interest Coverage Ratio)도 봐야 합니다.

APPLY TO MY ASSETS

내 자산에 직접 적용

📑 내 투자기업 재무분석 체크리스트

  • 영업이익 추이: 최근 3년 영업이익이 증가하고 있는가? 삼성처럼 급락하지는 않는가?
  • 영업이익율: 동종업계 평균보다 높은가? (반도체 10~15%, 제조업 평균 8~10%)
  • 부채비율: 50~100% 범위 내인가? 100%를 넘으면 위험신호
  • 영업현금흐름: 순이익이 아닌 "영업현금흐름"이 지속적으로 양수인가?
  • 현금 소진율: (투자CF + 금융CF) 크기가 영업CF 크기의 50% 이하인가?
  • 배당금: 배당금 지급이 현금흐름 범위 내에서 이루어지고 있는가?

⭐ 핵심 요점:

1. "순이익"만 보지 말고 "영업현금흐름"을 봐야 함

2. 부채비율 + 영업현금흐름 = 기업의 진짜 건강도

3. 빠르게 성장하는 기업이라도 현금이 음수면 위험

INVESTOR PSYCHOLOGY

오늘의 투자 심리

행동재무학 편향

회계 지표 맹신 (Accounting Naivety)

뇌과학 인간의 뇌는 "이익"이라는 단어에 긍정 신호를 받습니다. 특히 큰 숫자를 보면 충동적으로 반응합니다. 하지만 현금흐름이 음수면 아무리 이익이 크면 함정입니다.

엔지니어링 데이터 검증: 재무제표를 읽을 때 순이익, 영업이익, 영업현금흐름 세 개를 항상 함께 봅시다. 삼성처럼 순이익 4.8조지만 영업CF 18.5조인 경우, 혹은 그 반대 경우도 있습니다. 수치 간 불일치는 숨은 위험신호입니다.

대응: "좋은 이익이다"라는 판단을 하기 전에 "이 이익이 실제 현금으로 들어왔는가?"를 확인하세요. 특히 빠르게 성장하는 스타트업이나 신생 산업(반도체) 기업일수록 이익과 현금의 괴리가 큽니다.

REVIEW

복습 퀴즈

Q1. 재무제표 중 '특정 시점'의 자산·부채·자본을 나타내는 표는?

  • ① 손익계산서
  • ② 재무상태표 (Balance Sheet)
  • ③ 현금흐름표
  • ④ 주식보상 계산서
정답: ②
손익계산서는 일정 기간(분기/연도), 현금흐름표도 기간을 나타냅니다. 반대로 재무상태표는 '연도말' 같은 특정 시점의 스냅샷입니다.

Q2. 삼성 2023년 순이익 4.8조인데 영업현금흐름이 18.5조인 이유는?

  • ① 회계 오류
  • ② 감가상각 등 비현금 비용을 더했기 때문
  • ③ 과장된 수치
  • ④ 배당금 때문
정답: ②
감가상각, 대손충당금 등은 '회계상 비용'이지만 실제 현금은 나가지 않습니다. 따라서 순이익에 이들을 다시 더하면 더 큰 영업현금흐름이 나옵니다.
