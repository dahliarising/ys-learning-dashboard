CORE CONCEPTS

행동재무학 — 투자 실패의 심리

CONCEPT 01
① 손실 회피(Loss Aversion) & 전망 이론

Kahneman & Tversky의 연구: 손실의 고통 = 이익의 기쁨 × 2.25배

뇌과학 기전:
• 편도체(Amygdala) 활성화 → 공포 신호
• 배측 선조체(Dorsal Striatum) 억제 → 행동 정지

엔지니어 주의:
"이론적으로는 분산이 최적이지만, 손실하는 것이 두려워 현금 비중 과다"
→ 과거 시계 엔지니어들도 같은 편향(완벽 추구 = 행동 지연)

극복 규칙:
Rule 1: 월부 1,500만 중 "정해진 금액"만 투자 (감정 제외)
Rule 2: 포트폴리오 전체 손실률 계산 (개별 손실 아님)
Rule 3: 6개월마다만 확인 (하루 변동성 무시)
CONCEPT 02
② 확증 편향(Confirmation Bias) & 거울 뉴론

자신의 신념을 확인하는 정보만 수집하는 뇌의 기본 설정입니다.

뇌과학 기전:
• 거울 뉴론(Mirror Neuron) 활성화
• "나와 같은 사람이 성공했다" = 자기 검증
• 전두엽(Prefrontal Cortex) 억제 → 비판적 생각 중단

엔지니어 주의:
"내 분석이 옳다" → 반대 의견 무시 → "모델 과신(Overconfidence)"
예: 반도체 장기 낙관론만 강조, 사이클 변화 신호 무시

극복 규칙:
Rule 1: "내 주장의 반대 증거" 3가지 먼저 찾기
Rule 2: 분기마다 "가정 재검토" 세션
Rule 3: 다양한 관점(약세론, 중립론) 의도적 학습
CONCEPT 03
③ 처분 효과(Disposition Effect) & 손절 회피

손실 주식은 팔지 않고, 수익 주식만 빨리 파는 현상입니다.

뇌과학 기전:
• 전전두엽(Ventromedial PFC) 과도 활성
• 손실 인정 = 자존감 상처 → 회피
• 코르티솔 상승 → 스트레스 호르몬

엔지니어 주의:
"이미 투자한 비용은 돌아올 것"(Sunk Cost Fallacy)
→ 손실 종목에 계속 추가 투자

극복 규칙:
Rule 1: 손절 기준 사전 설정 (예: -15%, 분기마다 재평가)
Rule 2: 수익 종목과 손실 종목 같은 프로세스 사용
Rule 3: 종목 재평가 시 "지금 새로 사겠는가?" 질문
CONCEPT 04
④ 과잉 확신(Overconfidence) & 백테스트 최적화

자신의 예측 능력을 과평가하고, 과거 데이터로 과도하게 최적화합니다.

뇌과학 기전:
• 도파민(Dopamine) 상승 = 자신감 착각
• 성공 기억은 강화, 실패 기억은 약화 (선택적 기억)
• 과의측좌핵(Ventral Striatum) 활성화

엔지니어 주의 (가장 위험):
"백테스트 수익률 연 25%"
→ 2020~2022년 데이터만 사용 (낙우편향)
→ 실제 운용 시 수익 3~5% (충격)

극복 규칙:
Rule 1: 백테스트 기간 최소 10년 이상 (사이클 포함)
Rule 2: 수익률 가정 = (수익 + 손실) / 2 (보수적 추정)
Rule 3: 월 1회 "예측 오류 일지" 작성

DEEP DIVE

심화 학습

엔지니어의 특수한 투자 심리 — "분석 마비(Analysis Paralysis)"와 "모델 과신"

자동차 OEM 엔지니어의 경우: 정교한 분석 능력은 투자에 독이 될 수 있습니다.

  • • 분석 마비: "모든 경우의 수를 검토하겠다" → 투자 결정 지연 → 기회 상실
  • • 모델 과신: 완성도 높은 분석 모델 = 현실 예측력이라는 착각
  • • 백테스트 함정: 과거 데이터 최적화 ≠ 미래 성과 (overfitting)
  • • 확증편향: 반도체 장기 낙관론 → 약세 신호 무시

극복 전략:

1. "80% 확신으로 투자" — 100% 확신은 불가능 (베르나르드 쇼)
2. 백테스트 기간 3배 확대 (5년 → 15년)
3. 월 1회 "자신이 틀린 기억" 일지 작성
4. 분석 시간 제한: 최대 4주 (그 이상은 수익 증가 없음)

REVIEW

복습 퀴즈

Q1. 편도체가 과도하게 활성화되었을 때의 투자 행동은?

  • ① 높은 수익 추구
  • ② 손실에 대한 공포로 인한 행동 정지
  • ③ 분석적 결정
  • ④ 확신 있는 매매
정답: ②
편도체는 공포 중추입니다. 시장 하락 시 편도체 활성화 → 매도 압박 → 손실 실현.

Q2. 도파민 상승이 투자에 미치는 영향은?

  • ① 신중한 판단
  • ② 착각된 자신감으로 과잉거래
  • ③ 손실회피
  • ④ 분산투자
정답: ②
도파민은 '보상' 신경전달물질입니다. 과도한 상승은 자신감 착각 → 과잉확신 → 백테스트 최적화 함정.

Q3. 처분 효과를 극복하는 방법은?

  • ① 손실 종목에 계속 투자
  • ② 수익 종목만 빨리 팔기
  • ③ 손절 기준 사전 설정 & 감정 배제
  • ④ 분기마다 전체 교체
정답: ③
처분효과(손절 회피)는 자존감 상처를 피하려는 심리. 사전에 손절 기준(-15%)을 설정하면 감정 개입을 줄일 수 있습니다.

Q4. 엔지니어가 특히 주의해야 할 편향은?

  • ① 손실회피 (공통)
  • ② 과잉확신 & 확증편향 (엔지니어 특화)
  • ③ 처분효과 (공통)
  • ④ 모두 동일
정답: ②
엔지니어는 분석 능력이 높아 '모델 과신'과 '확증편향' 조합이 가장 위험합니다. 반도체 장기 낙관론 같은 '신념'을 강하게 가지기 쉽습니다.
