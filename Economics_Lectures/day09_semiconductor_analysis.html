CORE CONCEPTS

반도체 산업 구조 분석

CONCEPT 01
반도체 가치사슬 3단계

반도체 산업은 설계(팹리스)→제조(파운드리)→후공정(패키징·테스트)의 세 단계로 나뉩니다. 각 단계의 수익성과 기술 진입장벽이 다릅니다.

가치사슬 구조:
[팹리스] 설계, 개발, 영업 수익율 15~25% (높음) | 예: NVIDIA, AMD, Qualcomm
↓
[파운드리] 실제 제조 (극자외선 미세공정) 수익율 5~15% (중간) | 예: TSMC, Samsung, Intel
↓
[후공정] 패키징, 테스트, 품질검사 수익율 3~8% (낮음) | 예: ASE, 삼성 HY-X, SK하이닉스

// 삼성·SK하이닉스 = 파운드리 + 메모리(DRAM) 전문
// 메모리 칩 제조는 파운드리보다 수익율 더 낮음 (경쟁 과포화)

심화 학습

HBM 기술과 3D 적층 구조의 물리학

HBM (High Bandwidth Memory)의 핵심: 메모리 칩을 수직으로 여러 층 적층(3D stacking)하면 데이터 경로가 짧아져 속도가 극도로 빨라집니다.

HBM 구조 (SK하이닉스 HBM3 기준):
8개 메모리 칩을 수직으로 적층
+ 각 칩 사이 Through-Silicon-Via (TSV) 연결
= 칩 간 통신 거리 1~2 nm (기존 100배 단축)
// 물리적 거리 단축 → 전파 지연 감소 → 900GB/s 대역폭 달성

제조 난제: 3D 적층 시 칩 정렬 오차가 1마이크로미터만 넘어도 불량입니다. 따라서 극저온(-196°C) 진공 환경에서 정밀도 ±0.1마이크로미터의 장비로 제조해야 합니다. 이것이 삼성이 따라잡기 어려운 이유입니다.

SK하이닉스의 경쟁우위: 2018년부터 HBM 연구 → 2023년 HBM3 양산 → NVIDIA와 장기계약 → 공급 독점에 가까움

메모리 칩 수익률이 낮은 이유

DRAM은 '상품화된(commodity)' 제품으로, 기술 진입장벽은 높지만 일단 생산되면 가격 경쟁이 치열합니다.

DRAM 수익율이 낮은 이유:
1. 공급과잉: 중국(紫光), 한국(삼성·SK·마이크론) 동시 생산
2. 제품 동일성: 모두 같은 규격(DDR5, DDR6…) → 가격만 비교
3. 설계 난제 없음: 기술 공개된 지 오래 → 주로 제조비용 경쟁
// 팹리스(설계) 이윤 20% vs 메모리(제조) 이윤 5% = 4배 차이

반대로 HBM이 수익성이 높은 이유:

  • - 기술 차별화 (SK > 삼성 > 마이크론 순)
  • - 공급 부족 → 프리미엄 가격 가능 (단가 3배~5배)
  • - NVIDIA, AMD와 장기계약 → 안정적 매출
  • - AI 수요 지속 → 가격 상승 기대

뇌과학 새로운 기술(AI, 반도체)에 대해 뇌의 전측뇌피질이 지나친 낙관을 생성합니다. "AI 혁명이 온다"는 담론이 강할 때 투자자들은 실제 수익성보다 성장률에 집중하는 경향이 있습니다.

엔지니어링 데이터 기반 판단: SK하이닉스의 HBM 수익율을 현재 5% → 2025년 18% 전망하는 것이 합리적인지, NVIDIA와의 계약 규모가 실제로 얼마나 되는지 재무제표로 검증하세요. "AI가 핫하니까"라는 감정 투자를 피하세요.

대응: 산업 구조 분석 후 "어느 기업이 실제로 이윤을 뽑는가?"를 물어보세요. AI 투자 수혜도 2~3년 후입니다. 지금 과도하게 주가에 반영된 부분은 조정될 가능성이 높습니다.

REVIEW

복습 퀴즈

Q1. 반도체 가치사슬에서 수익율이 가장 높은 단계는?

  • ① 파운드리 (TSMC)
  • ② 팹리스 (NVIDIA)
  • ③ 후공정 (ASE)
  • ④ 메모리 (삼성·SK)
정답: ②
팹리스(설계)는 기술 진입장벽이 높아 수익율 20~25%. 파운드리는 12%, 후공정은 6%, 메모리는 5% 정도. 따라서 설계가 가장 수익성이 높습니다.

Q2. HBM(High Bandwidth Memory)이 일반 DRAM과 비교해 대역폭이 900GB/s인 이유는?

  • ① 더 빠른 클락 속도
  • ② 3D 적층으로 칩 간 거리 단축
  • ③ 더 많은 칩 사용
  • ④ 더 높은 전압
정답: ②
HBM은 8개의 메모리 칩을 수직으로 적층하고 TSV(Through-Silicon-Via)로 연결하여 칩 간 거리를 나노미터 단위로 단축. 전파 지연이 극도로 줄어들어 900GB/s 달성.
