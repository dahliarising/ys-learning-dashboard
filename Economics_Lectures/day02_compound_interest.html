CORE CONCEPTS

복리의 수식

CONCEPT 01
복리 공식 완전 정복

복리(Compound Interest)는 "이자에 이자가 붙는" 현상입니다. 투자에서 가장 중요한 공식이므로, 단계별로 이해해야 합니다.

① 기초: 1년 후 금액
FV = PV × (1 + r)
예: 1억 @ 8% → 1억 × 1.08 = 1.08억

② 5년 후 금액
FV = PV × (1 + r)^5
예: 1억 @ 8% × 5년 → 1억 × 1.08^5
→ 1억 × 1.4693 = 1.4693억

③ n년 후 금액 (일반식)
FV = PV × (1 + r)^n
FV: 미래 가치 (Future Value)
PV: 현재 가치 (Present Value)
r: 연 수익률 (8% = 0.08)
n: 년수
CONCEPT 02
Rule of 72 — 빠른 계산법

자산이 2배가 되는 데 몇 년이 걸릴까? 복잡한 계산 없이 대략 추정하는 법입니다.

Rule of 72:
2배 소요 시간(년) ≈ 72 ÷ 연 수익률(%)

예시:
8% 수익 → 72 ÷ 8 = 9년 (정확: 9.0년)
6% 수익 → 72 ÷ 6 = 12년 (정확: 11.9년)
4% 수익 → 72 ÷ 4 = 18년 (정확: 17.7년)
10% 수익 → 72 ÷ 10 = 7.2년 (정확: 7.3년)

// 놀라운 정확도! 금융 계획에서 빠른 의사결정에 유용
// 예: 3.28억이 2배가 되려면? 72 ÷ 8 = 약 9년
CONCEPT 03
저축 + 복리: 정기 납입의 위력

실제 투자는 한 번에 넣고 끝나는 것이 아니라, 매달 저축하면서 동시에 복리가 작동합니다. 이를 "연금 공식(Annuity)"이라 합니다.

정기 납입 + 복리 (월 300만):
FV = PMT × [((1+r)^n - 1) / r]
PMT: 정기 납입액
r: 월 수익률 (8% ÷ 12 ≈ 0.67%)
n: 납입 월수 (10년 = 120개월)

실제 예시 (월 300만, 10년, 연 8%):
r_월 = 0.08 ÷ 12 = 0.00667
FV = 300만 × [((1.00667)^120 - 1) / 0.00667]
FV = 300만 × 155.9
FV = 약 4.677억

// 순저축: 300만 × 120 = 3.6억
// 복리 이득: 4.677억 - 3.6억 = 1.077억 (30% 이상!)
CONCEPT 04
시간 가치의 역설: 현재의 1,000만 vs 10년 뒤의 1,000만

같은 돈도 시간에 따라 가치가 다릅니다. 현재의 1,000만이 10년 뒤 1,000만보다 훨씬 가치 있는 이유는 "지금 투자하면 복리가 작동"하기 때문입니다.

시점투자 금액10년 후 가치 (8%)차이
지금 (Day 1)1,000만1,000만 × 2.16 = 2,158만+1,158만
1년 뒤1,000만1,000만 × 1.999 = 1,999만+999만
5년 뒤1,000만1,000만 × 1.469 = 1,469만+469만
10년 뒤1,000만1,000만 × 1 = 1,000만±0

즉, 1년을 기다리면 약 159만원의 가치를 잃는다는 뜻입니다. 이것이 "지금 시작이 중요"한 이유입니다.

CONCEPT 05
복리와 인플레이션: 실질 수익률

8% 복리도 좋지만, 인플레이션을 고려해야 "실제 자산 증가"를 알 수 있습니다. 명목 수익률과 실질 수익률의 차이입니다.

실질 수익률 계산:
실질 수익률 ≈ 명목 수익률 − 인플레이션율
(정확: (1+명목) ÷ (1+인플레이션) − 1)

예시:
명목 8% 수익 − 2% 인플레이션
= 약 5.88% 실질 수익

// 현재 물가로 계산하면 "실제" 자산 증가는 8%보다 작음
// 해외 투자(달러/유로)는 환율 인플레이션도 고려 필요

심화 학습

연금 공식 완전 정복: 실제 투자 시뮬레이션

월 300만을 10년간 저축하면서 동시에 연 8% 복리가 작동할 때, 최종 자산이 정확히 얼마가 되는지 계산하는 방법입니다. 이것이 실제 투자의 현실입니다.

연금 현가 공식 (Future Value of Annuity):
FV = PMT × [((1+r)^n - 1) / r]
PMT = 월 저축액 = 300만
r = 월 수익률 = 8% ÷ 12 = 0.00667
n = 납입 개월 수 = 10년 × 12 = 120개월

계산 과정:
(1 + 0.00667)^120 = 1.08^10 = 2.1589
(2.1589 - 1) / 0.00667 = 1.1589 / 0.00667 = 173.6
FV = 300만 × 173.6 = 5.208억

// 순저축: 300만 × 120개월 = 3.6억
// 복리 이득: 5.208억 - 3.6억 = 1.608억 (45% 이상!)

이것이 복리의 위력입니다. 10년 만에 투자액의 45% 이상이 "공짜"로 증가했습니다. 30년이면 그 배수는 훨씬 커집니다.

Rule of 72의 수학적 증명

Rule of 72가 왜 정확할까요? 자연로그(ln)의 성질 때문입니다.

자산이 2배가 되는 조건:
2 = (1 + r)^n
양변에 ln 취하면: ln(2) = n × ln(1+r)
n = ln(2) / ln(1+r)

r이 작을 때 근사:
ln(1+r) ≈ r (r이 0.1 이하일 때)
따라서 n ≈ ln(2) / r = 0.693 / r

백분율로 표현하면:
n ≈ 69.3 / 수익률(%) ≈ 72 / 수익률(%)
// 69.3을 72로 올림하면 계산이 더 쉬워짐

따라서 Rule of 72는 단순한 추정이 아니라, 수학적으로 증명된 정확한 공식입니다!

1년 지연의 기회 비용: 직관적 이해

현재의 1,000만을 "지금 투자"하는 것과 "1년 뒤 투자"하는 것의 차이는 얼마일까요?

지금 투자: 1,000만 (30년, 연 8%)
FV = 1,000만 × 1.08^30
FV = 1,000만 × 10.06 = 1.006억

1년 뒤 투자: 1,000만 (29년, 연 8%)
FV = 1,000만 × 1.08^29
FV = 1,000만 × 9.31 = 9,310만

기회 비용:
1.006억 - 9,310만 = 750만
// 1년을 기다렸을 뿐인데, 최종 자산에서 750만을 영구히 잃음
// 월 저축 300만 기준으로는 약 2.5개월 저축을 포기한 것과 같음

결론: "지금 시작"이 가장 강력한 투자 전략입니다. 수익률 최적화보다 "시간 확보"가 훨씬 중요합니다.

🔧 ENGINEER'S PERSPECTIVE

엔지니어 시스템 사고

엔지니어 사고법 복리 = 지수함수 (Exponential Growth)

재무에서 복리는 공학의 지수함수와 정확히 같습니다. 단리는 y=P(1+rt) 형태의 1차 선형함수로 시간에 따라 일정하게 증가하지만, 복리는 y=P(1+r)^t 형태의 지수함수로 초기에는 느리다가 시간이 지날수록 폭발적으로 증가합니다. 반도체 트랜지스터 집적도(무어의 법칙)도 동일한 지수함수 패턴을 따르며, 이것이 기술 발전이 가속화되는 이유입니다.

- 단리: y = P(1 + rt) → 직선, 초기와 속도 동일 - 복리: y = P(1 + r)^t → 곡선, 시간 t에 따라 기울기 급증 - 무어의 법칙: 트랜지스터 집적도 = (2)^(t/18개월) → 18개월마다 2배 - 재정자유: 자산 = 원금 × (1+수익률)^년수 → 지수함수의 특성상 마지막 5년이 가장 큰 증가

Q4. Rule of 72의 수학적 근거는?

⚠️ 면책 고지
본 콘텐츠는 투자·자산관리 교육 목적으로 제작되었으며, 특정 금융 상품이나 투자 전략에 대한 추천이 아닙니다.
모든 투자 판단과 그에 따른 결과의 책임은 전적으로 본인에게 있습니다. 실제 투자 전 공인 재무설계사와 상담하시기 바랍니다.
수익률은 과거 데이터 기반 가정치이며 미래 수익을 보장하지 않습니다. 세금·규제 정보는 2026년 3월 기준이며 변경될 수 있습니다.
