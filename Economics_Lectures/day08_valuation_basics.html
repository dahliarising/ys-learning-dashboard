CORE CONCEPTS

기업 가치 평가 기초

CONCEPT 01
① PER (주가수익비율): 주가 vs 수익

기업의 순이익에 비해 현재 주가가 몇 배인지 나타냅니다.

공식:
PER = 현재 주가 / 주당순이익(EPS)

삼성전자 (2026년 3월 기준 예시):
주가: 70,000원 / EPS: 3,500원 = PER 20배
// 업계 평균(PER 15배)보다 높음 → 성장성 기대 or 고평가

해석:
• PER < 10배: 저평가 (단, 사업 부진 가능성 주의)
• PER 10~15배: 적정 평가
• PER > 20배: 고평가 (성장 기대 또는 투기)
CONCEPT 02
② PBR (주가순자산비율): 주가 vs 자산

주당순자산에 비해 현재 주가가 몇 배인지 나타냅니다. 자산 가치에 기반한 평가입니다.

공식:
PBR = 현재 주가 / 주당순자산(BPS)

SK하이닉스 (2026년 3월 기준 예시):
주가: 150,000원 / BPS: 120,000원 = PBR 1.25배
// 반도체 업계 PBR 1.0~1.3배 범위 내

해석:
• PBR < 1.0배: 순자산보다 저렴 (기업청산가치 이하)
• PBR 1.0~1.5배: 정상 범위
• PBR > 2.0배: 높은 기대감 (기술, 브랜드)
CONCEPT 03
③ EV/EBITDA: 엔터프라이즈 가치 vs 영업 수익력

기업의 영업 효율성을 순수하게 평가하는 지표입니다. 재무 구조(부채, 세금)의 영향을 배제합니다.

공식:
EV/EBITDA = (시가총액 + 순부채) / EBITDA
EBITDA = 영업이익 + 감가상각비 + 이자 비용

삼성전자 수정 분석:
시가총액: 2,800조 (주가 70,000 × 발행주식 4,000만주)
EBITDA: 140조 (가정)
EV/EBITDA = 2,800 / 140 = 20배
// 글로벌 반도체 기업(TSMC, Intel) 평균 15~18배 비교 → 약간 고평가

해석:
• EV/EBITDA < 10배: 저평가
• EV/EBITDA 10~15배: 적정
• EV/EBITDA > 20배: 고평가 또는 성장 기대
CONCEPT 04
④ DCF (자유현금흐름 할인가치 모형)

기업의 미래 현금 흐름을 현재가치로 할인하여 공정가치를 추정합니다. 가장 엄밀하지만 가정이 중요합니다.

간편 공식 (5년 단순화):
공정가치 = (FCF₁ + FCF₂ + ... + FCF₅) / (1+r)ⁿ + 종가가치 / (1+r)⁵
r = 할인율 (일반적으로 10% 사용)

삼성전자 단순 예측 (10조 연간 FCF 가정):
5년 현가 = 10조 / 1.1 + 10조 / 1.1² + ... + 10조 / 1.1⁵
= 9.1 + 8.3 + 7.5 + 6.8 + 6.2 = 약 38조
종가가치(2030년 이후): 50조 / 1.1⁵ = 약 31조
공정가치 = 38조 + 31조 = 약 69조
현재 시가(70조) vs 공정가치(69조) = 적정 평가
// 주의: FCF 3,000조/연이라는 기업 분석 필수

과학적 착각(Illusion of Precision) & 숫자 미신

PER 20배라는 "정확한 숫자"가 마치 기업의 진정한 가치를 나타낸다고 착각합니다. 하지만 모든 지표는 "과거 데이터 기반 추정"일 뿐입니다.

전전두엽 과신 (숫자 편향) 엔지니어 주의: 수식 의존성

극복 규칙: 지표는 비교 도구일 뿐입니다. PER 20배 < PER 25배라는 상대 비교는 유효하지만, "PER 20배 = 정확한 가치"라고 생각하면 위험합니다. 3가지 지표(PER, PBR, EV/EBITDA) 모두가 저평가를 시사할 때만 매수하세요.

REVIEW

복습 퀴즈

Q1. PER 20배는 무엇을 의미하는가?

  • ① 주가가 순이익의 20배
  • ② 20년 동안 이익 회수
  • ③ 주가가 자산의 20배
  • ④ 배당 수익률 20%
정답: ①
PER = 주가 / EPS이므로, PER 20배는 주가가 연간 순이익의 20배라는 의미입니다. 20년 안에 이익으로 회수한다는 해석은 잘못된 것입니다(실제로는 3~5년).

Q2. SK하이닉스의 PBR 1.25배는 어떤 의미인가?

  • ① 자산가보다 저가
  • ② 자산가보다 25% 비쌈
  • ③ 자산 가치와 동일
  • ④ 분석 불가
정답: ②
PBR 1.25배 = 주가가 순자산의 1.25배. 즉, 자산가 대비 25% 프리미엄. 반도체 기업으로선 적정 수준입니다.

Q3. EV/EBITDA가 중요한 이유는?

  • ① 가장 정확한 지표
  • ② 부채, 세금 영향 제거 → 영업 효율성 순수 평가
  • ③ 현금흐름 직결
  • ④ 국제 표준
정답: ②
EV/EBITDA는 부채와 세금 효과를 제거하므로 순수 영업 효율성을 비교할 수 있습니다. 글로벌 기업 비교에 유용합니다.

Q4. 학습자의 반도체 포트폴리오(2,800만)에 가장 필요한 분석은?

  • ① 매일 주가 추적
  • ② 분기 실적 후 PER/PBR/EV/EBITDA 비교
  • ③ 기술적 분석 학습
  • ④ 선물 투자로 확대
정답: ②
반도체는 사이클 산업입니다. 분기 실적 공시 후 지표를 비교하여 사이클 위치를 파악하는 것이 핵심입니다. 매일 추적은 노이즈이고, 기술적 분석은 초과 수익을 보장하지 않습니다.
