Lectures 15-18: 질량-스프링 시스템 모델링과 풀이

Lectures 15-18

1-DOF 질량-스프링 시스템의 운동방정식

가장 기본적인 진동 시스템입니다. 질량 $m$에 스프링 상수 $k$인 스프링이 연결되어 있습니다:

$$m\ddot{x} + kx = 0$$

이것은 2차 동차 상미분방정식(homogeneous ODE)입니다.

특성방정식과 해

$x(t) = e^{st}$를 시도해(trial solution) 대입하면:

$$ms^2 + k = 0 \quad \Rightarrow \quad s = \pm j\omega_n$$

여기서 $\omega_n = \sqrt{k/m}$은 고유진동수(natural frequency)입니다.

$$x(t) = A\cos(\omega_n t) + B\sin(\omega_n t)$$
물리적 의미: 감쇠가 없으면 시스템은 $\omega_n$ 주파수로 영원히 진동합니다. 진폭은 초기조건 $x(0)$과 $\dot{x}(0)$에 의해 결정됩니다.

비동차 질량-스프링 시스템 (외력이 있는 경우)

외력 $F(t)$가 작용하면:

$$m\ddot{x} + kx = F(t)$$

일반해 = 동차해(complementary) + 특수해(particular):

$$x(t) = x_c(t) + x_p(t)$$
핵심: 동차해는 시스템의 "자연스러운" 진동이고, 특수해는 외력에 의한 "강제" 응답입니다.

Lecture 19: 상태공간 표현 (State Space Form)

Lecture 19

고차 미분방정식을 1차 미분방정식의 연립 시스템으로 변환하는 방법입니다.

변환 방법

$m\ddot{x} + c\dot{x} + kx = F(t)$에서 상태변수를 정의합니다:

$x_1 = x$ (변위)

$x_2 = \dot{x}$ (속도)

그러면:

$$\dot{x}_1 = x_2$$
$$\dot{x}_2 = \frac{1}{m}\left(F(t) - cx_2 - kx_1\right)$$

행렬 형태로 쓰면:

$$\dot{\mathbf{x}} = \mathbf{A}\mathbf{x} + \mathbf{B}u$$
$$\begin{bmatrix} \dot{x}_1 \ \dot{x}_2 \end{bmatrix} = \begin{bmatrix} 0 & 1 \ -k/m & -c/m \end{bmatrix} \begin{bmatrix} x_1 \ x_2 \end{bmatrix} + \begin{bmatrix} 0 \ 1/m \end{bmatrix} F(t)$$
핵심: 상태공간 표현은 MIMO 시스템, 수치 시뮬레이션, 현대 제어이론의 기초입니다. Section 4에서 이 형태로 선형화합니다.

Lectures 20-25: 라플라스 변환 (Laplace Transform)

Lectures 20-25

라플라스 변환의 정의

시간 영역 함수 $f(t)$를 주파수 영역 $F(s)$로 변환합니다:

$$\mathcal{L}\{f(t)\} = F(s) = \int_0^{\infty} f(t)e^{-st}\,dt$$
왜 사용하는가? 미분방정식(풀기 어려움)을 대수방정식(풀기 쉬움)으로 바꿔줍니다!

핵심 라플라스 변환 쌍

$f(t)$$F(s)$
$1$$\frac{1}{s}$
$e^{at}$$\frac{1}{s-a}$
$\sin(\omega t)$$\frac{\omega}{s^2+\omega^2}$
$\cos(\omega t)$$\frac{s}{s^2+\omega^2}$
$t^n$$\frac{n!}{s^{n+1}}$

미분의 라플라스 변환 (가장 중요!)

$$\mathcal{L}\{\dot{f}(t)\} = sF(s) - f(0)$$
$$\mathcal{L}\{\ddot{f}(t)\} = s^2F(s) - sf(0) - \dot{f}(0)$$
초기조건 포함: 라플라스 변환은 초기조건을 자동으로 포함합니다. 이것이 ODE를 풀 때 매우 편리한 이유입니다.

적분의 라플라스 변환

$$\mathcal{L}\left\{\int_0^t f(\tau)d\tau\right\} = \frac{F(s)}{s}$$

예제: 라플라스로 질량-스프링 풀기

$m\ddot{x} + kx = 0$, $x(0) = x_0$, $\dot{x}(0) = 0$에 라플라스를 적용하면:

$m[s^2X(s) - sx_0] + kX(s) = 0$

$(ms^2+k)X(s) = msx_0$

$X(s) = \frac{msx_0}{ms^2+k} = \frac{sx_0}{s^2+\omega_n^2}$

역변환하면:

$$x(t) = x_0\cos(\omega_n t)$$
결론: 라플라스 변환으로 미분방정식을 대수적으로 풀 수 있습니다. $s$ 영역에서 조작 후 역변환으로 시간 해를 구합니다.

Lectures 26-27: 전달함수 (Transfer Function)

Lectures 26-27

SISO 전달함수

초기조건이 0일 때, 입력 $U(s)$와 출력 $Y(s)$ 사이의 비율입니다:

$$G(s) = \frac{Y(s)}{U(s)}$$

예: $m\ddot{x}+c\dot{x}+kx=F(t)$의 전달함수:

$$G(s) = \frac{X(s)}{F(s)} = \frac{1}{ms^2+cs+k}$$

MIMO 시스템

여러 입력과 여러 출력이 있는 경우, 전달함수는 행렬이 됩니다:

$$\mathbf{Y}(s) = \mathbf{G}(s)\mathbf{U}(s)$$
참고: 전달함수는 선형 시불변(LTI) 시스템에서만 정의됩니다. 이것이 비선형 시스템을 선형화해야 하는 또 다른 이유입니다.

Lectures 28-31: 디랙 델타 함수

Lectures 28-31

디랙 델타 함수 $\delta(t)$

"순간적인 충격"을 수학적으로 표현하는 함수입니다:

$$\delta(t) = \begin{cases} \infty & t=0 \ 0 & t \neq 0 \end{cases}, \quad \int_{-\infty}^{\infty}\delta(t)\,dt = 1$$

핵심 성질

선별 성질: $\int_{-\infty}^{\infty} f(t)\delta(t-a)\,dt = f(a)$

라플라스 변환: $\mathcal{L}\{\delta(t)\} = 1$

시간 이동: $\mathcal{L}\{\delta(t-a)\} = e^{-as}$

물리적 의미: 디랙 델타는 망치로 한 번 치는 것과 같습니다. 시스템에 $\delta(t)$를 입력하면 나오는 응답이 임펄스 응답(impulse response)이며, 이것은 시스템의 고유 특성을 완전히 담고 있습니다.

Lectures 32-35: 감쇠 유형과 안정성

Lectures 32-35

$m\ddot{x}+c\dot{x}+kx=0$의 특성방정식 $ms^2+cs+k=0$의 근:

$$s = \frac{-c \pm \sqrt{c^2-4mk}}{2m} = -\zeta\omega_n \pm \omega_n\sqrt{\zeta^2-1}$$

여기서 감쇠비 $\zeta = \frac{c}{2\sqrt{mk}}$입니다.

감쇠 유형조건응답 특성
과감쇠 (Overdamped)$\zeta > 1$진동 없이 느리게 감쇠
임계감쇠 (Critically damped)$\zeta = 1$진동 없이 가장 빠르게 감쇠
부족감쇠 (Underdamped)$0 < \zeta < 1$진동하면서 감쇠
비감쇠 (Undamped)$\zeta = 0$영원히 진동

안정성과 극점 (Poles)

전달함수의 분모를 0으로 만드는 $s$ 값이 극점입니다:

$$\text{시스템 안정} \iff \text{모든 극점의 실수부} < 0$$
핵심 규칙: 극점이 복소평면의 왼쪽 반평면(LHP)에 있으면 안정, 오른쪽(RHP)에 있으면 불안정입니다.

Lectures 36-39: 단위계단, 질량-스프링-댐퍼 해, 임펄스 응답

Lectures 36-39

단위계단 함수 $u(t)$ (Heaviside)

$$u(t) = \begin{cases} 0 & t<0 \ 1 & t \geq 0 \end{cases}, \quad \mathcal{L}\{u(t)\} = \frac{1}{s}$$

질량-스프링-댐퍼의 완전한 해

$m\ddot{x}+c\dot{x}+kx=F(t)$를 라플라스로 풀면:

$$X(s) = \frac{F(s) + (ms+c)x(0) + m\dot{x}(0)}{ms^2+cs+k}$$

임펄스 응답과 전달함수의 관계

$$h(t) = \mathcal{L}^{-1}\{G(s)\}$$
핵심: 전달함수의 역라플라스가 임펄스 응답입니다. 임펄스 응답을 알면 임의의 입력에 대한 응답을 합성곱으로 구할 수 있습니다.

Lectures 40-43: 시변 vs 시불변 시스템

Lectures 40-43

시불변(Time-Invariant) 시스템

시스템 파라미터가 시간에 따라 변하지 않는 시스템입니다:

$$m\ddot{x} + c\dot{x} + kx = F(t) \quad (m,c,k \text{ = 상수})$$

시변(Time-Variant) 시스템

파라미터가 시간의 함수인 경우:

$$m(t)\ddot{x} + c(t)\dot{x} + k(t)x = F(t)$$
중요: 라플라스 변환과 전달함수는 LTI(Linear Time-Invariant) 시스템에서만 사용 가능합니다. 시변 시스템에는 다른 해석 방법이 필요합니다.

Lectures 44-46: 오일러 수치해법

Lectures 44-46

해석적으로 풀 수 없는 미분방정식을 컴퓨터로 수치적으로 푸는 방법입니다.

전진 오일러법 (Forward Euler)

$$x(t+\Delta t) \approx x(t) + \Delta t \cdot \dot{x}(t)$$

상태공간 형태에서:

$$\mathbf{x}_{k+1} = \mathbf{x}_k + \Delta t \cdot f(\mathbf{x}_k, t_k)$$
장점: 구현이 매우 간단합니다.
단점: $\Delta t$가 크면 불안정해질 수 있습니다. 비선형/디랙 델타 입력에도 적용 가능합니다.

Lectures 47-53: 일반 입력, 합성곱, 비선형 시스템

Lectures 47-53

합성곱 적분 (Convolution Integral)

임의의 입력 $f(t)$에 대한 선형 시스템의 출력:

$$x(t) = \int_0^t h(t-\tau)f(\tau)\,d\tau = h(t) * f(t)$$

여기서 $h(t)$는 임펄스 응답, $*$은 합성곱 연산입니다.

라플라스 영역에서: 합성곱은 단순한 곱셈이 됩니다! $X(s) = H(s) \cdot F(s)$. 이것이 라플라스 변환의 가장 강력한 성질입니다.

비선형 시스템과 라플라스

비선형 시스템에는 라플라스/전달함수/합성곱을 직접 적용할 수 없습니다:

$\sin\theta$ → 라플라스 불가 (변수의 비선형 함수)

$\cos\theta \cdot \dot{\theta}$ → 두 변수의 곱, 라플라스 불가

Section 2 완료! 핵심 요약: 선형 ODE는 라플라스 변환, 전달함수, 합성곱으로 완벽하게 풀 수 있지만, 비선형 시스템에는 이 도구들을 사용할 수 없습니다. 따라서 Section 3에서 비선형 시스템을 선형화하는 방법을 배웁니다.
