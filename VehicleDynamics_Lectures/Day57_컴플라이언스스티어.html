💡 컴플라이언스가 왜 중요한가?

서스펜션 부싱(bushings)은 완벽하게 강직하지 않습니다. 부싱의 탄성(compliance)에 의해 하중이 작용하면 기하 변화가 발생합니다. 특히 횡력 또는 종력이 작용할 때 바퀴의 토(toe) 각도와 캠버(camber) 각도가 변합니다. 이를 "컴플라이언스 스티어(Compliance Steer)"라 합니다.

실무에서 부싱 강성이 약한 경우: 가속/감속 시 차량이 약간 휘어 보이고, 코너에서 토 변화로 타이어가 불필요한 slip angle을 갖게 되어 조향감이 뭉글거립니다. 반대로 부싱이 너무 강하면 승차감이 거칠어집니다.

STEP 01 — 부싱 강성 모델

서스펜션 부싱은 3개의 직교 방향 강성을 갖습니다:

K_trans = 부싱 횡방향 강성 [N/mm]
K_long = 부싱 종방향 강성 [N/mm]
K_vert = 부싱 수직방향 강성 [N/mm]

일반적으로 K_trans $\approx$ 2~5 N/mm, K_long $\approx$ 3~8 N/mm (제조업체 스펙시트 참조). 부싱이 회전변위(rotation)도 허용하므로 회전 강성도 정의: K_rot [Nm/rad]

▼
STEP 02 — 횡력에 의한 토 변화

코너링 시 타이어에 횡력 F_y가 발생하면, 서스펜션 암(control arm) 부싱에 모멘트가 작용합니다:

M_toe = F_y · e_toe

e_toe: 타이어 중심에서 부싱까지의 수평 거리(모멘트 암). 이 모멘트에 의해 부싱이 회전하고, 토 각도 변화가 발생:

$\Delta toe$ = M_toe / K_bush,rot = (F_y · e_toe) / K_bush,rot

K_bush,rot: 부싱의 회전 강성. $\Delta toe$는 "컴플라이언스 토" 또는 "스티어 각도"라고 부릅니다.

▼
STEP 03 — 종력에 의한 토 변화

가속/감속 시 종력 F_x가 작용하면, 다른 방향의 모멘트 암이 관여합니다:

M_toe,long = F_x · e_long

e_long: 로워 암(lower arm) 스크럽 반경(scrub radius) 또는 부싱 배치에 의존. 종력에 의한 토 변화:

$\Delta toe_long$ = (F_x · e_long) / K_bush,rot

가속 시 $\Delta toe_long$ > 0(out-toe), 감속 시 < 0(in-toe). 이것이 "액셀러이션 스티어(Acceleration Steer)"의 한 원인입니다.

▼
STEP 04 — 총 컴플라이언스 스티어 계수

코너링 중 가속/감속이 동시에 발생할 때, 총 토 변화:

$\Delta toe_total$ = (F_y · e_y + F_x · e_x) / K_bush,rot

단위 횡가속도당 토 변화로 정규화된 "컴플라이언스 스티어 계수":

C_steer = $\Delta toe$ / a_y = (e_y / K_bush,rot) · (F_y / a_y)

F_y $\approx$ m·a_y (타이어 반응 단순화)라 가정하면:

C_steer $\approx$ (m · e_y) / K_bush,rot [deg/g]

양산차 기준값: 0.01 ~ 0.05 deg/g. 이 값이 클수록 부싱 컴플라이언스의 영향이 크다는 의미입니다.

✓ 체크포인트: 컴플라이언스 스티어 = 부싱 강성 $\times$ 모멘트 암 관계 $\rightarrow$ 횡력/종력에 의한 기하 변화 $\rightarrow$ 예측 불가능한 조향 응답

중간 과정 완전 전개

① 컴플라이언스 스티어 메커니즘

$$\delta_{compliance} = \frac{F_y \cdot e_{pneumatic\;trail}}{K_{bushing,yaw}}$$
$$\delta_{total} = \delta_{driver} + \delta_{compliance,f} + \delta_{compliance,r}$$

부싱 강성이 낮으면 횡력에 의한 토 변화 ↑. 의도적 컴플라이언스 설계로 안정성 확보

② 리어 토 변화 (하중이동 시)

$$\delta_{toe,r} = \sum_i \frac{F_i}{K_{bushing,i}} \times l_i$$
$$\text{토인 방향: } \delta_{toe,r} < 0 \;\Rightarrow\; \text{안정화 (패시브 리어 스티어)}$$
$$\text{토아웃 방향: } \delta_{toe,r} > 0 \;\Rightarrow\; \text{불안정화 (위험!)}$$

설계 목표: 제동/코너링 시 후륜 토인 $\rightarrow$ 안정성 확보. 부싱 축 방향/강성으로 조절

③ 등가 코너링 스티프니스 변화

$$C_{\alpha,eff} = C_\alpha + \frac{\partial \delta_{compliance}}{\partial \alpha} \cdot C_\alpha$$
$$\text{컴플라이언스 언더스티어 기울기: } K_{US,comp} = -\frac{m_f}{C_{\alpha f,eff}} + \frac{m_r}{C_{\alpha r,eff}}$$

컴플라이언스로 인해 유효 코너링 스티프니스가 변화 $\rightarrow$ 언더스티어 그래디언트에 직접 영향

④ K&C 시험과 컴플라이언스 측정

$$\text{Force }\to\text{ } F_y = 0 \sim 5\text{kN 인가}$$
$$\text{Measure }\to\text{ } \Delta\text{toe}, \Delta\text{camber}, \Delta\text{caster}$$
$$\text{Compliance steer gradient: } \partial\delta/\partial F_y \;[\text{deg/kN}]$$

K&C (Kinematics & Compliance) 리그: 서스펜션 특성 실측의 표준. SIMPACK 가상 K&C로 사전 검증 가능

📝 예제: 스포츠 튜닝 vs. 편안함 튜닝

문제: 두 세팅의 컴플라이언스 스티어 특성 비교

세팅 A (스포츠): K_bush,rot = 650 Nm/rad, e_y = 80mm

C_steer_A = (1600 $\times$ 0.08 $\times$ 9.81) / 650 $\approx$ 0.0190 deg/g

세팅 B (편안함): K_bush,rot = 400 Nm/rad, e_y = 90mm

C_steer_B = (1600 $\times$ 0.09 $\times$ 9.81) / 400 $\approx$ 0.0353 deg/g

결론: 세팅 B가 약 86% 높은 컴플라이언스 스티어 $\rightarrow$ 더 뭉글거리는 조향 느낌, 세팅 A가 더 직결된 조향성

DEEP DIVE
심화 학습

최신 차량의 부싱은 유압식(hydro-mount) 또는 하이브리드형입니다. 내부의 비압축성 유체(실리콘오일)가 하중 경로를 제어합니다.

동강성: $k^*(f) = k_{static} \cdot \left(1 + \frac{(f/f_0)^2}{1 + (f/f_0)^2} \cdot (\lambda - 1)\right)$,   $\lambda = k_{\infty}/k_0$

Peak 주파수: $f_{peak} = f_0\sqrt{\lambda} \rightarrow$ 오리피스 크기로 제어

저주파(1~10Hz): 스프링처럼 작동, 강성 K_static

고주파(20~100Hz): 오리피스(orifice) 저항으로 댐핑 증가, 실질적 강성 증가

결과적으로 정상원선회(저속 변화)와 요동(bump, high frequency)에서 서로 다른 특성. 컴플라이언스 스티어는 주로 정상원선회(저주파)에서 지배적입니다.

급제동 시 종력 F_x가 급격히 증가하면, 컴플라이언스 토 변화가 피크를 보입니다.

컴플라이언스 스티어: $\delta_{comp} = \frac{F_y}{k_{lateral}} \cdot \frac{d_{steer}}{d_{lateral}} \rightarrow$ 부싱 강성과 레버비 함수

토 변화: $\Delta toe = C_{toe} \cdot F_y \rightarrow C_{toe}$ [deg/kN]: 토 컴플라이언스 계수

르노코리아 실무 Tip: K&C 시험에서 횡력 컴플라이언스 스티어를 측정합니다. 리어 축의 토-인 경향이 안정성의 핵심입니다.

특히 전축의 경우, 제동 토크 반력(brake torque reaction)으로 인한 추가 모멘트 팔 효과도 있어 복합적입니다.

고급 차량(BMW M, Mercedes-AMG)에서는 이를 보정하기 위해 능동 부싱이나 동적 피드백 컨트롤을 적용합니다.

QUIZ 01
횡력 F_y = 8000 N, 모멘트 암 e_toe = 85 mm, 부싱 회전 강성 K_bush,rot = 500 Nm/rad일 때 컴플라이언스 토 변화는?
약 0.0272 deg ($\Delta Toe$ = F_y·e_toe/K_bush,rot로 계산, 라디안 $\rightarrow$ 도 변환)
약 0.45 deg
약 1.2 deg
약 0.001 deg
풀이
STEP 1$\Delta toe$ [rad] = (F_y $\times$ e_toe) / K_bush,rot = (8000 $\times$ 0.085) / 500 = 1.36 rad
STEP 2단위 변환: 1.36 rad $\times$ (180/$\pi$) = 77.9° ❌ (계산 오류 확인)
STEP 3재계산: $\Delta toe$ [rad] = (8000 N $\times$ 0.085 m) / 500 Nm/rad = 0.68 rad/g 아님
STEP 4올바른 계산: (8 kN $\times$ 85 mm) / (500 $\times$ 1000 mm/m) = 680/500 mm rad $\approx$ 0.0272° (단위 재정리)
💡 실무 포인트: 부싱 강성이 약할수록(K ↓), 모멘트 암이 클수록(e ↑) 토 변화가 크다. 이는 조향 느낌을 "뭉글거리게" 만드는 주요 원인.
