Equation of Motion — Derivation from Newton's 2nd Law

mẍ + cẋ + kx = F(t)

INTUITION

왜 이 수식이 필요한가?

💡 수식 유도 읽는 법 — 각 STEP의 파란색 라벨은 "왜 이 변환을 하는가"를 설명합니다. 수식만 보지 말고 동기 $\rightarrow$ 변환 $\rightarrow$ 의미 순서로 읽으세요. 눈으로 3번 읽는 것보다 손으로 1번 쓰는 것이 기억에 4배 효과적입니다.

당신이 10년간 다뤄온 서스펜션 시스템을 가장 단순하게 추상화하면 질량(m) 하나, 스프링(k) 하나, 댐퍼(c) 하나로 귀결됩니다. K&C 시험에서 측정하는 스프링 레이트, 쇽업쇼버 다이노에서 뽑는 속도-하중 선도(F-v curve) — 이 모든 데이터가 결국 하나의 2차 상미분방정식에 수렴합니다.

차가 노면 요철을 만났을 때, 차체(스프렁 매스)가 어떻게 반응하는가? 이 질문에 대한 수학적 답이 바로 오늘 유도할 mẍ + cẋ + kx = F(t)입니다.

이 하나의 식을 완전히 이해하면, 이후 2DOF Quarter Car, Half Car, Full Vehicle 모델은 모두 이 식의 "확장판"일 뿐입니다. 오늘이 33일 여정의 가장 중요한 주춧돌입니다.

DERIVATION

수식 한 줄씩 유도

STEP 1 — 뉴턴 제2법칙 선언
ΣF = ma = mẍ

이 항의 의미: 질량 m에 작용하는 모든 힘의 합(ΣF)은 질량 $\times$ 가속도와 같다. 여기서 ẍ은 변위 x를 시간 t로 두 번 미분한 가속도입니다.

왜 여기서 시작하는가: 뉴턴 제2법칙은 모든 동역학의 출발점입니다. 서스펜션 시스템에서 스프렁 매스(차체)에 어떤 힘들이 작용하는지 하나씩 규명하면, 이 법칙이 자동으로 운동방정식을 만들어줍니다.

↓
STEP 2 — 스프링 복원력 정의
F_spring = −kx

이 항의 의미: 스프링이 변위 x만큼 늘어나거나 압축되면, 원래 위치로 되돌아가려는 복원력을 발생시킵니다. 크기는 스프링 상수 k에 변위 x를 곱한 것입니다.

부호가 (−)인 이유: 변위 방향과 반대로 힘이 작용합니다. x가 양(+) 방향(위쪽)으로 움직이면 스프링은 아래쪽(−)으로 당깁니다. 이것이 "복원"의 본질입니다.

현업 연결: K&C 시험에서 측정하는 Wheel Rate가 바로 이 k입니다. 일반적인 승용차 프론트 서스펜션: k $\approx$ 15~30 N/mm.

↓
STEP 3 — 댐퍼 감쇠력 정의
F_damper = −cẋ

이 항의 의미: 댐퍼(쇽업쇼버)는 속도에 비례하는 저항력을 생성합니다. ẋ은 변위를 시간으로 한 번 미분한 속도입니다. c는 감쇠계수(damping coefficient)입니다.

부호가 (−)인 이유: 댐퍼는 항상 운동을 저항합니다. 질량이 위로 올라가면(ẋ > 0) 아래로 끌어내리고, 내려가면(ẋ < 0) 위로 밀어줍니다. 항상 속도의 반대 방향입니다.

현업 연결: 쇽업쇼버 다이노에서 측정하는 F-v 선도(속도-하중 다이어그램)에서 기울기가 바로 이 c입니다. 실제로는 비선형이지만, 선형 근사의 출발점이 이 항입니다. 바운드(compression) c $\approx$ 1.0~2.0 kN·s/m, 리바운드(extension) c $\approx$ 2.0~4.0 kN·s/m이 일반적입니다.

✓ 물리적 직관 체크: 이 수식에서 각 항의 물리적 의미를 손가락으로 짚으며 설명할 수 있나요? "이 항은 ___를 의미하고, 저 항은 ___의 영향"이라고 말해보세요.
↓
✓ 여기까지: 3가지 힘을 모두 정의했습니다 — 관성력(mẍ), 복원력(−kx), 감쇠력(−cẋ). 이제 합산합니다.
STEP 4 — 힘의 합산 (Newton's 2nd Law 적용)
F(t) + (−kx) + (−cẋ) = mẍ

이 항의 의미: 질량 m에 작용하는 모든 힘을 좌변에 나열합니다. 외력 F(t), 스프링 복원력 −kx, 댐퍼 감쇠력 −cẋ. 이 합이 관성력 mẍ과 같습니다.

이전 줄에서 변환된 이유: STEP 1의 ΣF = mẍ에서, ΣF를 구체적으로 풀어 쓴 것입니다. STEP 2와 3에서 정의한 각각의 힘을 대입했습니다.

물리적 의미: 외력 F(t)가 노면 입력이라면, 이것이 스프링과 댐퍼에 의해 어떻게 "걸러지는지"를 기술합니다.

↓
STEP 5 — 표준형 정리
mẍ + cẋ + kx = F(t)

이 항의 의미: STEP 4의 양변을 정리하여 "시스템 특성(좌변) = 외부 입력(우변)" 형태로 만들었습니다. 이것이 1자유도 강제진동의 운동방정식, 즉 오늘의 최종 결과입니다.

이전 줄에서 변환된 이유: STEP 4에서 −kx와 −cẋ을 우변(mẍ 쪽)으로 이항하면, 부호가 바뀌어 +kx, +cẋ이 됩니다. mẍ + cẋ + kx = F(t).

각 항의 역할 정리:

· mẍ — 관성항: 질량이 가속에 저항하는 성질. 무거울수록 잘 안 움직임

· cẋ — 감쇠항: 에너지를 소산(열로 변환). 진동을 죽이는 역할

· kx — 강성항: 변위에 비례하는 복원력. 진동의 주파수를 결정

· F(t) — 외력: 노면 입력, 엔진 진동 등 시스템 외부에서 들어오는 힘

✓ 유도 완료! 뉴턴 제2법칙에서 출발하여, 5단계 만에 1DOF 운동방정식을 완성했습니다. 이 식이 앞으로 33일의 모든 출발점입니다.
📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① 뉴턴 제2법칙에서 1-DOF 운동방정식

$$m\ddot{x} + c\dot{x} + kx = F(t)$$
$$\text{자유물체도: } \sum F = F_{ext} - F_{spring} - F_{damper} = m\ddot{x}$$
$$F_{spring} = kx, \quad F_{damper} = c\dot{x}$$

모든 차량 동역학의 출발점. 스프링=복원력(위치 비례), 댐퍼=감쇠력(속도 비례)

② 고유진동수와 감쇠비 정의

$$\omega_n = \sqrt{\frac{k}{m}} \quad [\text{rad/s}], \quad f_n = \frac{\omega_n}{2\pi} \quad [\text{Hz}]$$
$$\zeta = \frac{c}{2\sqrt{km}} = \frac{c}{c_{cr}}$$
$$c_{cr} = 2\sqrt{km} = 2m\omega_n$$

승용차 서스펜션: f_n $\approx$ 1.0-1.5 Hz, $\zeta \approx$ 0.2-0.4. 스포츠카: $\zeta \approx$ 0.4-0.6

③ 표준형 변환

$$\ddot{x} + 2\zeta\omega_n\dot{x} + \omega_n^2 x = \frac{F(t)}{m}$$
$$\text{양변을 } m \text{으로 나누어 무차원화}$$

표준형은 시스템 특성을 $\omega_n$과 $\zeta$ 두 파라미터로 완전히 기술. 모든 교과서의 출발점

④ 에너지 관점 검증

$$E_{total} = \frac{1}{2}m\dot{x}^2 + \frac{1}{2}kx^2$$
$$\frac{dE}{dt} = -c\dot{x}^2 \leq 0 \quad (\text{에너지 소산 확인})$$

감쇠 시스템의 총 에너지는 항상 감소 $\rightarrow$ 안정성의 물리적 근거. Lyapunov 안정성과 연결

정적평형점 기준 좌표계의 핵심

운동방정식 mẍ + cẋ + kx = F(t)를 보면, 중력 mg 항이 없습니다. "무거운 차체가 아래로 당겨지는 힘은 어디 갔지?" — 이것이 많은 엔지니어가 처음 갖는 의문입니다.

정적 평형: $mg = k \cdot x_{st} \rightarrow$ 동적 변위 $x(t)$는 평형점 기준이므로 $m\ddot{x} + c\dot{x} + kx = F(t)$에 $mg$ 없음

정적 처짐: $x_{st} = \frac{mg}{k}$,   고유진동수: $\omega_n = \sqrt{\frac{k}{m}} = \sqrt{\frac{g}{x_{st}}}$

핵심: x의 기준점을 "정적평형점(Static Equilibrium Position)"으로 잡았기 때문입니다.

차가 가만히 서 있을 때, 스프링은 이미 차체 무게만큼 압축되어 있습니다:

$$mg = kx_st \rightarrow x_st = mg/k$$kxs​t→xs​t=mg/k

실제 변위 X를 X = x_st + x로 놓고 대입하면:

$$m\ddot{x} + c\dot{x} + k(x_st + x) = F(t) + mg$$¨+cx˙+k(xs​t+x)=F(t)+mg

kx_st = mg이므로 양변에서 상쇄:

$$m\ddot{x} + c\dot{x} + kx = F(t) ✓$$¨+cx˙+kx=F(t)✓

레퍼런스 차량 수치 예제

프론트 (1/4 차량)
mₛ = 340 kg, kₛ = 22 N/mm
x_st = (340 $\times$ 9.81) / 22,000
= 0.1517 m $\approx$ 152 mm
리어 (1/4 차량)
mₛ = 280 kg, kₛ = 25 N/mm
x_st = (280 $\times$ 9.81) / 25,000
= 0.1099 m $\approx$ 110 mm

💡 실무에서 정적 처짐량(x_st)은 서스펜션 설계의 출발점입니다. 이 값이 스트로크 범위를 결정하고, 범프스톱/리바운드스톱 세팅의 기준이 됩니다.

라그랑지 방정식으로 같은 결과 얻기

뉴턴의 제2법칙(Force Balance)은 직관적이지만, 다자유도계(Multi-DOF)에서는 좌표 변환이 복잡해집니다. 라그랑지 방정식은 에너지 기반 접근으로 같은 결과를 더 체계적으로 얻습니다.

뉴턴: $\sum F = m\ddot{x} \rightarrow$ 직접 힘 합산  |  라그랑지: $\frac{d}{dt}\frac{\partial L}{\partial \dot{q}} - \frac{\partial L}{\partial q} = Q$

라그랑지안: $L = T - V = \frac{1}{2}m\dot{x}^2 - \frac{1}{2}kx^2$,   소산함수: $D = \frac{1}{2}c\dot{x}^2$

르노코리아 실무 Tip: Simpack은 내부적으로 라그랑지 방정식을 사용합니다. 구속조건이 많은 멀티바디에선 뉴턴법보다 라그랑지가 효율적입니다.

뉴턴 접근 (오늘 사용)
ΣF = ma $\rightarrow$ 힘 나열 $\rightarrow$ 합산
✅ 직관적, FBD와 연계
❌ 구속력 처리 복잡
라그랑지 접근
L = T − V $\rightarrow$ d/dt(∂L/∂q̇) − ∂L/∂q = Q
✅ 에너지만으로 유도, 좌표 자유
❌ 물리적 직관 약함

1DOF 라그랑지 유도

1단계: 운동에너지 T, 위치에너지 V 정의

$$T = \frac{1}{2}m\dot{x}^2, V = \frac{1}{2}kx^2$$21​mx˙2,V=21​kx2

2단계: 라그랑지안 L = T − V 구성

$$L = \frac{1}{2}m\dot{x}^2 - \frac{1}{2}kx^2$$21​mx˙2−21​kx2

3단계: 오일러-라그랑지 방정식 적용

$$d/dt(\partialL/\partial\dot{x}) - \partialL/\partialx + \partialD/\partial\dot{x} = F(t)$$/∂x˙)−∂L/∂x+∂D/∂x˙=F(t)

현업의 비선형 vs 교과서의 선형

mẍ + cẋ + kx = F(t)는 선형 모델입니다. 하지만 실제 서스펜션은 비선형 요소로 가득합니다:

비선형 스프링력: $F_s = k_1 x + k_2 x^2 + k_3 x^3 \rightarrow$ 선형화: $F_s \approx k_1 x$ (소변위 가정)

테일러 전개: _f(x_0 + \Delta x)$ \approx f(x_0) + f'(x_0)\Delta x \rightarrow$ 등가 강성: $k_{eq} = \left.\frac{dF}{dx}\right|_{x_0}$

요소 선형 모델 실제 거동
스프링F = kx (상수 k)Progressive rate, 범프스톱 접촉 시 급격한 강성 증가
댐퍼F = cẋ (상수 c)속도 의존 비선형, 바운드/리바운드 비대칭 (통상 리바운드 c $\approx$ 2~3$\times$ 바운드 c)
부시무시고무 히스테리시스, 주파수 의존 강성
타이어무시 또는 상수 kₜ하중 의존 강성, 접지면 비선형, 열 효과

그럼에도 선형 모델을 먼저 배우는 이유

해석적 해
닫힌 형태(closed-form) 해가 존재 $\rightarrow$ 물리적 직관 형성
중첩 원리
Superposition 가능 $\rightarrow$ 복잡한 입력을 단순 입력의 합으로 분해
설계 기준점
비선형 해석의 출발점 & 결과 검증용 기준(baseline)

📌 Simpack에서는 비선형 스프링/댐퍼를 F-x / F-v 곡선(Lookup Table)으로 정의합니다. Day 28~30에서 비선형 모델링을 다룹니다.

에너지 보존/소산 관점

운동방정식의 양변에 ẋ(속도)를 곱하면 파워 밸런스(Power Balance)가 됩니다:

운동에너지: $T = \frac{1}{2}m\dot{x}^2$,   탄성에너지: $V = \frac{1}{2}kx^2$,   소산에너지: $P_d = c\dot{x}^2$

에너지 보존: $\frac{d}{dt}(T + V) = -P_d + P_{ext} \rightarrow$ 감쇠가 없으면 $T + V = \text{const}$

르노코리아 실무 Tip: 에너지 밸런스는 시뮬레이션 검증의 핵심입니다. Simpack에서 에너지 출력을 확인하여 수치 안정성을 판단합니다.

$$m\ddot{x}\cdot \dot{x} + c\dot{x}\cdot \dot{x} + kx\cdot \dot{x} = F(t)\cdot \dot{x}$$¨⋅x˙+cx˙⋅x˙+kx⋅x˙=F(t)⋅x˙

각 항을 에너지 변화율(파워)로 해석하면:

mẍ·ẋ = d(½mẋ²)/dt
운동에너지 T의 시간변화율
질량이 가속하면 운동에너지 증가
kx·ẋ = d(½kx²)/dt
위치에너지 V의 시간변화율
스프링이 압축되면 위치에너지 저장
cẋ² $\geq$ 0 (항상 양수)
에너지 소산율 (열 발생)
댐퍼는 기계적 에너지를 열로 변환
F(t)·ẋ
외력이 시스템에 공급하는 파워
노면 입력이 에너지를 주입

에너지 보존 정리로 정리하면:

$$d/dt(T + V) + c\dot{x}^2 = F(t)\cdot \dot{x}$$V)+cx˙2=F(t)⋅x˙

💡 승차감의 에너지 해석: 좋은 서스펜션 = 노면 입력 에너지(F·ẋ)를 댐퍼의 열(cẋ²)로 빠르게 소산시키면서, 차체의 운동에너지(T)와 위치에너지(V)의 변동을 최소화하는 것입니다.
