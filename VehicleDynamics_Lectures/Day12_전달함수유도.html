Day 05~06에서 '진폭비와 위상각'을 배웠는데, 이를 일반화한 것이 전달함수입니다. 라플라스 변환(s=σ+jω)을 사용하면 미분방정식이 대수 방정식이 되어 다루기 쉬워집니다. 차량 NVH에서 전달함수는 '노면→차체' 경로의 진동 특성을 완전히 기술하는 도구입니다. Simpack의 FRA(Frequency Response Analysis)가 바로 전달함수를 수치적으로 구하는 것.

이 항의 의미: X_s에 대해 정리하면 전달함수. 분자: 입력(노면)이 스프링·댐퍼를 통해 전달. 분모: 시스템 동특성(관성+감쇠+강성).

부호/변환 이유: s=jω 대입: |H(jω)| = Day 05의 진폭비와 관계. 분모=0이 되는 s값이 극점(pole) → 고유진동수 결정.

📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① 운동방정식의 라플라스 변환

$$m\ddot{x} + c\dot{x} + kx = F(t) \xrightarrow{\mathcal{L}} ms^2X(s) + csX(s) + kX(s) = F(s)$$
$$X(s)(ms^2 + cs + k) = F(s)$$

변환 이유: 미분방정식 → 대수방정식으로 변환. 초기조건 = 0 가정 (강제응답만 관심)

② 전달함수 H(s) 도출

$$H(s) = \frac{X(s)}{F(s)} = \frac{1}{ms^2 + cs + k} = \frac{1/m}{s^2 + 2\zeta\omega_n s + \omega_n^2}$$

표준형: 분모의 계수에서 ω_n = √(k/m), ζ = c/(2√(mk)) 직접 읽기 가능

③ 주파수응답함수: s = jω 대입

$$H(j\omega) = \frac{1/m}{(j\omega)^2 + 2\zeta\omega_n(j\omega) + \omega_n^2} = \frac{1/m}{(\omega_n^2 - \omega^2) + j(2\zeta\omega_n\omega)}$$
$$|H(j\omega)| = \frac{1/m}{\sqrt{(\omega_n^2-\omega^2)^2 + (2\zeta\omega_n\omega)^2}}$$

크기(Magnitude)는 진폭비, 위상(Phase)은 arctan(허수/실수). 보드 플롯: |H|을 dB로, Phase를 degree로 표시

④ 보드 플롯의 3구간 해석

$$\text{저주파 } (\omega \ll \omega_n): |H| \approx \frac{1}{k} = \frac{1}{\omega_n^2 m} \quad (\text{정적 강성 지배})$$
$$\text{공진 } (\omega = \omega_n): |H| \approx \frac{1}{2\zeta\omega_n^2 m} = \frac{Q}{k} \quad (\text{감쇠만 지배})$$
$$\text{고주파 } (\omega \gg \omega_n): |H| \approx \frac{1}{m\omega^2} \quad (\text{관성 지배, }-40\text{dB/dec 감쇠})$$

설계 인사이트: 고주파 절연은 질량↑(무거울수록 좋음), 공진 억제는 ζ↑(감쇠↑). 서스펜션은 이 둘의 트레이드오프!

DEEP DIVE

심화 학습

전달함수 H(s)의 본질은 분모=0의 근(극점, poles)과 분자=0의 근(영점, zeros)으로 결정됩니다.

전달함수: $H(s) = \frac{N(s)}{D(s)} = \frac{\prod(s - z_i)}{\prod(s - p_j)}$ → 영점 $z_i$, 극점 $p_j$

2차 시스템 극점: $p = -\zeta\omega_n \pm j\omega_n\sqrt{1-\zeta^2}$

극점(Poles): 분모 mₛ·s²+c_s·s+kₛ=0의 근 → s = −ζωₙ ± jω_d (Day 04와 동일!) → 극점의 실수부 = 감쇠, 허수부 = 진동 주파수

영점(Zeros): 분자 c_s·s+kₛ=0의 근 → s = −kₛ/c_s → 이 주파수에서 전달이 0 (완전 차단)

Bode Plot 읽기: · 극점 주파수 근처: 공진 피크 · 영점 주파수 근처: 반공진(dip) · 고주파(ω≫ωₙ): s² 지배 → −40 dB/decade 감쇠

Day 13에서 이를 '전달률(Transmissibility)' 개념으로 발전시킵니다.

■ 극점 (Poles) — 분모 = 0의 근

극점 실수부 $= -\zeta\omega_n$: 감쇠 속도 결정  |  허수부 $= \omega_d$: 진동 주파수 결정

영점: $H(z_i) = 0$ → 해당 주파수에서 전달이 차단됨 (anti-resonance)

시스템의 고유 특성을 결정: s = −σ ± jωd

σ = ζωₙ: 감쇠 속도 (σ 클수록 빨리 감쇠)

ωd = ωₙ√(1−ζ²): 감쇠 진동 주파수

■ 영점 (Zeros) — 분자 = 0의 근

특정 주파수에서 출력이 0 → "anti-resonance"

쿼터카 H(s)에서 영점: s = −kₛ/cₛ → 실수축 위의 점

■ Simpack에서 확인

FRA 결과의 magnitude plot에서 극점 = peak, 영점 = dip으로 나타남

■ Magnitude Plot (크기)

바디 전달률: $T_{body}(\omega) = \frac{X_s}{X_r} = \frac{k_s + j\omega c}{k_s + k_t - m_s\omega^2 + j\omega c}$

$|T_{body}| < 1$ (방진 영역): $\omega > \omega_n\sqrt{2}$ → $f > f_n\sqrt{2}$

르노코리아 실무 Tip: Simpack에서 Sine Sweep 가진 후 바디 가속도/노면 변위로 FRF를 구합니다. Bode Plot 기울기가 -40 dB/dec인지 확인하세요.

|H(jω)| > 1 (0 dB 이상): 진동 증폭 영역 → 승차감 악화

|H(jω)| < 1 (0 dB 이하): 진동 절연 영역 → 서스펜션이 일하는 영역!

피크 높이 = Q-factor ≈ 1/(2ζ): 감쇠가 작을수록 피크가 높음

■ Phase Plot (위상)

0° 근처: 입출력이 동위상 → 차체가 노면을 그대로 추종

−90°: 공진 부근 → 에너지가 가장 많이 흡수되는 지점

−180° 접근: 입출력 역위상 → 차체가 노면과 반대로 움직임 (고주파)

■ 레퍼런스 차량 타겟

body bounce peak < 3 dB (|H| < 1.4), 절연 시작점 < 2 Hz
