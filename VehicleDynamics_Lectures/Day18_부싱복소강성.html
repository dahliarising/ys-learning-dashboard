INTUITION

왜 이 수식이 필요한가?

💡 수식 유도 읽는 법 — 각 STEP의 파란색 라벨은 "왜 이 변환을 하는가"를 설명합니다. 수식만 보지 말고 동기 → 변환 → 의미 순서로 읽으세요. 눈으로 3번 읽는 것보다 손으로 1번 쓰는 것이 기억에 4배 효과적입니다.

고무 부싱은 스프링이면서 동시에 댐퍼다. 변형하면 일부 에너지는 저장(탄성)되고 일부는 열로 소산(점성)된다. 이 두 성질을 하나로 표현하는 것이 복소강성 K* = K' + iK''이다. K'(저장강성)은 복원력, K''(손실강성)은 감쇠력을 담당한다. 주파수가 올라가면 K'도 K''도 변한다 — 이것이 '주파수 의존성'이다.

Step 1: 조화 가진 하의 힘-변위 관계
x(t)=X0ei\omegat,F(t)=K∗⋅x(t)=(K′+iK′′)X0ei\omegatx(t) = X_0 e^{i\omegat}, F(t) = K* \cdot x(t) = (K' + iK'') X_0 e^{i\omegat}x(t)=X0​ei\omegat,F(t)=K∗⋅x(t)=(K′+iK′′)X0​ei\omegat

이 항의 의미: 조화 변위 입력에 대해 힘은 변위와 같은 주파수지만 위상이 다르다. K'은 변위와 동위상 성분(in-phase), K''은 90° 앞선 성분(quadrature).

부호/변환 이유: K'' > 0이면 힘이 변위보다 앞서고, 에너지가 소산된다(히스테리시스 루프 면적 > 0).

이전 단계와의 연결: Day 06에서 위상각 φ를 배웠다. 부싱의 위상 지연은 η = tanδ로 표현된다.

↓
Step 2: 손실계수(Loss Factor) 정의
η=K′′/K′=tan⁡δη = K''/K' = \tan \deltaη=K′′/K′=tanδ

이 항의 의미: η(에타)는 1사이클에서 소산된 에너지 / (2π × 최대 저장 에너지)의 비율이다. δ는 힘과 변위 사이의 위상차(loss angle).

부호/변환 이유: η ≈ 0.05~0.3이 일반 고무 범위. η가 클수록 NVH(소음진동) 감쇠에 유리하지만 열 발생도 증가.

이전 단계와의 연결: Day 03의 감쇠비 ζ와 관계: 1DOF에서 η ≈ 2ζ (작은 감쇠 근사).

↓
Step 3: 주파수 의존성 모델
K′(f)=K0⋅(1+α⋅log⁡10(f/f0)),η(f)≈η0⋅(f/f0)βK'(f) = K_0 \cdot (1 + \alpha \cdot \log_{10}(f/f_0)), \quad \eta(f) \approx \eta_0 \cdot (f/f_0)^{\beta}K′(f)=K0​⋅(1+α⋅log10​(f/f0​)),η(f)≈η0​⋅(f/f0​)β

이 항의 의미: 고무의 저장강성은 주파수가 10배 올라갈 때 α만큼 증가(동적 경화 효과). 손실계수도 주파수에 따라 변한다.

부호/변환 이유: α > 0 → 주파수 증가 시 부싱이 뻣뻣해짐. β > 0이면 고주파에서 손실 증가.

이전 단계와의 연결: Day 05 강제진동에서 k는 상수였다. 부싱에서는 k(ω)로 주파수 함수가 된다.

Step 4: Kelvin-Voigt 모델과의 관계
K∗=k+i\omegac→K′=k,K′′=\omegac→η=\omegac/kK* = k + i\omegac \rightarrow K' = k, K'' = \omegac \rightarrow η = \omegac/kK∗=k+i\omegac→K′=k,K′′=\omegac→η=\omegac/k

이 항의 의미: Kelvin-Voigt(스프링 + 댐퍼 병렬) 모델에서 복소강성을 유도하면 손실강성이 주파수에 비례한다.

부호/변환 이유: 이 모델은 고주파에서 η가 무한대로 발산하는 비물리적 문제가 있다 → 실무에서는 테이블 입력이 안전.

이전 단계와의 연결: Day 17의 Maxwell 모델은 직렬, Kelvin-Voigt는 병렬. 두 모델의 차이를 복소강성으로 비교할 수 있다.

심화 학습

부싱 동특성 시험은 일반적으로 1~100Hz 주파수 스위프로 수행한다. 진폭은 ±0.1~±1.0mm (소진폭)와 ±3~±5mm(대진폭)를 나눠서 측정한다. 큰 이유: (1) 고무의 Payne 효과 — 진폭이 커지면 K'이 감소하고 η가 증가한다. (2) Mullins 효과 — 첫 사이클은 이후와 다르므로 컨디셔닝 사이클이 필요하다. Simpack에 입력할 때는 실차 작동 진폭에 해당하는 데이터를 선택해야 한다.

■ Kelvin-Voigt (스프링 ∥ 댐퍼)

K* = k + jωc → η = ωc/k: 주파수에 비례하여 η 증가

장점: 간단 / 단점: 고주파에서 비현실적 η 예측

■ Maxwell (스프링 — 댐퍼 직렬)

저주파: 댐퍼가 자유 → 강성 ≈ 0 / 고주파: 댐퍼가 잠김 → 강성 = k

장점: 응력 이완 표현 / 단점: 정적 강성 = 0

■ Generalized Maxwell (Prony Series)

K*(ω) = K_∞ + Σ(Kᵢ·(jωτᵢ)/(1+jωτᵢ))

실제 고무의 넓은 주파수 의존성을 가장 정확히 재현

Simpack: Bushing > Viscoelastic 모드에서 Prony 계수 입력 가능

기억 고리

🧠 암기법 — "복소강성 = 찰떡 + 스프링"
K' (저장 강성) = 🧊 스프링 (에너지 저장·반환)
K'' (손실 강성) = 🍯 찰떡 (에너지 흡수·열 변환)
K* = K'(1 + iη) → η = 찰떡/스프링 비율 = 손실계수!
