DERIVATION

수식 한 줄씩 유도

STEP 1 — 디커플링 근사 (바운스 주파수)
$$f_{bounce} \approx \frac{1}{2\pi}\sqrt{\frac{k_f+k_r}{m}}$$

이 항의 의미: 커플링이 약할 때(a·k_f $\approx$ b·kᵣ) 바운스는 총 스프링 강성을 총 질량으로 나눈 1DOF와 유사.

부호/변환 이유: k_f+kᵣ: 전후 스프링의 병렬 합. m = 차체 전체 질량 (하프카이므로 좌우 합산).

이전 단계와의 연결: Day 09의 쿼터카 $\omega$₁$\approx \sqrt{\cdot}$(kₛ/mₛ)와 같은 구조. kₛ $\rightarrow$ k_f+kᵣ, mₛ $\rightarrow$ m.

↓
STEP 2 — 디커플링 근사 (피치 주파수)
$$f_{pitch} \approx \frac{1}{2\pi}\sqrt{\frac{a^2 k_f + b^2 k_r}{I_y}}$$

이 항의 의미: 피치 복원 모멘트(a²k_f+b²kᵣ)를 피치 관성모멘트(I_y)로 나눈 것. 회전 자유도의 '스프링-질량'이 a²k+I_y.

부호/변환 이유: a², b² 제곱이 붙는 이유: 모멘트 = 힘 $\times$ 레버암, 강성모멘트 = 강성 $\times$ 레버암².

이전 단계와의 연결: Day 10 STEP 3의 피치 방정식에서 커플링 항을 무시(z=0)하면 이 근사식.

↓
STEP 3 — Simpack 교차 검증 절차
오차=∣f수식−fSimpack∣/fSimpack×100허용 범위:<5\text{오차} = |f_\text{수식} - f_Simpack| / f_Simpack \times 100% \text{허용 범위}: < 5% (\text{선형 모델})오차=∣f수식​−fS​impack∣/fS​impack×100허용 범위:<5

이 항의 의미: 수식 결과와 Simpack Eigenvalue 결과를 비교. 5% 이내면 모델 정합성 확인. 차이가 크면 파라미터(특히 motion ratio, 부시 강성) 확인.

부호/변환 이유: Simpack은 비선형 요소를 선형화하여 고유치를 구하므로, 선형 수식과의 차이는 비선형성 정도를 나타냄.

이전 단계와의 연결: 이 검증 루틴이 디지털 트윈 구축의 기본. Day 30~33에서 전체 모델 검증에 동일한 방법 사용.

📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① 하프카 운동방정식 행렬 전개

$$m_s \ddot{z}_s + c_f(\dot{z}_s - l_f\dot{\theta} - \dot{z}_{uf}) + c_r(\dot{z}_s + l_r\dot{\theta} - \dot{z}_{ur}) + k_f(z_s - l_f\theta - z_{uf}) + k_r(z_s + l_r\theta - z_{ur}) = 0$$
$$I_y \ddot{\theta} - c_f l_f(\dot{z}_s - l_f\dot{\theta} - \dot{z}_{uf}) + c_r l_r(\dot{z}_s + l_r\dot{\theta} - \dot{z}_{ur}) - k_f l_f(z_s - l_f\theta - z_{uf}) + k_r l_r(z_s + l_r\theta - z_{ur}) = 0$$

핵심: 바운스(z_s)와 피치($\theta$)가 커플링. l_f ≠ l_r이면 off-diagonal 항 존재 $\rightarrow$ 모드가 순수 바운스/피치가 아닌 결합 모드!

② 고유치 문제 설정: [K - $\omega$²M]{$\phi$} = 0

$$\begin{bmatrix} k_f+k_r - \omega^2 m_s & -k_f l_f + k_r l_r \\ -k_f l_f + k_r l_r & k_f l_f^2 + k_r l_r^2 - \omega^2 I_y \end{bmatrix} \begin{Bmatrix} Z \\ \Theta \end{Bmatrix} = \begin{Bmatrix} 0 \\ 0 \end{Bmatrix}$$

디커플링 조건: k_f·l_f = k_r·l_r이면 off-diagonal = 0 $\rightarrow$ 바운스와 피치 완전 독립. 실차 적용: 프론트 스프링 약, 리어 스프링 강하게 하여 근사적 디커플링 추구

③ 특성방정식 (Characteristic Equation)

$$\omega^4 - \omega^2\left(\frac{k_f+k_r}{m_s} + \frac{k_f l_f^2 + k_r l_r^2}{I_y}\right) + \frac{(k_f+k_r)(k_f l_f^2 + k_r l_r^2) - (k_r l_r - k_f l_f)^2}{m_s I_y} = 0$$

판별: 이 4차 $\rightarrow$ 2차 방정식의 두 근 $\omega$₁², $\omega$₂²가 바운스/피치 고유진동수. 작은 값이 바운스(~1.0-1.5Hz), 큰 값이 피치(~1.5-2.0Hz)

④ 모드형상비 (Mode Shape Ratio)

$$\frac{\Theta}{Z}\bigg|_{\omega=\omega_i} = \frac{\omega_i^2 m_s - (k_f+k_r)}{k_r l_r - k_f l_f}$$

Θ/Z > 0이면 바운스 우세, Θ/Z < 0이면 피치 우세. 르노 튜닝: 승차감을 위해 바운스 모드의 피치 성분 최소화 $\rightarrow$ 디커플링 지향

DEEP DIVE

심화 학습

수식에서 k_f, kᵣ은 '휠 레이트(wheel rate)'입니다. 실제 스프링 레이트와 다릅니다!

Motion Ratio: $MR = \frac{\Delta x_{spring}}{\Delta x_{wheel}} \rightarrow$ 휠 레이트: $k_{wheel} = k_{spring} \cdot MR^2$

설치비(Installation Ratio): $IR = MR^2 \rightarrow c_{wheel} = c_{damper} \cdot IR$

관계식: k_wheel = kₛpring $\times$ MR² 여기서 MR = Motion Ratio = 스프링 변위 / 휠 변위

맥퍼슨 스트럿: MR $\approx$ 0.9~1.0 (거의 1:1) 더블 위시본: MR $\approx$ 0.6~0.8 (레버비 존재) 멀티링크: MR $\approx$ 0.5~0.9 (설계에 따라 다양)

OEM 실무: K&C 시험에서 측정하는 것은 wheel rate. 스프링 업체에 발주할 때는 spring rate. MR은 서스펜션 기구학(Kinematics)에서 결정되며, Simpack K&C 해석으로 구합니다. Day 16~17에서 서스펜션 기구학과 motion ratio를 상세히 다룹니다.

■ 커플링 상태 (a·k_f ≠ b·kᵣ)

커플링 계수: $e = k_r b - k_f a \rightarrow e = 0$이면 디커플링

DI(Dynamic Index): $DI = \frac{\kappa^2}{a \cdot b} \rightarrow DI = 1$이면 바운스-피치 완전 분리

르노코리아 실무 Tip: 실차에서 DI = 1 정확히 맞추기 어렵습니다. DI = 0.9~1.1 범위가 실무 목표입니다.

전방 범프 통과 시 바운스와 피치가 동시에 발생 $\rightarrow$ '시소' 같은 복합 모션

탑승자는 상하 + 전후 가속도를 동시에 느낌 $\rightarrow$ 불쾌감 증가

■ 디커플링 상태 (a·k_f = b·kᵣ)

바운스 모드: 차체가 순수하게 상하로만 진동 (flat ride)

피치 모드: 차체가 CG 중심으로만 회전

■ 레퍼런스 차량 실차 확인

a=1.05m,b=1.55m,kf=22N/mm,kr=26N/mma = 1.05m, b = 1.55m, k_f = 22 N/mm, k_r = 26 N/mma=1.05m,b=1.55m,kf​=22N/mm,kr​=26N/mm
a⋅kf=1.05×22=23.1vsb⋅kr=1.55×26=40.3a\cdot k_f = 1.05\times 22 = 23.1 vs b\cdot k_r = 1.55\times 26 = 40.3a⋅kf​=1.05×22=23.1vsb⋅kr​=1.55×26=40.3

레퍼런스 차량는 완전 디커플링이 아님 $\rightarrow$ 약간의 바운스-피치 커플링 존재

이것이 Flat Ride 튜닝의 출발점: kᵣ을 낮추거나 CG를 후방으로 이동

■ Maurice Olley의 실험적 발견 (1934)

Olley 조건: $\frac{f_{rear}}{f_{front}} > 1 \rightarrow$ 통상 $f_r/f_f \approx 1.1\text{~}1.3$

$f_{front} = \frac{1}{2\pi}\sqrt{\frac{k_f}{m_f}}$,   $f_{rear} = \frac{1}{2\pi}\sqrt{\frac{k_r}{m_r}}$

후축 고유진동수를 전축보다 10~20% 높게 설정하면 승차감이 향상된다:

frear/ffront≈1.1 1.2f_rear / f_front \approx 1.1 ~ 1.2fr​ear/ff​ront≈1.1 1.2

■ 물리적 이유

전축이 먼저 범프를 만남 $\rightarrow$ 차체 전방이 먼저 올라감 (피치 시작)

후축이 시간차를 두고 범프를 만남 $\rightarrow$ 이때 전방은 이미 내려오기 시작

fᵣear > f_front이면 후축 응답이 더 빨라 피치 모션을 상쇄!

■ 설계 식

최적 주파수비=kr/mr/kf/mf≈1.1 1.2\text{최적 주파수비} = \sqrt{k_r/m_r} / \sqrt{k_f/m_f} \approx 1.1~1.2최적 주파수비=kr​/mr​​/kf​/mf​​$\approx$1.1 1.2

Simpack에서 eigenvalue analysis 후 바운스/피치 모드 주파수 확인 $\rightarrow$ Flat Ride 달성 여부 판단

실차 적용

🚗 레퍼런스 차량 하프카 모드 해석 실습

mₛ = 1360 kg (전체 스프렁)

I_y = 2100 kg·m²

a = 1.05 m (전축~CG)

b = 1.55 m (CG~후축)

k_f = 44 kN/m (양쪽 합)

kᵣ = 52 kN/m (양쪽 합)

바운스 주파수:

f_b = (1/2$\pi$)$\sqrt{\cdot}$((44+52)$\times$10³/1360) = 1.34 Hz ✓

피치 주파수:

f_p = (1/2$\pi$)$\sqrt{\cdot}$((1.05²$\times$44+1.55²$\times$52)$\times$10³/2100) = 1.52 Hz ✓

Flat Ride 비율: f_p/f_b = 1.13 $\rightarrow$ Olley 범위(1.1~1.2) 내! ✓
