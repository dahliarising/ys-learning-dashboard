💡 토(Toe) — 직진성과 타이어 수명의 열쇠

토(Toe)는 차량을 위에서 내려다봤을 때 좌·우 바퀴가 차체 중심선에 대해 안쪽(토-인) 또는 바깥쪽(토-아웃)으로 향하는 각도입니다. 불과 $\pm$0.1° 차이로 타이어 편마모 패턴이 달라지고, 고속 직진 안정성이 급변합니다.

핵심 직관: 토-인은 양 타이어에 안쪽으로 미끄러지려는 사이드포스를 미리 부여해, 외란(노면 불균일, 바람 등)에 대한 복원력을 만듭니다. 반대로 토-아웃은 코너 진입 시 바깥 바퀴가 먼저 슬립각을 확보해 턴인 응답이 빨라집니다.

Bump Toe(범프 토): 서스펜션이 압축될 때 토 각이 변하는 현상. 이를 제어하지 않으면 노면 요철마다 차가 좌우로 흔들리는 '원더링(wandering)' 현상이 발생합니다. 서스펜션 기구학 설계에서 범프 토 기울기($\Delta toe$/$\Delta jounce$)를 0에 가깝게 유지하는 것이 핵심 목표입니다.

0-5m 토-인/토-아웃 물리적 의미와 사이드포스 발생 원리
5-15m 정적 토 $\rightarrow$ 동적 토(범프 토, 롤 토) 수식 유도
15-20m Simpack KC 해석으로 범프 토 커브 추출
20-25m 실차 토 편마모 사례와 alignment 보정 전략

Step 1 / 4 — 정적 토 각 정의

정적 토 각 정의

토(Toe)는 상면도에서 바퀴의 중심선과 차량의 종방향 중심선(CL)이 이루는 각도입니다.

토-인(Toe-in): 전륜이 중심선 방향으로 향함 $\rightarrow$ 직진 안정성 향상

토-아웃(Toe-out): 전륜이 바깥쪽으로 벌어짐 $\rightarrow$ 선회 응답성 향상

일반 승용차 정적 토: 전륜 토-인 5~10분(분각), 후륜 토-인 0~5분

$\delta_{toe}$ = arctan((d_f - d_r) / L_tread)

해석:

여기서 d_f = 전방 측정 거리(wheel center 간격), d_r = 후방 측정 거리, L_tread = 측정 기준선(휠 중심선 간격)입니다. 개별 토 각은 분각으로 표현되며, 전체 토 각 = 2 $\times$ 개별 토 각(좌우 바퀴의 합)입니다. 서스펜션 조정 및 정렬(alignment)의 핵심 파라미터입니다.

✅ 체크포인트

토의 정적 정의와 측정 방법을 이해했으면 계속 진행.

Step 2 / 4 — 범프 토(Bump Steer)

범프 토 - 수직 이동에 따른 토 변화

바퀴가 수직 방향으로 이동할 때(버프/리바운드), 타이로드와 컨트롤암이 서로 다른 아크를 그리므로 토가 변합니다.

범프 토 변화는 타이로드 기하학, 컨트롤암 길이, 너클(knuckle) 형상에 의존합니다.

좌우 바퀴의 토 변화가 다르면 요(yaw) 모멘트 발생 $\rightarrow$ 언더스티어/오버스티어 악화

목표: 범프/리바운드에서 토 변화 최소화(제로 범프 스티어)

$\Delta \delta_{toe} \approx \frac{r_{tie}\sin\theta_{tie} - r_{arm}\sin\theta_{arm}}{L_{knuckle}}$

해석:

여기서 r_tie = 타이로드 유효 길이, r_arm = 컨트롤암 유효 길이, $\theta_{tie}$ = 타이로드 각속변위, $\theta_{arm}$ = 컨트롤암 각속변위, L_knuckle = 너클의 스티어링 축과 바퀴 중심 간 거리입니다. 범프 스티어는 가감속 시 차량 안정성과 직진성에 큰 영향을 미칩니다. 현대 차량은 링키지 설계를 최적화하여 범프 스티어를 거의 0에 가깝게 조정합니다.

✅ 체크포인트

범프 토의 발생 메커니즘과 기하학적 표현을 이해했으면 계속 진행.

Step 3 / 4 — 롤 스티어(Roll Steer)

롤 스티어 - 바디 롤 중 토의 변화

차량이 선회할 때 바디가 롤(roll)하면, 좌우 바퀴의 수직 변위가 서로 반대 방향입니다.

롤 각 $\phi$에 따른 수직 변위: $\Delta z_roll$ = $\phi \times$ t_w/2 (t_w = 트랙 폭)

범프 토 경향이 있으면 롤 중 좌우 바퀴의 토가 불일치 $\rightarrow$ 요 모멘트 생성

롤 스티어는 언더스티어(roll-in) 또는 오버스티어(roll-out) 특성을 제어하는 설계 도구입니다.

$\delta_{roll}$_steer = (∂$\delta_{toe}$/∂z) $\times \Delta$z_roll = (∂$\delta_{toe}$/∂z) $\times$ ($\phi \times$ t_w/2)

해석:

여기서 ∂$\delta_{toe}$/∂z = 범프 토 민감도(rad/m), $\phi$ = 바디 롤 각도(rad), t_w = 트랙 폭(m)입니다. 선회 중 롤 스티어는 차량의 요 응답 특성을 조정하여 코너링 안정성을 향상시킵니다. 슈퍼카 및 경주차는 롤 스티어를 적극 활용하여 선회 반응성을 극대화합니다.

✅ 체크포인트

롤 스티어와 선회 응답의 관계를 이해했으면 계속 진행.

Step 4 / 4 — 운용 조건에서의 총 토

운용 조건에서의 총 토 각

실제 운용 환경에서 바퀴가 경험하는 토는 정적 값에서 범프 토, 컴플라이언스 토, 압축 효과를 고려합니다.

브레이킹 중: 무게 이동에 의한 전륜 압축 $\rightarrow$ 범프 토 발생 $\rightarrow$ 안정성을 위해 토-인 선호

가속 중: 후륜 압축, 전륜 신장 $\rightarrow$ 후륜 토-아웃, 전륜 토-인 경향

목표: 모든 운용 조건에서 안정성과 승차감의 균형 $\rightarrow$ 범프 스티어 최소화, 코너링 특성 최적화

$\delta_{toe}$_total = $\delta_{toe}$_static + $\Delta \delta_{toe}$_bump + $\delta_{compliance}$_toe + $\delta_{load}$_effect

해석:

여기서 $\delta_{toe}$_static = 정적 토 각도, $\Delta \delta_{toe}$_bump = 범프/리바운드 시 변화, $\delta_{compliance}$_toe = 컴플라이언스 특성(고무 부싱 변형), $\delta_{load}$_effect = 하중 이동에 따른 변화입니다. 현대 서스펜션은 멀티링크 구조를 통해 모든 운용 영역에서 토를 최적으로 제어하여 직진 안정성, 선회 응답성, 승차감을 동시에 만족합니다. Simpack 멀티바디 시뮬레이션을 통해 모든 운동 시나리오에서 토의 변화를 검증합니다.

📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① 토 각도 정의

$$\delta_{toe} = \arctan\frac{d_{front} - d_{rear}}{L_{track}}$$
$$\text{토인: } \delta_{toe} > 0 \;(\text{전방 좁음})$$
$$\text{토아웃: } \delta_{toe} < 0 \;(\text{전방 넓음})$$

전륜 토인: 직진 안정성 ↑, 타이어 마모 약간 ↑. 후륜 토인: 고속 안정성의 핵심

② 범프 스티어 (Bump Steer)

$$\frac{\partial \delta_{toe}}{\partial z_{wheel}} \quad [\text{deg/mm}]$$
$$\text{이상적: } \frac{\partial \delta}{\partial z} = 0 \;(\text{IC가 타이로드 축 연장선 위})$$

범프 스티어 > 0.01 deg/mm이면 노면 입력에 의한 조향 불안정. 타이로드 위치로 조절

③ 롤 스티어

$$\frac{\partial \delta_{toe}}{\partial \phi_{roll}} \quad [\text{deg/deg}]$$
$$\text{전륜 롤 언더스티어: } \delta_{toe,f}/\phi > 0$$
$$\text{후륜 롤 오버스티어 방지: } \delta_{toe,r}/\phi < 0 \;(\text{토인 방향})$$

롤 스티어는 과도 응답에 큰 영향. 멀티링크 서스펜션의 장점 = 롤 스티어 독립 튜닝 가능

④ 정적 토 설정값

$$\delta_{toe,static} = \delta_{toe,target} - \delta_{toe,settling}$$
$$\text{전륜: } \approx 0 \sim +0.1^{\circ} \;(\text{토인})$$
$$\text{후륜: } \approx +0.1 \sim +0.2^{\circ} \;(\text{토인})$$

주행 중 부싱 변형으로 토가 변화 $\rightarrow$ 정적 설정 시 주행 중 목표값 도달하도록 보정

🚗 사례 1: C-세그먼트 전륜 범프 토 최적화

배경: 고속도로 직진 주행 시 운전자가 핸들을 미세하게 좌우로 보정하는 '원더링' 클레임 발생. 정렬(alignment) 상태는 정상이나, 노면 요철 통과 시 차가 쏠리는 증상.

분석: KC(Kinematics & Compliance) 시험에서 전륜 범프 토 기울기 측정: $\Delta toe$ = +0.08°/10mm jounce (토-아웃 방향). 범프 시 좌우 바퀴가 벌어지며 불안정한 방향 변화 유발.

계산: 타이로드 외측점 높이를 6mm 하향 조정 $\rightarrow$ 범프 토 기울기: +0.08° $\rightarrow$ −0.01°/10mm (거의 0에 수렴). 타이로드의 순간 회전축과 로어암 순간 회전축의 교점이 바퀴 중심 높이에 가깝게 배치되도록 기구학 보정.

결과: 원더링 지수(steering correction frequency) 120$\rightarrow$80 km/h 구간에서 40% 감소. 타이어 내측 편마모(feathering) 해소. 10,000 km 주행 후 타이어 마모 깊이 차이 0.8mm $\rightarrow$ 0.2mm 이내.

🏁 사례 2: 후륜 정적 토 세팅에 따른 주행 특성 변화

설명: 동일 차량에서 후륜 토-인을 0', 3', 6'(분각)으로 변경하며 고속 레인체인지 테스트.

결과: 0' — 후륜이 미끄러지기 쉬워 오버스티어 경향, 3' — 안정적 중립 스티어, 6' — 후륜 그립 과잉으로 둔한 응답(언더스티어 증가). 양산 세팅은 3'으로 결정.

의미: 후륜 토-인 1분각(0.017°)은 약 0.15mm의 좌우 간격 차이인데, 이 미세한 차이가 한계 영역 거동을 결정합니다. 서스펜션 엔지니어가 토 값에 민감한 이유입니다.

토 기하학 심화

타이로드 기하학이 만드는 의도치 않은 조향

범프 스티어(Bump Steer)란 서스펜션이 범프/리바운드할 때 의도치 않게 토 각이 변하는 현상입니다. 원인은 타이로드의 순간중심이 서스펜션 암의 순간중심과 일치하지 않기 때문입니다.

범프 스티어 계수: $\dfrac{\partial\delta_{\text{toe}}}{\partial z} \bigg|_{z=0}$   [deg/mm] — 서스펜션 스트로크 대비 토 변화율

범프 스티어 그래디언트: $\frac{d\delta_{toe}}{dz}$ [°/mm]. 목표: 전축 $|\frac{d\delta}{dz}| \lt 0.01$ °/mm, 후축은 의도적으로 $\frac{d\delta}{dz} \lt 0$ (범프 토인 → 안정성 향상).

기하학적 조건: 범프 스티어를 0으로 만들려면, 타이로드 아우터 볼조인트가 서스펜션 순간중심과 휠 중심을 잇는 직선 위에 놓여야 합니다. 실무에서는 $\pm$0.1°/25mm 이내를 목표로 합니다.

범프 스티어가 토-인(toe-in) 방향이면 주행 안정성은 높아지지만 타이어 마모가 증가합니다. 토-아웃(toe-out) 방향이면 불안정해지므로 절대 피해야 합니다.

K&C 리그에서 범프 스티어 측정 시, $\pm$50mm 범프 범위에서 토 변화 곡선을 측정하고 직선 구간의 기울기(°/mm)로 평가합니다. 르노코리아 기준은 전륜 $\leq$ 0.05°/25mm, 후륜 $\leq$ 0.03°/25mm입니다.

정적 토 ≠ 동적 토

정렬 장비에서 측정하는 정적 토(static toe)는 차량이 정지 상태에서의 값입니다. 하지만 주행 중에는 구동력, 제동력, 횡력에 의한 컴플라이언스 토(compliance toe)가 추가됩니다.

① Ackermann 조건: $\cot\delta_o - \cot\delta_i = \dfrac{w}{L}$   ($w$: 윤거, $L$: 축거)

② 조향 오차: $\Delta\delta = \delta_{\text{actual}} - \delta_{\text{Ackermann}}$   ③ 100% Ackermann: $\tan\delta_i = \dfrac{L}{R - w/2}$

④ 토인 총량: $\text{Total Toe} = \delta_L + \delta_R$   (양수 = Toe-in, 음수 = Toe-out)

르노코리아 실무 Tip: 르노코리아 전륜은 약 0~0.1° toe-out, 후륜은 0.1~0.2° toe-in이 일반적 — 타이어 마모와 직진 안정성의 트레이드오프입니다.

타이로드 높이 최적화: 이론적 제로 범프스티어 조건은 타이로드 외측 볼조인트가 순간중심-휠중심 연결선의 연장선 위에 위치할 때 달성됩니다.

후륜의 컴플라이언스 토: 코너링 시 후륜에 작용하는 횡력이 부싱 탄성을 통해 토-인 방향 변화를 만들면 언더스티어 경향이 강화되어 안정적입니다. 반대로 토-아웃이면 오버스티어가 됩니다.

이것이 멀티링크 리어 서스펜션의 핵심 설계 철학입니다: 5개 링크의 부싱 강성 방향을 정교하게 설계하여, 횡력 입력 시 자연스럽게 토-인이 발생하도록 만듭니다. BMW의 'Integral Active Steering'이 대표적 사례입니다.
