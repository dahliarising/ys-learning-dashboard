💡 이번 강의의 목표

캠버($\gamma$)는 정면에서 본 바퀴의 기울어진 각도입니다. 음의 캠버는 바퀴 상단이 안쪽으로 기울어진 상태인데, 코너링할 때 타이어가 접지면을 최대로 활용하게 됩니다. 바운스할 때 캠버가 변하면, 그로 인한 횡력 변화가 차의 조종 반응을 직접 좌우합니다.

캠버 각 정의

정면(전방)에서 본 바퀴의 수직선 대비 기울기를 캠버라고 합니다.

양의 캠버: 상단이 바깥쪽으로 기울어짐 (레이싱 스포츠카)

음의 캠버: 상단이 안쪽으로 기울어짐 (일반 승용차, 코너링 안정성)

일반 승용차 정적 캠버: -0.5°~-1.5°

γ∈[−2°,+2°]

각도 부호 규약: 정면에서 본 바퀴의 상단이 안쪽으로 향할 때 음(-)입니다.

Step 5 / 5

캠버 스러스트 (Camber Thrust)

타이어가 캠버 각도를 가지면 횡력이 발생합니다. 이를 캠버 스러스트라고 합니다.

F_camber = C_$\gamma \times \gamma$

C_$\gamma$: 캠버 강성 [N/°]. 타이어 종류, 하중, 온도에 따라 변함.

일반적으로 C_$\gamma \approx$ 200~500 N/°

Fcamber​=Cγ​×γ

해석:

음의 캠버는 코너링할 때 바퀴를 '안쪽으로 당기는' 횡력을 생성합니다.

📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① 캠버각 정의 및 부호 규약

$$\gamma = \text{타이어 중심면과 수직선 사이의 각도}$$
$$\gamma > 0: \text{Positive (상단 바깥쪽)}, \quad \gamma < 0: \text{Negative (상단 안쪽)}$$

SAE 좌표계 기준. 코너링 시 바깥바퀴에 네거티브 캠버 $\rightarrow$ 캠버 스러스트↑ $\rightarrow$ 그립↑

② 범프 캠버 변화 (더블위시본)

$$\Delta\gamma_{bump} \approx -\frac{z_{bump}}{L_{IC,front}} \quad [\text{rad}]$$
$$\text{목표: } \frac{d\gamma}{dz} \approx -0.3^{\circ} \sim -0.8^{\circ}/\text{cm}$$

설계 원리: 범프 시 네거티브 캠버 증가 $\rightarrow$ 코너링 중 하중이동 시 바깥바퀴 접지력 극대화

③ 롤 캠버 변화

$$\Delta\gamma_{roll} = \text{Roll Camber Coefficient} \times \phi_{roll}$$
$$\text{RC coeff} = 1 - \frac{d\gamma}{d\phi} \quad (\text{이상적: } 0, \; \text{맥퍼슨: } \approx 0.5 \sim 0.8)$$

맥퍼슨은 롤 캠버가 크므로(바깥바퀴 포지티브 방향) $\rightarrow$ 다중링크/더블위시본 대비 코너링 불리

④ 캠버 스러스트 (Camber Thrust)

$$F_{y,camber} = C_\gamma \cdot \gamma$$
$$C_\gamma \approx 0.1 \sim 0.15 \times C_\alpha \quad (C_\alpha: \text{코너링 강성})$$

캠버 스러스트는 코너링포스의 10-15% 수준. 작지만 한계 영역에서 중요한 추가 그립!

캠버 기하학 심화

캠버 각이 만드는 횡력의 물리적 원리

타이어가 캠버 각 $\gamma$만큼 기울면, 접지면(contact patch)에서 트레드 요소가 바닥에 닿는 순간 원뿔 위를 구르는 것과 같은 효과가 발생합니다. 이때 발생하는 횡력이 캠버 스러스트입니다.

캠버 스러스트: $F_{y,\gamma} = C_\gamma \cdot \gamma$ 여기서 $C_\gamma$는 캠버 강성(camber stiffness, 일반적으로 $C_\alpha$의 $5\sim15\%$ 수준). 캠버 1°당 횡력 약 $50\sim150$ N (승용 타이어 기준).

캠버 스러스트 Fy_camber $\approx$ C$\gamma$ · $\gamma$ 로 근사하며, 캠버 강성 C$\gamma$는 일반적으로 코너링 강성 C$\alpha$의 10~20% 수준입니다. 작아 보이지만, 코너링 초기(슬립각이 작을 때)에는 무시할 수 없는 기여를 합니다.

설계 Trade-off: 네거티브 캠버(-$\gamma$)는 코너링 시 외측 타이어의 접지면을 넓혀 그립을 높이지만, 직진 주행 시 타이어 안쪽 편마모를 유발합니다. 승용차는 보통 -0.5° ~ -1.5°, 레이싱카는 -3° ~ -4°까지 설정합니다.

르노코리아 실무에서는 캠버 커브(bump camber curve)를 설계할 때, 정적 캠버와 동적 캠버(롤 시 캠버 변화)를 합산한 '유효 캠버(effective camber)'를 기준으로 타이어 성능 영역을 최적화합니다.

같은 캠버인데 원인이 다르다

휠의 캠버는 두 가지 독립적인 메커니즘으로 변합니다. 범프 캠버(Bump Camber)는 서스펜션이 범프/리바운드할 때 기구학적으로 발생하며, 순간중심 위치에 의해 결정됩니다.

롤 시 캠버 변화: $\gamma_{roll} = \phi \cdot (1 - \frac{d\gamma}{d\phi})$ 여기서 $\phi$는 롤 각도, $\frac{d\gamma}{d\phi}$는 롤 캠버 계수. 이상적으로 $\frac{d\gamma}{d\phi} \approx 0.8\sim1.0$ (롤 시 타이어 직립 유지).

롤 캠버(Roll Camber)는 차체가 롤할 때 차체 기울기가 직접 휠에 전달되는 성분입니다. 롤 캠버 $\approx$ 롤 각($\phi$) $\times$ 1.0이며, 이것은 기구학과 무관하게 항상 발생합니다.

코너링 시 유효 캠버 = 정적 캠버 + 범프 캠버(범프량) + 롤 캠버(롤 각). 좋은 서스펜션 설계란, 롤 캠버로 잃어버리는 캠버를 범프 캠버로 보상하여 외측 타이어가 항상 최적 캠버 영역에 있도록 만드는 것입니다.

더블위시본은 범프 캠버 기울기를 -0.5°~ -1.0°/25mm로 자유롭게 튜닝할 수 있지만, MacPherson은 구조적으로 -0.2°~ -0.4°/25mm 수준에 제한됩니다.
