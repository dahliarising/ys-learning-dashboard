Day 02에서 비감쇠 자유진동(c=0)을 풀었습니다. 현실 세계에서 감쇠 없는 시스템은 존재하지 않죠. 자동차 쇼크 업소버(댐퍼)가 바로 감쇠를 담당합니다. 그런데 감쇠가 '얼마나 강한가'를 어떻게 표현할까요? 절대값 c [N·s/m]는 시스템마다 스케일이 달라 비교가 어렵습니다. 그래서 c를 '임계감쇠계수 cᵪ'로 나눠서 무차원 비율 ζ(제타)를 만듭니다. ζ=1이 경계점: ζ<1이면 진동하면서 감쇠(부족감쇠), ζ=1이면 가장 빨리 정지(임계감쇠), ζ>1이면 진동 없이 느리게 복귀(과감쇠). 승용차 서스펜션은 보통 ζ=0.2~0.4 범위의 부족감쇠 영역에 있습니다.

DERIVATION

수식 한 줄씩 유도

STEP 1 — 감쇠 포함 자유진동 방정식
mx¨+cx˙+kx=0m\ddot{x} + c\dot{x} + kx = 0mx¨+cx˙+kx=0

이 항의 의미: 외력 F(t)=0, 감쇠 c≠0인 자유진동 방정식입니다. Day 01의 일반 방정식에서 우변만 0으로 놓은 것.

부호/변환 이유: 세 항 모두 좌변에 양(+)으로 모여 있습니다. 관성항 + 감쇠항 + 복원항 = 0.

이전 단계와의 연결: Day 01: mx''+cx'+kx=F(t) → Day 02: mx''+kx=0 (c=0) → 오늘: mx''+cx'+kx=0 (c≠0, F=0)

↓
STEP 2 — 특성방정식과 임계감쇠계수 cᵪ
ms2+cs+k=0→판별식Δ=c2−4mkms^2 + cs + k = 0 \rightarrow \text{판별식} \Delta = c^2 - 4mkms2+cs+k=0→판별식Δ=c2−4mk

이 항의 의미: x=Ae^(st) 가정 후 대입하면 2차 특성방정식. 판별식 Δ의 부호가 해의 성격을 결정합니다. Δ=0이 되는 c값이 임계감쇠계수.

부호/변환 이유: Δ=0 → c² = 4mk → cᵪ = 2√(mk). 이것이 '진동과 비진동의 경계'를 만드는 감쇠값입니다.

이전 단계와의 연결: Day 02에서 c=0일 때 s²=−k/m (순허수근)이었는데, 이제 c가 추가되어 근의 성질이 바뀝니다.

↓
STEP 3 — 감쇠비 ζ 정의
ζ=c/ccr=c/(2mk)=c/(2mωn)\zeta = c / c_{cr} = c / (2\sqrt{mk}) = c / (2m\omega_n)ζ=c/ccr​=c/(2mk​)=c/(2mωn​)

이 항의 의미: 감쇠비 ζ(제타)는 실제 감쇠계수 c를 임계감쇠계수 cᵪ로 나눈 무차원 수. ζ<1 부족감쇠, ζ=1 임계감쇠, ζ>1 과감쇠.

부호/변환 이유: cᵪ = 2√(mk) = 2mωₙ (∵ ωₙ=√(k/m)). 세 가지 동치 표현 모두 자주 사용됩니다.

이전 단계와의 연결: ωₙ = √(k/m)은 Day 02에서 유도. cᵪ = 2mωₙ으로 쓰면 ζ = c/(2mωₙ)이 가장 깔끔합니다.

DEEP DIVE

심화 학습

승용차 서스펜션의 ζ가 정확히 1(임계감쇠)이 아닌 이유는 무엇일까요?

ζ=1이면 진동 없이 가장 빨리 평형점으로 돌아옵니다. 이론적으로 최적 같지만, 실제로는 너무 딱딱한 느낌(harsh ride)을 줍니다.

ζ가 작을수록(0에 가까울수록): 부드럽지만 출렁임이 오래 지속 → 멀미 유발 ζ가 클수록(1에 가까울수록): 출렁임은 빨리 멈추지만 충격 전달이 직접적 → 불쾌

최적점은 '1~2회 진동 후 수렴'하는 ζ≈0.2~0.4 영역. - 컴포트 지향(세단): ζ≈0.2~0.3 - 스포츠 지향: ζ≈0.3~0.5 - 레이싱: ζ≈0.5~0.7 (타이어 접지력 우선)

K&C 시험에서 바운스 감쇠비를 측정할 때, Logarithmic Decrement(대수감쇠율) 방법으로 ζ를 역산합니다. δ = ln(x_n/x_{n+1}) → ζ = δ/√(4π²+δ²). 이 공식은 Day 04에서 유도합니다.

임계감쇠계수 유도의 수학적 배경

특성방정식 ms² + cs + k = 0의 판별식 D = c² − 4mk입니다.

D=0→c2=4mk→ccr=2mkD = 0 \rightarrow c^2 = 4mk \rightarrow c_{cr} = 2\sqrt{mk}D=0→c2=4mk→ccr​=2mk​

D = 0은 중근 조건, 즉 진동과 비진동의 경계입니다.

D > 0 (과감쇠)
ζ > 1
진동 없이 느리게 복귀
D = 0 (임계감쇠)
ζ = 1
가장 빠르게 복귀 (이론상)
D < 0 (부족감쇠)
ζ < 1
진동하며 복귀 (승용차)

💡 승용차가 ζ = 0.2~0.4를 사용하는 이유: 임계감쇠(ζ=1)는 진동이 없지만 복귀가 느립니다. 부족감쇠 영역에서 약간의 진동을 허용하되, 1~2 사이클 내 수렴시키는 것이 승차감과 조종 안정성의 최적 밸런스입니다.

레퍼런스 차량 프론트 감쇠비 산출

바운드 (Compression)
c_bump = 1,800 N·s/m
cᵪ = 2√(22,000 × 340) = 5,469 N·s/m
ζ_bump = 1800/5469 = 0.329
리바운드 (Extension)
c_reb = 3,800 N·s/m
cᵪ = 5,469 N·s/m
ζ_reb = 3800/5469 = 0.695

모든 진동계는 운동에너지(KE)와 위치에너지(PE)의 교환으로 이해할 수 있습니다:

Etotal=KE+PE=12mx˙2+12kx2E_{total} = KE + PE = \frac{1}{2}m\dot{x}^2 + \frac{1}{2}kx^2Etotal​=KE+PE=21​mx˙2+21​kx2

감쇠가 없으면 E_total = 일정 (보존). 감쇠가 있으면 매 사이클마다 에너지가 열로 손실됩니다.

차량 적용: 댐퍼가 흡수하는 에너지 = 승차감의 핵심. 에너지 소산율이 높을수록 빠르게 안정.

등가 감쇠비(평균): ζ_eq ≈ (ζ_bump + ζ_reb)/2 ≈ 0.512

💡 리바운드 감쇠가 바운드보다 약 2배 큰 이유: 범프 통과 시 빠른 압축(바운드)을 허용하되, 복귀(리바운드)는 천천히 하여 차체 안정성을 확보합니다. 이 비대칭이 승차감의 핵심 튜닝 포인트입니다.
