Day 12의 전달함수에서 '노면 변위 입력 대비 차체 변위 출력'의 크기가 전달률 T(ω)입니다. Day 05의 진폭비 X/X_st와 비슷하지만 결정적 차이: 전달률은 '기초 가진(base excitation)' 문제. r=√2가 마법의 숫자: 이 점에서 모든 감쇠비의 전달률 곡선이 T=1로 교차합니다. r>√2 영역에서만 진정한 방진이 일어납니다.

DERIVATION

수식 한 줄씩 유도

STEP 1 — 기초 가진 모델 설정
mx¨+c(x˙−zr′)+k(x−zr)=0m\ddot{x} + c(\dot{x}-z_r') + k(x-z_r) = 0mx¨+c(x˙−zr′​)+k(x−zr​)=0

이 항의 의미: 노면 z_r(t)가 기초(base)로 작용. 상대변위(x−z_r)에 스프링력, 상대속도(x'−z_r')에 댐퍼력. Day 08 차체 방정식과 동일 구조.

부호/변환 이유: 우변=0: 외력이 아닌 기초 운동으로 전달. 외력 가진(Day 05)과 구분되는 '기초 가진' 문제.

이전 단계와의 연결: Day 08 STEP 2: mₛ·x_s'' + c_s(x_s'−x_u') + kₛ(x_s−x_u) = 0에서 x_u → z_r로 치환.

↓
STEP 2 — 전달률 공식 유도
T(ω)=∣X/Zr∣=(k2+(cω2)/((k−mω2)2+(cω)2))=(1+(2ζr2)/((1−r2)2+(2ζr)2))T(\omega) = |X/Z_r| = \sqrt{(k^2+(c\omega}^2) / ((k-m\omega^2)^2+(c\omega)^2)) = \sqrt{(1+(2\zetar}^2) / ((1-r^2)^2+(2\zetar)^2))T(ω)=∣X/Zr​∣=(k2+(cω​2)/((k−mω2)2+(cω)2))=(1+(2ζr​2)/((1−r2)2+(2ζr)2))

이 항의 의미: x=Xe^(jωt), z_r=Z_r·e^(jωt) 대입 후 |X/Z_r| 계산. 분자에 (2ζr)²이 추가된 것이 Day 05 진폭비와의 차이.

부호/변환 이유: 분자의 (2ζr)²: 댐퍼를 통한 전달 경로. 고주파에서 댐퍼가 오히려 진동을 전달하는 원인.

이전 단계와의 연결: Day 05: X/X_st = 1/√((1−r²)²+(2ζr)²). 전달률은 분자에 √(1+(2ζr)²)가 곱해진 형태.

↓
STEP 3 — √2 교차점과 방진 조건
r=2에서T=1(모든ζ에 대해)r>2:T<1(방진 영역)r<2:T>1(증폭 영역)r = \sqrt{2}\text{에서} T = 1 (\text{모든} \zeta\text{에 대해}) r > \sqrt{2}: T < 1 (\text{방진 영역}) r < \sqrt{2}: T > 1 (\text{증폭 영역})r=2​에서T=1(모든ζ에 대해)r>2​:T<1(방진 영역)r<2​:T>1(증폭 영역)

이 항의 의미: T=1 조건을 풀면 분자=분모, 정리하면 r²=2 → r=√2. 이 점에서 감쇠비에 무관하게 T=1.

부호/변환 이유: 놀라운 사실: r>√2 영역에서 감쇠를 늘리면(ζ↑) 오히려 T가 증가! 댐퍼가 진동을 전달하는 통로가 되기 때문.

이전 단계와의 연결: Day 05의 공진 피크와 달리, 전달률은 고주파에서 '감쇠의 역효과'가 존재.

심화 학습

전달률 그래프에서 발견되는 서스펜션 설계의 근본적 딜레마:

감쇠 증가(ζ↑)의 효과: ✓ 공진 피크 감소 (좋음) — 방지턱 넘을 때 출렁임 감소 ✗ 고주파 방진 악화 (나쁨) — 미세 요철의 진동이 더 전달됨

감쇠 감소(ζ↓)의 효과: ✗ 공진 피크 증가 (나쁨) — 큰 요철에서 심하게 출렁 ✓ 고주파 방진 개선 (좋음) — 부드러운 주행감

해결책: 1. 가변 감쇠 서스펜션(CDC): 주파수/상황에 따라 ζ 자동 조절 2. 비선형 감쇠: 바운드(압축)와 리바운드(신장) 감쇠비를 다르게 3. 부시 댐핑: 서스펜션 부시의 고주파 감쇠 최적화

OEM에서 CDC 쇼크를 사용한다면, 이 딜레마를 시스템적으로 해결하는 것.

■ 전달률 공식에서 r = √2 대입

T=(1+(2ζ⋅22)/((1−2)2+(2ζ⋅2)2))T = \sqrt{(1+(2\zeta\cdot \sqrt{2}}^2) / ((1-2)^2+(2\zeta\cdot \sqrt{2})^2))T=(1+(2ζ⋅2​​2)/((1−2)2+(2ζ⋅2​)2))
=(1+8ζ2/(1+8ζ2))=1= \sqrt{(1+8\zeta^2} / (1+8\zeta^2)) = 1=(1+8ζ2​/(1+8ζ2))=1

분자와 분모가 정확히 같아짐! → T = 1 (ζ에 무관)

■ 의미

r < √2: 감쇠를 늘리면 공진 피크가 낮아짐 (좋음)

r > √2: 감쇠를 늘리면 오히려 전달률이 증가 (나쁨!)

■ 서스펜션 딜레마

공진(r≈1) 제어: 감쇠↑ 필요 / 고주파 절연(r>√2): 감쇠↓ 필요

→ 이것이 세미액티브 댐퍼(CDC)의 존재 이유!

■ 승차감(Comfort) 최적화

낮은 kₛ → 낮은 ωₙ → 방진 시작점이 낮아짐 → 더 넓은 절연 영역

하지만 스프링이 너무 소프트하면 롤/피치 과대 → 핸들링 악화

■ 핸들링(Handling) 최적화

높은 kₛ → 롤 억제 → 코너링 안정성 ↑

하지만 절연 영역이 좁아짐 → 승차감 악화

■ 레퍼런스 차량 타협점

fₙ = 1.28 Hz는 승용차 타겟 1.0~1.5 Hz 범위 내

추가 롤 제어는 안티롤바로 분담 → 스프링은 승차감 위주 설계

EXAMPLE

실차 적용

🚗 레퍼런스 차량 전달률 분석

fₙ = 1.28 Hz

ζ = 0.512

√2·fₙ = 1.81 Hz

방진 시작 주파수

공진 피크 전달률:

T_peak ≈ 1/(2ζ√(1−ζ²)) = 1/(2×0.512×0.859) = 1.14 (매우 낮음!)

10 Hz 전달률:

r = 10/1.28 = 7.81 → T ≈ 0.016 (98.4% 절연!) ✓

QUESTION 01 — 수식 빈칸 채우기

전달률 T=1이 되는 주파수비 r은?

A. 1
B. √2 ≈ 1.414
C. 2
D. π
📝 풀이: T=1이 되는 주파수비
1T(r) = √((1+(2ζr)²)/((1−r²)²+(2ζr)²))
2T=1 → 분자 = 분모
31+(2ζr)² = (1−r²)²+(2ζr)² → 1 = (1−r²)²
41−r² = ±1 → r² = 0 (trivial) or r² = 2
✓r = √2 ≈ 1.414 → 정답: B
💡 이 점은 ζ값에 무관 — 모든 전달률 곡선이 이 점을 통과!
QUESTION 02 — 물리적 의미 연결

r>√2 영역에서 감쇠를 늘리면 전달률은 어떻게 되는가?

A. 감소 (더 좋은 방진)
B. 증가 (방진 악화)
C. 변화 없음
D. 진동 정지
📝 풀이: r>√2에서 감쇠 증가 효과
1r > √2: 절연(방진) 영역
2이 영역에서 감쇠↑ → 댐퍼가 '다리' 역할 → 진동 전달 경로 추가
✓전달률 증가 (방진 악화) → 정답: B
💡 역설적! 감쇠는 공진에서는 좋지만 고주파에서는 해로움 → CDC의 존재 이유
QUESTION 03 — Simpack 적용

A. 시스템 불안정
