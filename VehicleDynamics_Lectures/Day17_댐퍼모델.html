왜 이 수식이 필요한가?

댐퍼(쇽업쇼버)는 운동 에너지를 열로 변환하는 장치다. 오일이 좁은 밸브를 통과할 때 저항이 생기고, 이 저항력이 F = c·v (감쇠력 = 감쇠계수 × 속도)다. 실제 댐퍼는 바운드(압축)와 리바운드(인장)에서 감쇠력이 다르고, 고속에서는 블로오프 밸브가 열려 감쇠력 증가가 둔화된다.

DERIVATION

수식 한 줄씩 유도

Step 1: 선형 점성 감쇠 모델
Fd=c⋅x˙F_d = c \cdot \dot{x}Fd​=c⋅x˙

이 항의 의미: 감쇠력 F_d는 피스톤 속도 ẋ에 비례한다. c는 감쇠계수[N·s/m].

부호/변환 이유: ẋ > 0(리바운드/인장)이면 F > 0(인장력), ẋ < 0(바운드/압축)이면 F < 0(압축력). 항상 운동 방향을 저항한다.

이전 단계와의 연결: Day 03에서 감쇠비 ζ = c/(2√(km))를 정의했다. 여기서 c의 물리적 실체를 다룬다.

↓
Step 2: 바운드/리바운드 비대칭 모델
Fd=cb⋅x˙(x˙<0,바운드)∣Fd=cr⋅x˙(x˙>0,리바운드)F_d = c_b \cdot \dot{x} (\dot{x} < 0, \text{바운드}) | F_d = c_r \cdot \dot{x} (\dot{x} > 0, \text{리바운드})Fd​=cb​⋅x˙(x˙<0,바운드)∣Fd​=cr​⋅x˙(x˙>0,리바운드)

이 항의 의미: 리바운드 감쇠 c_r이 바운드 감쇠 c_b보다 2~4배 크다. 이는 범프 입력 시 빠르게 흡수(부드러운 바운드)하고, 스프링 복원 시 천천히 돌아오게(강한 리바운드) 하기 위함이다.

부호/변환 이유: c_r/c_b ≈ 2~4. 이 비율이 승차감과 접지력의 균형을 결정한다.

이전 단계와의 연결: Day 03의 감쇠비 ζ는 선형 모델 기준이었다. 비대칭 모델에서는 등가 감쇠비를 에너지 등가법으로 구한다.

↓
Step 3: 블로오프 밸브 (Blow-off) 효과
F=c1⋅x˙(∣x˙∣≤vb),F=c1⋅vb+c2⋅(x˙−vb)(∣x˙∣>vb),c2<c1F = c_1\cdot \dot{x} (|\dot{x}| \leq v_b), F = c_1\cdot v_b + c_2\cdot (\dot{x} - v_b) (|\dot{x}| > v_b), c_2 < c_1F=c1​⋅x˙(∣x˙∣≤vb​),F=c1​⋅vb​+c2​⋅(x˙−vb​)(∣x˙∣>vb​),c2​<c1​

이 항의 의미: 저속 구간에서는 높은 감쇠 c₁으로 제어성을 확보하고, 고속(v_b 이상)에서는 낮은 c₂로 전환되어 충격을 완화한다.

부호/변환 이유: v_b는 블로오프 속도(0.1~0.3 m/s). 이 지점에서 F-v 기울기가 꺾인다.

이전 단계와의 연결: Day 05의 강제진동에서 감쇠는 일정하다고 가정했다. 실차에서는 이 비선형이 핵심이다.

Step 4: Maxwell 모델 (댐퍼 + 직렬 스프링)
F=c⋅(x˙1−x˙2),x1−x2=F/krod+∫F/cdtF = c \cdot (\dot{x}_1 - \dot{x}_2), x_1 - x_2 = F/k_rod + \intF/c dtF=c⋅(x˙1​−x˙2​),x1​−x2​=F/kr​od+∫F/cdt

이 항의 의미: 실제 쇽업쇼버의 로드(rod)에는 탄성이 있다. Maxwell 모델은 스프링과 댐퍼를 직렬로 연결하여 고주파에서의 댐퍼 강성 효과를 표현한다.

부호/변환 이유: 주파수가 높아지면 댐퍼 임피던스가 증가하여 마치 강체처럼 행동 → 고주파 전달 증가.

이전 단계와의 연결: Day 16의 직렬 스프링 합성 개념이 여기서 '스프링 + 댐퍼 직렬'로 확장된다.

📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① 점성 감쇠(Viscous Damping) 기본

$$F_d = c\dot{x}$$

이상적 선형 모델. 실제 유압 댐퍼는 밸브 특성에 따라 비대칭·비선형

② 비대칭 댐핑: 리바운드/컴프레션

$$F_d = \begin{cases} c_{comp}\,\dot{x} & \dot{x} < 0 \text{ (compression)} \\ c_{reb}\,\dot{x} & \dot{x} > 0 \text{ (rebound)} \end{cases}$$
$$\text{일반적 비율: } c_{reb}/c_{comp} \approx 2.5 \sim 4.0$$

설계 이유: 리바운드 감쇠↑로 차체 복원 제어, 컴프레션 감쇠↓로 충격 흡수 (승차감 확보)

③ 디그레시브(Degressive) 특성

$$F_d = c_0 \dot{x} \cdot \frac{1}{1 + |\dot{x}|/v_0}$$
$$\text{저속: } F \approx c_0 \dot{x} \quad \text{고속: } F \approx c_0 v_0 \text{ (포화)}$$

블로우오프 밸브가 열리면서 고속 영역 감쇠력 포화. 이점: 소진폭 진동에는 충분한 감쇠, 대진폭 충격에는 과도한 힘 방지

④ 등가 감쇠 계수

$$c_{eq} = \frac{E_{dissipated}}{\pi \omega X_0^2}$$
$$E_{dissipated} = \oint F_d\,dx = \text{F-V 히스테리시스 루프 면적}$$

비선형 댐퍼의 등가 선형화. 주의: 등가 감쇠는 진폭과 주파수에 따라 변함 → 작동점 명시 필수

심화 학습

CDC(Continuous Damping Control)는 솔레노이드 밸브로 오리피스 면적을 실시간 조절한다. MR(Magneto-Rheological) 댐퍼는 자기장으로 MR 유체의 점도를 변화시킨다. 둘 다 c(t)를 제어 입력으로 사용하며, Simpack에서는 Co-simulation 또는 User Subroutine으로 구현한다. Skyhook 제어: c = c_max if ẋ_body · (ẋ_body - ẋ_wheel) > 0, else c_min.

■ 선형 댐퍼의 문제

F = cv → 큰 속도 입력(범프) 시 힘이 과도하게 커짐

→ 탑승자에게 큰 충격(harsh impact) 전달

■ 블로오프(Blow-off) 밸브

설정 속도(v_blow) 이상에서 바이패스 유로 개방

v < v_blow: F = c₁·v (높은 감쇠 — 바디 제어)

v > v_blow: F = c₁·v_blow + c₂·(v−v_blow), c₂ << c₁

→ 고속 입력에서 감쇠력 증가율이 둔화 → 충격 완화

■ 레퍼런스 차량 댐퍼 설정

바운드 블로오프: ~0.3 m/s에서 기울기 변화

리바운드 블로오프: ~0.15 m/s에서 기울기 변화

평균 등가 감쇠:

c_eq ≈ (c_b + c_r)/2 = 2200 → ζ ≈ c_eq/(2√(kₛ·mₛ)) ≈ 0.40
