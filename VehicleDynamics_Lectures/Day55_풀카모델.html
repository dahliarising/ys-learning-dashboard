풀카 모델
Full-Vehicle Model & 7-DOF Dynamics

q=[z,θ,ϕ,zFL,zFR,zRL,zRR]T
7-DOF 모델 4-Post Rig Roll-Pitch 커플링
INTUITION

왜 풀카 모델인가?

Quarter-car나 Half-car 모델은 수직 진동이나 좌우 대칭 모드만 분석합니다. 하지만 실제 주행에서는 좌·우 노면이 다르고, 앞·뒤 하중이 비대칭이며, Roll(횡 기울기)과 Pitch(종 기울기)가 동시에 발생합니다.

풀카 모델 = 3 Body DOF + 4 Wheel DOF = 7-DOF
차체(Sprung Mass): Bounce(z), Roll(ϕ), Pitch($\theta$)
바퀴(Unsprung Mass): zFL, zFR, zRL, zRR

이 모델 하나로 승차감, 조종안정성, 4-Post Rig 시뮬레이션까지 모두 커버할 수 있습니다.

핵심 통찰: 풀카 모델은 단순히 "자유도를 늘린 것"이 아니라, Roll-Pitch 커플링이라는 새로운 물리 현상을 포착합니다. 좌회전 시 Roll이 발생하면 동시에 Pitch도 변하는데, 이는 Quarter-car에서는 절대 보이지 않는 현상입니다.

DERIVATION

7-DOF 운동방정식 유도

STEP 1
좌표 정의 및 기하학적 관계

차체 중심(CG)의 수직 변위 z, Roll각 ϕ, Pitch각 $\theta$를 정의합니다. 각 코너의 차체 변위는:

zs,FL=z−aθ+tfϕ
zs,FR=z−aθ−tfϕ

여기서 a = CG에서 전축까지 거리, b = CG에서 후축까지 거리, tf = 전륜 반 트레드, tr = 후륜 반 트레드입니다.

STEP 2
Bounce 운동방정식 (z 방향)

뉴턴의 제2법칙을 차체 수직 방향에 적용하면:

Msz̈=−∑ki(zs,i−zu,i)−∑ci(zs,i̇−zu,i̇)

4개 코너(FL, FR, RL, RR) 스프링·댐퍼 힘의 합입니다.

STEP 3
Roll 운동방정식 (ϕ 축)

Roll축 모멘트 균형:

Ixxϕ̈−Ixzθ̈=∑Mroll,i

여기서 Ixz 항이 Roll-Pitch 커플링을 만듭니다. Ixz ≠ 0이면 Roll 가속이 Pitch에 영향을 줍니다.

STEP 4
Pitch 운동방정식 ($\theta$ 축) 및 Wheel EOM

Pitch 모멘트:

Iyyθ̈−Ixzϕ̈=∑Mpitch,i

4개 Unsprung Mass 각각의 수직 운동:

mu,izu,ï=ki(zs,i−zu,i)+ci(zs,i̇−zu,i̇)−kt,i(zu,i−zr,i)

여기서 kt,i는 타이어 강성, zr,i는 노면 입력입니다.

📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① 7-DOF 풀카 상태벡터

$$\mathbf{x} = [z_s, \dot{z}_s, \phi, \dot{\phi}, \theta, \dot{\theta}, z_{uf}, \dot{z}_{uf}, z_{ur}, \dot{z}_{ur}, ...]^T$$
$$\text{7-DOF: bounce}(z_s), \text{roll}(\phi), \text{pitch}(\theta), 4 \times \text{wheel}(z_{ui})$$

풀카 = 하프카(전/후) + 롤 자유도. 실무에서는 14-DOF(조향 + 엔진 마운트)까지 확장

② 바운스-피치 커플링

$$m\ddot{z}_s = -(k_f + k_r)z_s - (c_f + c_r)\dot{z}_s + (k_r l_r - k_f l_f)\theta$$
$$I_y\ddot{\theta} = (k_r l_r - k_f l_f)z_s - (k_f l_f^2 + k_r l_r^2)\theta$$
$$\text{디커플링 조건: } k_f l_f = k_r l_r \;\Rightarrow\; \text{바운스-피치 독립}$$

디커플링 조건이 만족되면 바운스/피치 모드가 분리 $\rightarrow$ 각각 독립적으로 튜닝 가능. 실무 목표!

③ 모드 주파수 설계 목표

$$f_{bounce} \approx 1.0\sim1.3 \;\text{Hz}, \quad f_{pitch} \approx 1.2\sim1.5 \;\text{Hz}$$
$$f_{roll} \approx 1.5\sim2.0 \;\text{Hz}$$
$$\text{비율: } f_{pitch}/f_{bounce} \approx 1.05\sim1.20 \;\text{(flat ride 조건)}$$

Flat Ride: 피치 주파수 > 바운스 주파수 $\rightarrow$ 노면 입력 시 앞/뒤 진동 위상이 맞아 '떠가는' 느낌 감소

④ 상태공간 표현

$$\dot{\mathbf{x}} = \mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{u}$$
$$\mathbf{y} = \mathbf{C}\mathbf{x} + \mathbf{D}\mathbf{u}$$
$$\mathbf{u} = [z_{road,fl}, z_{road,fr}, z_{road,rl}, z_{road,rr}]^T$$

MATLAB/Simulink에서 ss() 함수로 직접 구현. 고유값 분석으로 모드 주파수/감쇠비 즉시 확인

SIMPACK

SIMPACK 풀카 모델 구축 워크플로

01
차체 Body 정의

SIMPACK Body 요소로 Sprung Mass 생성: 질량(Ms), CG 위치, 관성 텐서(Ixx, Iyy, Izz, Ixz) 입력. Body $\rightarrow$ Properties $\rightarrow$ Inertia $\rightarrow$ Manual에서 6개 관성모멘트 성분을 설정합니다.

02
서스펜션 연결

4개 코너에 Force Element $\rightarrow$ Spring-Damper Parallel 배치. 전륜/후륜 별도 강성·감쇠 값 적용. ARB는 Force Element $\rightarrow$ Torsion Spring으로 좌·우 바퀴 상대 변위에 반력을 줍니다.

03
타이어 모델 연결

Tire $\rightarrow$ SIMPACK Road에서 Belgian Block, ISO Class C 등 노면 프로파일 정의. 각 Unsprung Mass에 Point-Follower 또는 Pac2002 타이어 모델을 연결합니다.

04
4-Post Rig 가진 설정

4개 독립 가진(zr,FL~zr,RR)을 시간 이력 또는 PSD 입력으로 정의. Excitation $\rightarrow$ Time History에서 각 코너별 노면 프로파일을 설정하여 실제 시험과 동일한 조건을 재현합니다.

SIMPACK 7-DOF 모델 설정 스크립트
! === 7-DOF Full-Vehicle Model Setup ===
! Body: Sprung Mass
$body sprung_mass
  mass = 1400.0          ! [kg]
  Ixx  = 600.0           ! [kg*m^2] Roll inertia
  Iyy  = 2200.0          ! [kg*m^2] Pitch inertia
  Izz  = 2500.0          ! [kg*m^2] Yaw inertia
  Ixz  = 120.0           ! [kg*m^2] Roll-Pitch coupling

! Suspension corners
$force_element spring_damper_FL
  type     = parallel_spring_damper
  stiffness = 25000.0    ! [N/m]
  damping   = 2000.0     ! [Ns/m]
  body1    = sprung_mass
  body2    = unsprung_FL
  marker1  = m_susp_FL   ! (-a, +tf, 0)
  marker2  = m_wheel_FL

! Anti-Roll Bar (Front)
$force_element arb_front
  type     = torsion_spring
  stiffness = 28000.0    ! [Nm/rad]
  body_L   = unsprung_FL
  body_R   = unsprung_FR

! 4-Post Rig Excitation
$excitation four_post_FL
  type   = time_history
  file   = "road_profile_FL.dat"
  column = 2
EXAMPLE

실전 예제: 4-Post Rig 모드 분리 분석

문제: 중형 세단이 4-Post Rig에서 0.5~20Hz 랜덤 가진을 받을 때, Bounce/Roll/Pitch/Warp 각 모드의 공진 주파수와 감쇠비를 구하라.
풀이 1
Bounce 모드 고유진동수

4개 코너 스프링이 병렬 연결:

fbounce=(1/2π)(2kf+2kr)/Ms

= (1/2$\pi$) $\sqrt{(2 \times 25000 + 2 \times 22000}$ / 1400) = (1/2$\pi$) $\sqrt{94000/1400}$ = (1/2$\pi$) $\sqrt{67.14}$ = 1.30 Hz

풀이 2
Roll 모드 고유진동수

Roll 유효 강성에는 스프링과 ARB 모두 기여합니다:

Kroll,eff = 2kf·tf² + 2kr·tr² + kARB,f + kARB,r

= $2 \times 2$5000$\times$0.78² + $2 \times 2$2000$\times$0.77² + 28000 + 15000

= 30,420 + 26,103 + 43,000 = 99,523 Nm/rad

froll = (1/2$\pi$)$\sqrt{Kroll,eff/Ixx}$ = (1/2$\pi$)$\sqrt{99523/600}$ = 2.05 Hz

풀이 3
Pitch 모드 고유진동수

Kpitch,eff = 2kf·a² + 2kr·b²

= $2 \times 2$5000$\times$1.15² + $2 \times 2$2000$\times$1.55²

= 66,125 + 105,710 = 171,835 Nm/rad

fpitch = (1/2$\pi$)$\sqrt{171835/2200}$ = (1/2$\pi$)$\sqrt{78.11}$ = 1.41 Hz

풀이 4
Bounce 감쇠비

$\zeta$bounce = Σc / (2$\sqrt{Σk · Ms}$)

= ($2 \times 2$000 + 2$\times$1800) / (2$\sqrt{94000 \times 1400}$)

= 7600 / (2 $\times$ 11,467.8) = 7600 / 22,935.6 = 0.331 (33.1%)

Q1. 7-DOF 풀카 모델에서 Roll-Pitch 커플링이 발생하는 근본 원인은?

① 전·후 스프링 강성이 다르기 때문
② 전·후 트레드가 다르기 때문
③ 관성 곱(Product of Inertia) Ixz ≠ 0이기 때문
④ 타이어 강성의 비선형성 때문
정답: ③ Ixz(관성 곱)가 0이 아니면 Roll 가속도가 Pitch 운동방정식에 나타나고, 그 역도 성립합니다. 이것이 바로 커플링의 물리적 원인입니다.

Q2. 4-Post Rig에서 Warp 모드 가진을 만들기 위한 4개 액추에이터 위상 조합은?

① FL↑ FR↑ RL↑ RR↑ (모두 동위상)
② FL↑ FR↓ RL↓ RR↑ (대각선 동위상)
③ FL↑ FR↓ RL↑ RR↓ (좌우 반대)
④ FL↑ FR↑ RL↓ RR↓ (전후 반대)
정답: ② Warp = (FL - FR - RL + RR)/4. 대각선 코너가 같은 방향으로, 인접 코너는 반대 방향으로 움직이는 "비틀림" 모드입니다. 차체의 비틀림 강성을 평가하는 핵심 모드입니다.

DEEP DIVE

심화 학습

7-DOF에 Yaw, Lateral, Longitudinal, 그리고 4바퀴 회전($\omega$)을 추가하면 14-DOF가 됩니다. 이 모델은 조향 응답과 제동 안정성까지 해석할 수 있어 실제 차량 개발에서 가장 많이 사용됩니다. 특히 ESC(Electronic Stability Control) 알고리즘 개발에 필수적인 모델입니다.

풀카 DOF: 바디 $6$ DOF ($x, y, z, \phi, \theta, \psi$) + 휠 $4 \times 1$ DOF (바운스) + 스티어링 $= 11\text{~}14$ DOF

질량 행렬: $[M]_{14\times14}$,   강성 행렬: $[K]_{14\times14} \rightarrow$ 고유치 $14$개

추가 자유도: Yaw(ψ), Lateral(y), Longitudinal(x), $\omega$FL, $\omega$FR, $\omega$RL, $\omega$RR

실제 서스펜션에서 스프링은 Progressive Rate(하중에 따라 강성 증가), 댐퍼는 속도 의존 비선형 특성을 가집니다. 7-DOF 모델에 이 비선형성을 추가하면:

비선형 스프링: $F_s = k_1 x + k_3 x^3 \rightarrow$ 진폭 의존 고유진동수: $\omega_n(A) = \omega_0\sqrt{1 + \frac{3k_3 A^2}{4k_1}}$

비대칭 댐퍼: $c(\dot{x}) = \begin{cases} c_b & \dot{x} \gt 0 \\ c_r & \dot{x} \lt 0 \end{cases}$ → $c_r/c_b \approx 2\text{~}3$

• Bump Stop: 큰 변위에서 급격한 강성 증가 $\rightarrow$ Jounce/Rebound 리미트

• 댐퍼 Blow-off: 고속 영역에서 밸브가 열려 감쇠력 포화

• 부시 히스테리시스: 고무 부시의 에너지 흡수로 추가 감쇠 발생

이러한 비선형 효과는 승차감 설계의 핵심이며, 선형 모델과 실차 사이의 간극을 줄이는 열쇠입니다.

풀카 모델의 가장 큰 가치는 Ride(승차감)와 Handling(조종성) 간의 트레이드오프를 정량적으로 분석할 수 있다는 점입니다.

승차감 지표: $a_w = \sqrt{\int W^2(f) \cdot S_a(f) \, df} \rightarrow$ ISO 2631

핸들링 지표: $\Delta F_z/F_z$ (접지 하중 변동률) → < 0.3$이면 양호

르노코리아 실무 Tip: 풀카 모델에서 Ride와 Handling을 동시에 평가하려면 Sine Sweep + Step Steer를 순차 실행합니다.

전략 1: 전후 Roll 강성 배분
전륜 ARB 강성↑ $\rightarrow$ Understeer 증가, Roll 감소. 후륜 ARB 강성↑ $\rightarrow$ Oversteer 경향, 후축 Ride 악화. 일반적으로 전:후 = 60:40 ~ 65:35 배분.

전략 2: Bounce/Pitch 주파수 분리
fpitch/fbounce $\approx$ 1.0~1.2일 때 최적. 비율이 1보다 크면 "Flat Ride"(갤럽 없는 주행감), 1보다 작으면 "Pitching" 발생.

전략 3: 감쇠비 최적화
Bounce $\zeta$ = 0.2~0.4, Roll $\zeta$ = 0.3~0.5가 일반적. 세미 액티브 댐퍼(CDC)를 사용하면 Comfort/Sport 모드 전환이 가능합니다.

실제 4-Post Rig 시험과 7-DOF 시뮬레이션을 비교하면 일반적으로:

4-Post 입력: $z_i(t) = A\sin(2\pi f t + \phi_i)$,   $i = FL, FR, RL, RR$

상관성: $\frac{|H_{sim}(f) - H_{test}(f)|}{H_{test}(f)} \lt 10\%$ (주파수 $1\text{~}30$ Hz 대역)

• Bounce 주파수: $\pm$3% 이내 일치 (가장 정확)

• Roll 주파수: $\pm$5% (ARB 부시 강성 불확실성)

• Pitch 주파수: $\pm$4% (CG 위치 정확도 의존)

• Warp 주파수: $\pm$10% (차체 비틀림 강성 미반영)

상관성을 높이려면: ① 정확한 관성 측정(K&C 시험), ② 비선형 부시 특성 반영, ③ 차체 유연성(FEM) 추가가 필요합니다.

7-DOF 풀카 모델은 디지털 트윈의 물리 코어 역할을 합니다. 실시간 센서 데이터(가속도계, 자이로, 서스펜션 스트로크)를 입력받아 칼만 필터로 상태를 추정하고, 머신러닝으로 비선형 파라미터를 온라인 업데이트합니다.

실시간 요건: $t_{compute} \lt \Delta t_{step}$ → ROM: $n_{mode} \approx 20\text{~}50$개로 축소

칼만 필터: $\hat{x}_{k+1} = A\hat{x}_k + Bu_k + K(y_k - C\hat{x}_k) \rightarrow$ 센서 데이터로 상태 보정

구현 파이프라인:

1. 7-DOF 물리 모델 $\rightarrow$ State-Space 표현 (A, B, C, D 행렬)

2. Extended Kalman Filter $\rightarrow$ 실시간 상태 추정

3. Neural Network $\rightarrow$ 비선형 댐퍼/부시 특성 학습

4. Predictive Control $\rightarrow$ 세미 액티브 댐퍼 최적 제어

이 파이프라인은 현재 BMW, Mercedes의 E-ABC, Audi의 Predictive Active Suspension에서 실제 구현되어 양산 적용되고 있습니다.

핵심 암기 포인트

7-DOF = 3 Body + 4 Wheel
차체: Bounce(z) + Roll(ϕ) + Pitch($\theta$) / 바퀴: zFL, zFR, zRL, zRR
Roll-Pitch 커플링 = Ixz
관성 곱이 0이 아니면 Roll↔Pitch가 연결됨. CG가 대각선에서 벗어날수록 커플링 증가.
4-Post Rig 모드 분해 공식
Bounce = (FL+FR+RL+RR)/4
Roll = (FL−FR+RL−RR)/4
Pitch = (FL+FR−RL−RR)/4
Warp = (FL−FR−RL+RR)/4
Flat Ride 조건
fpitch / fbounce $\approx$ 1.0~1.2 $\rightarrow$ 갤럽 없는 쾌적한 승차감. a/L과 kf/kr 비율로 조정.
실무 감쇠비 범위
Bounce $\zeta$ = 0.2~0.4 | Roll $\zeta$ = 0.3~0.5 | Pitch $\zeta$ = 0.25~0.45
