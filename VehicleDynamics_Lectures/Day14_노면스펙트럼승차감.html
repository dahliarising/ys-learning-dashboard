노면 스펙트럼·승차감 지표

Road Spectrum (PSD) & Ride Comfort Index (ISO 2631)

INTUITION

왜 이 수식이 필요한가?

💡 수식 유도 읽는 법 — 각 STEP의 파란색 라벨은 "왜 이 변환을 하는가"를 설명합니다. 수식만 보지 말고 동기 → 변환 → 의미 순서로 읽으세요. 눈으로 3번 읽는 것보다 손으로 1번 쓰는 것이 기억에 4배 효과적입니다.

지금까지 깔끔한 cos(ωt) 입력만 다뤘지만, 실제 노면은 무수히 많은 주파수가 섞인 랜덤 입력입니다. 이를 다루는 도구가 PSD(Power Spectral Density, 파워스펙트럼밀도). 노면 PSD는 ISO 8608로 표준화되어 있고, 이를 전달함수에 통과시키면 차체 응답 PSD를 얻습니다. 최종적으로 ISO 2631 가중함수를 적용하면 승차감 지표 a_w(weighted RMS acceleration)를 계산할 수 있습니다.

DERIVATION

수식 한 줄씩 유도

STEP 1 — 노면 PSD 모델 (ISO 8608)
Sz(n)=Sz(n0)×(n/n0)(−w)n0=0.1cycle/m,w≈2S_z(n) = S_z(n_0) \times (n/n_0)^(-w) n_0 = 0.1 cycle/m, w \approx 2Sz​(n)=Sz​(n0​)×(n/n0​)(−w)n0​=0.1cycle/m,w≈2

이 항의 의미: n은 공간주파수(cycle/m), S_z(n₀)는 기준 공간주파수에서의 PSD 값. 도로 등급(A~H)에 따라 S_z(n₀) 값이 다름.

부호/변환 이유: 지수 −w ≈ −2: 저주파(긴 파장)가 에너지의 대부분. 고주파(짧은 파장)는 에너지가 작음. 이것이 노면의 기본 특성.

이전 단계와의 연결: 공간주파수 n → 시간주파수 f = n × V (V=차속). 속도가 빠를수록 같은 요철이 높은 주파수 가진이 됨.

↓
STEP 2 — 응답 PSD 계산
Sa(f)=∣Hacc(j2πf)∣2×Sz,input(f)S_a(f) = |H_{acc}(j2\pi f)|^2 \times S_{z,input}(f)Sa​(f)=∣Hacc​(j2πf)∣2×Sz,input​(f)

이 항의 의미: 시스템의 출력 PSD = |전달함수|² × 입력 PSD. H_acc는 가속도 전달함수(= (j2πf)²·H(j2πf)). Day 12의 전달함수를 활용.

부호/변환 이유: 제곱 |H|²: PSD는 에너지 밀도이므로 진폭이 아닌 파워(진폭²)로 전달.

이전 단계와의 연결: Day 12: H(s) = (c_s·s+kₛ)/(mₛ·s²+c_s·s+kₛ). 여기에 s=j2πf, ×s²(가속도).

↓
STEP 3 — ISO 2631 가중 RMS 가속도
aw=∫0∞W2(f⋅Sa(f)df)a_w = \sqrt{\int_0^\infty W^2(f} \cdot S_a(f) df)aw​=∫0∞​W2(f​⋅Sa​(f)df)

이 항의 의미: W(f)는 ISO 2631의 주파수 가중함수. 인체가 민감한 4~8 Hz 대역에 가중치를 크게 부여. a_w가 승차감의 최종 숫자.

부호/변환 이유: 적분 → 전 주파수 대역의 에너지를 합산. 루트 → RMS(Root Mean Square). a_w < 0.315 m/s²이면 '쾌적'.

이전 단계와의 연결: Day 13의 전달률로 방진 성능을 평가했다면, 여기서는 실제 도로에서의 승차감 점수를 계산.

✓ 물리적 직관 체크: 이 수식에서 각 항의 물리적 의미를 손가락으로 짚으며 설명할 수 있나요? "이 항은 ___를 의미하고, 저 항은 ___의 영향"이라고 말해보세요.
↓
STEP 4 — 승차감 등급 판정 (ISO 2631)
aw<0.315:쾌적0.315 0.63:약간 불쾌0.63 1.0:불쾌>1.0:매우 불쾌a_w < 0.315: \text{쾌적} 0.315~0.63: \text{약간 불쾌} 0.63~1.0: \text{불쾌} > 1.0: \text{매우 불쾌}aw​<0.315:쾌적0.315 0.63:약간 불쾌0.63 1.0:불쾌>1.0:매우 불쾌

이 항의 의미: a_w 값으로 승차감을 객관적으로 등급화. 서스펜션 튜닝의 최종 목표: 타겟 도로에서 a_w를 허용 범위 이내로.

부호/변환 이유: 단위 m/s². 중력가속도 g ≈ 9.81 m/s² 대비 ~3%~10% 수준이 체감 영역.

이전 단계와의 연결: 이 지표가 K&C 시험, 실차 주행 평가, Simpack 가상 시험의 공통 평가 척도.

이 체계가 서스펜션 엔지니어의 핵심 워크플로우: 1. 차량 파라미터 설정 → 2. 모델 구축(쿼터카/하프카) → 3. 고유진동수 확인 4. 전달함수/전달률 분석 → 5. 실제 노면 가진 적용 → 6. 승차감 지표 평가

PART 3(Day 15~)에서는 서스펜션 하드웨어 구성요소(스프링, 댐퍼, 부시, 스태빌라이저)의 비선형 특성과 모델링을 배웁니다.

■ 노면 분류 체계

기준 주파수 n₀ = 0.1 cycle/m에서의 PSD 값으로 등급 결정:

A (매우 좋음): S(n₀) = 1×10⁻⁶ m³ — 신축 고속도로

B (좋음): 4×10⁻⁶ — 일반 고속도로

C (보통): 16×10⁻⁶ — 시내 도로

D (나쁨): 64×10⁻⁶ — 노후 도로

■ 실무 적용

OEM 승차감 평가: B~C 등급 노면 사용

Simpack Road Profile: ISO 8608 기반 랜덤 노면 자동 생성 가능

EXAMPLE

실차 적용

🚗 레퍼런스 차량 승차감 평가 시나리오

노면: ISO B등급

차속: 80 km/h (22.2 m/s)

평가 기준: ISO 2631

타겟: a_w < 0.4 m/s²

가진 주파수 범위:

n = 0.01~10 cycle/m → f = 0.22~222 Hz (@80km/h)

인체 민감 대역:

QUESTION 01 — 수식 빈칸 채우기

노면 PSD의 공간주파수 n을 시간주파수 f로 변환하는 식은 f = n × ___?

A. ωₙ
B. V (차속)
C. 2π
D. k/m
📝 풀이: 공간→시간 주파수 변환
1n: 공간주파수 [cycle/m] — 도로 위의 파형
2f: 시간주파수 [Hz] — 차에 전달되는 진동
3차가 V [m/s]로 달리면: 1초에 V미터 이동
✓f = n × V → 정답: B (차속)
💡 같은 노면도 속도가 다르면 다른 주파수로 느껴짐! 고속 → 고주파
QUESTION 02 — 물리적 의미 연결

ISO 2631에서 a_w = 0.5 m/s²의 승차감 등급은?

A. 쾌적
B. 약간 불쾌
C. 불쾌
D. 매우 불쾌
📝 풀이: ISO 2631 승차감 등급
1a_w < 0.315: 쾌적 (Not uncomfortable)
20.315 ≤ a_w < 0.63: 약간 불쾌 (A little uncomfortable)
✓a_w = 0.5 m/s² → 0.315~0.63 범위 → 약간 불쾌 → 정답: B
💡 승용차 타겟: a_w < 0.4 m/s² (쾌적~약간불쾌 경계)
QUESTION 03 — Simpack 적용

Simpack에서 노면 가진을 설정할 때 반드시 함께 지정해야 하는 파라미터는?

A. 온도
B. 차량 속도
C. 풍속
D. 노면 마찰계수

2f = n × V → 차속이 있어야 시간 영역 변환 가능
