고유치 문제

Generalized Eigenvalue Problem

$$[K - \omega^2M]{\varphi} = {0}$$
⏱ 포모도로 25분
📐 수식 4단계
⭐ 난이도 ★★★★☆
🔁 복습 D+1, D+3, D+7
INTUITION

왜 이 수식이 필요한가?

💡 수식 유도 읽는 법 — 각 STEP의 파란색 라벨은 "왜 이 변환을 하는가"를 설명합니다. 수식만 보지 말고 동기 $\rightarrow$ 변환 $\rightarrow$ 의미 순서로 읽으세요. 눈으로 3번 읽는 것보다 손으로 1번 쓰는 것이 기억에 4배 효과적입니다.

다자유도(MDOF) 시스템은 자유도 수만큼의 고유진동수와 모드 형상을 가진다. 고유치 문제는 '외력 없이 스스로 진동할 수 있는 주파수와 그때의 변형 패턴'을 찾는 것이다. 차체 모달 해석에서 bending, torsion 모드를 찾는 것이 바로 이 고유치 문제를 푸는 것이다.

DERIVATION

수식 한 줄씩 유도

Step 1: 자유진동 가정
$${x(t)} = {\varphi} \cdot e^{i\omega t} \rightarrow [K]{\varphi}\cdot e^{i\omega t} = \omega^2[M]{\varphi}\cdot e^{i\omega t}$$

이 항의 의미: 조화 자유진동 x = $\phi$·$e^{i\omega t}$ 를 운동방정식 [M]{ẍ}+[K]{x}={0}에 대입하면, 시간 항이 소거되고 진폭 벡터 {$\phi$}에 대한 대수 방정식이 남는다.

부호/변환 이유: ẍ = −$\omega^2$x이므로 [M]의 부호가 뒤집혀 [K] = $\omega^2$[M] 형태가 된다.

이전 단계와의 연결: Day 01의 1DOF 자유진동 m·ẍ + k·x = 0 $\rightarrow \omega^2$ = k/m의 MDOF 일반화다.

↓
Step 2: 일반화된 고유치 문제
$$[K - \omega^2M]{\varphi} = {0} \rightarrow \det [K - \omega^2M] = 0$$

이 항의 의미: 비자명 해($\phi$ ≠ 0)가 존재하려면 행렬 [K−$\omega^2$M]의 행렬식이 0이어야 한다. 이 조건이 특성 방정식이며, n차 다항식의 근이 n개 고유진동수를 준다.

부호/변환 이유: $\omega^2$ > 0 (양수) $\rightarrow$ 진동 모드. $\omega^2 = 0$ $\rightarrow$ 강체 모드(구속 조건 부족). $\omega^2$ < 0 $\rightarrow$ 좌굴(불안정).

이전 단계와의 연결: Day 09에서 쿼터카($2 \times 2$)의 특성 방정식을 풀었다. 여기서 $n \times n$ 으로 일반화한다.

↓
Step 3: 모드 형상 벡터의 직교성
$${\varphi_i}ᵀ [M] {\varphi_j} = 0 (i \neq j)$$

이 항의 의미: 서로 다른 모드 형상은 질량 매트릭스에 대해 직교한다. 이 성질이 모드 중첩법의 기반이다.

보충: 강성 직교성도 동시에 성립: $${\varphi_i}ᵀ [K] {\varphi_j} = 0 (i \neq j)$$ 모드 형상은 질량과 강성 매트릭스 모두에 대해 직교한다.

부호/변환 이유: 직교성 = '모드 간 에너지가 섞이지 않는다'. 각 모드를 독립적으로 분석할 수 있다.

이전 단계와의 연결: Day 10의 하프카에서 바운스/피치 모드가 특정 조건에서 디커플링(직교)되는 것을 보았다.

✓ 물리적 직관 체크: 이 수식에서 각 항의 물리적 의미를 손가락으로 짚으며 설명할 수 있나요? "이 항은 ___를 의미하고, 저 항은 ___의 영향"이라고 말해보세요.
↓
Step 4: 감쇠 시스템의 복소 고유치
$$λ = -\sigma \pm i\omega_d, \sigma = \zeta\omega_n, \omega_d = \omega_n\sqrt{1-\zeta^2}$$​

이 항의 의미: 감쇠가 있으면 고유치가 복소수가 된다. 실수부 $\sigma$ 는 감쇠율(exponential decay), 허수부 $\omega_d$ 는 감쇠 고유진동수.

부호/변환 이유: $\sigma$ > 0이면 안정(진동 감쇠), $\sigma$ < 0이면 불안정(진동 성장). 차량에서 $\sigma$ < 0은 shimmy, flutter 등의 자려진동.

이전 단계와의 연결: Day 04(부족감쇠 자유진동)에서 1DOF의 $\lambda$ = −$\zeta \omega_n$ $\pm$ i$\omega_d$ 를 유도했다. MDOF로 확장.

📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① 일반 고유치 문제 설정

$$[K]\{\phi\} = \lambda[M]\{\phi\}, \quad \lambda = \omega^2$$
$$\det([K] - \lambda[M]) = 0 \quad \Rightarrow \quad \text{특성방정식}$$

M, K가 대칭이고 M이 양정치이면 모든 고유치 $\lambda$ 는 실수이고 양수

② 2-DOF 예제 완전 전개

$$\begin{vmatrix} k_1+k_2-\omega^2 m_1 & -k_2 \\ -k_2 & k_2+k_3-\omega^2 m_2 \end{vmatrix} = 0$$
$$(k_1+k_2-\omega^2 m_1)(k_2+k_3-\omega^2 m_2) - k_2^2 = 0$$
$$m_1 m_2 \omega^4 - [(k_1+k_2)m_2 + (k_2+k_3)m_1]\omega^2 + [(k_1+k_2)(k_2+k_3) - k_2^2] = 0$$

2차 방정식 $\omega^2$ 에 대해 근의 공식 적용 $\rightarrow$ 두 고유진동수 $\omega_1$, $\omega_2$ 산출

③ 고유벡터(모드형상) 결정

$$\frac{\phi_2}{\phi_1}\bigg|_{\omega=\omega_i} = \frac{k_1+k_2-\omega_i^2 m_1}{k_2}$$

제1모드: 같은 방향 운동(in-phase). 제2모드: 반대 방향 운동(out-of-phase)

④ 직교성(Orthogonality) 조건

$$\{\phi\}_i^T [M] \{\phi\}_j = 0 \quad (i \neq j)$$
$$\{\phi\}_i^T [K] \{\phi\}_j = 0 \quad (i \neq j)$$

물리적 의미: 서로 다른 모드는 에너지 교환 없이 독립적으로 진동. 모드 중첩법의 이론적 기반!

심화 학습

MAC(Modal Assurance Criterion)은 두 모드 벡터의 코사인 유사도다: MAC_ij = $|\{\phi_A\}_i^T \{\phi_B\}_j|^2 / (\{\phi_A\}_i^T \{\phi_A\}_i)(\{\phi_B\}_j^T \{\phi_B\}_j)$. MAC = 1이면 완전 일치, 0이면 직교. 실무에서 Simpack 모델을 실차 해머링 데이터와 비교할 때 MAC 매트릭스를 작성하여 각 모드의 일치도를 평가한다. 대각 성분이 0.9 이상이면 모델이 신뢰할 수 있다고 판단한다.

고유치 문제: ([K] - \omega^2 [M])$\{\phi\} = 0 \rightarrow \det([K] - \omega^2 [M]) = 0$

MAC: $MAC_{ij} = \frac{|\{\phi_i^{test}\}^T \{\phi_j^{sim}\}|^2}{(\{\phi_i^{test}\}^T\{\phi_i^{test}\})(\{\phi_j^{sim}\}^T\{\phi_j^{sim}\})} \rightarrow MAC > 0.9$ 이면 일치

■ 복소 고유치: s = $\sigma \pm$ j$\omega_d$

감쇠 고유치: (\lambda [M] + [C])$\{\phi\} + [K]\{\phi\}/\lambda = 0 \rightarrow$ 복소 고유치 $\lambda = \sigma \pm j\omega_d$

$\sigma = -\zeta\omega_n$ (감쇠율),   $\omega_d = \omega_n\sqrt{1-\zeta^2}$ (감쇠 고유진동수)

르노코리아 실무 Tip: Simpack Eigenvalue Analysis 결과에서 Real Part가 양수이면 불안정 모드입니다. 반드시 모든 모드가 안정($\sigma$ < 0)인지 확인하세요.

$\sigma$ (실수부): 감쇠율 $\rightarrow$ |$\sigma$| 클수록 빨리 감쇠

$\omega_d$ (허수부): 감쇠 진동 주파수 $\rightarrow$ fd = $\omega_d$/(2$\pi$)

■ 안정성 판별

$\sigma$ < 0: 안정 (좌반면) ← 모든 서스펜션 모드는 여기!

$\sigma = 0$: 임계 (순수 진동, 비감쇠)

$\sigma$ > 0: 불안정 (발산) $\leftarrow$ 이것이 나오면 모델 오류!

■ 감쇠비 추출

$$\zeta = |\sigma|/\sqrt{\sigma^2 + \omega_d^2} = \cos (\theta), \theta = \text{실수축과의 각도}$$​=cos($\theta$),$\theta$=실수축과의 각도
