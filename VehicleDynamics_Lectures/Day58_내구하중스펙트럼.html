INTUITION
왜 하중 스펙트럼이 필요한가?

💡 설계 수명 = 하중 $\times$ 재료 $\times$ 형상

실도로 주행 하중은 완벽히 불규칙합니다. 신호등 정지, 포트홀 통과, 고속도로 순항, 벨기 블록(Belgian Block) — 모든 순간 서스펜션 부품에는 다른 크기와 방향의 하중이 가해집니다. 정적 해석 한 번으로는 피로 수명을 절대 예측할 수 없습니다.

하중 스펙트럼(Load Spectrum)은 이 불규칙 하중 이력을 체계적으로 분류하여 "어떤 크기의 하중이 몇 번 반복되는가"를 정량화한 것입니다. S-N 곡선(재료)과 응력 집중 계수(형상)를 결합하면 피로 수명 예측이 가능해집니다.

르노코리아의 서스펜션 개발 현장에서는 프랑스 시험장(CERAM, Mortefontaine)에서 취득한 24채널(4륜 $\times$ 6분력) 하중 데이터를 Rainflow 카운팅으로 변환하여 nCode 또는 FEMFAT에 입력하는 것이 표준 프로세스입니다.

0-5m하중 불규칙성
5-12mRainflow 원리

DERIVATION
레인플로우 카운팅과 Miner 법칙
STEP 01 — 하중 이력의 Peak-Valley 추출

원시 하중 시계열 F(t)에서 극값(peak/valley)만 추출합니다. 중간의 단조 구간은 피로에 기여하지 않으므로 제거합니다:

F(t) $\rightarrow$ {P₁, V₁, P₂, V₂, ...} where dF/dt = 0

의미: 24채널 데이터에서 초당 512 샘플 $\times$ 3600초 = 약 184만 포인트를 수천 개의 극값으로 압축합니다.

▼
STEP 02 — 레인플로우 카운팅 (Rainflow Counting)

ASTM E1049 표준에 따른 4-Point 알고리즘: 극값 시퀀스를 "비가 지붕에서 흘러내리듯" 순환하며 닫힌 히스테리시스 루프(사이클)를 추출합니다:

사이클 = {(S_a,i , S_m,i)} , i = 1, ..., k

S_a = 진폭(Amplitude) = (Peak - Valley) / 2

S_m = 평균(Mean) = (Peak + Valley) / 2

각 사이클은 진폭-평균 좌표계(Haigh Diagram)에서 한 점으로 표현됩니다. 이것이 바로 하중 스펙트럼입니다.

▼
STEP 03 — S-N 곡선 (Wöhler Curve)

재료의 피로 강도를 나타내는 S-N 곡선은 응력 진폭(S)과 파단까지의 반복 횟수(N)의 관계입니다:

Sm · N = C (Basquin 법칙)

m = S-N 기울기 (강재: 3~5, 알루미늄: 5~8)

C = 재료 상수

log-log 좌표에서 직선으로 표현되며, 각 응력 수준 S_a,i에서 허용 반복 횟수 N_i를 읽어옵니다.

▼
STEP 04 — Palmgren-Miner 누적 손상 법칙

각 응력 수준에서의 실제 반복 횟수(n_i)를 허용 반복 횟수(N_i)로 나누어 합산합니다. 이 합이 1에 도달하면 파괴가 발생합니다:

D = Σ(ni / Ni) ≤ 1

D = 1 $\rightarrow$ 이론적 파괴 시점

안전계수 적용: 실무에서는 D ≤ 0.3~0.5를 설계 기준으로 사용합니다 (산포, 환경, 부식 고려).

✅ Checkpoint: 하중 이력 $\rightarrow$ Peak/Valley $\rightarrow$ Rainflow 사이클 $\rightarrow$ S-N 곡선 참조 $\rightarrow$ Miner 누적 손상 D 계산. D < 1이면 설계 수명 만족!
📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① 레인플로우 카운팅 알고리즘

$$\text{Range: } \Delta\sigma_i = |\sigma_{peak} - \sigma_{valley}|$$
$$\text{Mean: } \sigma_{m,i} = \frac{\sigma_{peak} + \sigma_{valley}}{2}$$
$$\text{결과: 하중 히스토그램 } n_i(\Delta\sigma_i, \sigma_{m,i})$$

레인플로우: 히스테리시스 루프 기반 사이클 추출. ASTM E1049 표준. 피로 해석의 출발점

② Miner의 선형 피로 누적 법칙

$$D = \sum_{i=1}^{k} \frac{n_i}{N_i} \leq 1.0$$
$$N_i = N_0 \left(\frac{\sigma_0}{\Delta\sigma_i}\right)^m \quad (\text{S-N 곡선})$$

D = 1이면 파괴. 실무 안전계수: D $\leq$ 0.3~0.5 적용. m $\approx$ 3(강재), 5-8(알루미늄)

③ PSD $\rightarrow$ 시간 이력 변환

$$x(t) = \sum_{k=1}^{N} \sqrt{2 S_{xx}(f_k) \Delta f} \cdot \cos(2\pi f_k t + \phi_k)$$
$$\phi_k \sim \text{Uniform}(0, 2\pi) \quad (\text{랜덤 위상})$$

PSD 기반 가상 노면 생성 $\rightarrow$ MBD 시뮬레이션 입력. ISO 8608 노면 등급(A~H) 활용

④ 등가 손상 하중 (Equivalent Damage Load)

$$F_{eq} = \left(\frac{\sum n_i F_i^m}{N_{eq}}\right)^{1/m}$$
$$\text{내구 목표: } N_{target} = 300{,}000 \;\text{km} \times \text{road mix factor}$$

르노 내구 기준: 유럽 고객 도로 믹스(도심 40%, 국도 35%, 고속 25%) 기반 등가 하중 산출

QUIZ 58-1
Palmgren-Miner 법칙에서 D = 0.6의 물리적 의미는?
부품 강도의 60%까지 하중이 가해졌다
피로 수명의 60%가 남았다
이론적 피로 수명의 60%가 소진되었다
S-N 곡선의 기울기가 0.6이다
SOLUTION
개념 D = Σ(n_i/N_i)는 누적 손상 비율을 의미
핵심 D = 0.6 $\rightarrow$ 이론적 파괴(D=1) 대비 60% 수명 소진
실무 잔여 수명 = 1 - 0.6 = 0.4 (40% 남음)
핵심: Miner의 D는 선형 누적 손상 비율이며, D=1이 이론적 파괴 시점입니다. 실무에서는 안전계수를 적용하여 D $\leq$ 0.3~0.5를 기준으로 합니다.

DEEP DIVE
심화 학습

PSD(Power Spectral Density)는 정상(stationary) 랜덤 하중의 주파수 특성을 나타내며, 노면 프로파일 분류(ISO 8608)나 진동 스펙 정의에 사용됩니다. 반면 레인플로우는 비정상(non-stationary) 하중 포함 모든 시계열에서 피로 사이클을 추출합니다.

PSD 기반: $\sigma_{eq} = \sqrt{\int_0^\infty G_{\sigma\sigma}(f) \, df} \rightarrow$ 정상 랜덤 하중에 적합

레인플로우: $D = \sum_i \frac{n_i}{N_i}$ (Miner 법칙) $\rightarrow N_i = C / S_i^m$ (S-N 곡선)

고속도로 순항처럼 정상 랜덤에 가까운 경우 PSD 기반 Dirlik 법이 효율적이지만, 도심 + 고속 + 험로 복합 주행에서는 반드시 Rainflow를 사용해야 합니다. 르노의 내구 프로세스는 항상 Rainflow 기반입니다.

실제 서스펜션 부품은 단축이 아닌 다축 응력 상태에 놓입니다. 너클의 경우 수직 하중(Fz)과 횡방향 하중(Fy)이 동시에 작용하여 복합 응력이 발생합니다.

누적 손상: $D = \sum_{i=1}^{k} \frac{n_i}{N_i} \leq 1 \rightarrow$ Palmgren-Miner 선형 누적 손상

등가 하중: $F_{eq} = \left(\sum_i \frac{n_i F_i^m}{\sum n_i}\right)^{1/m} \rightarrow$ SN 기울기 $m$ 에 의존

Critical Plane Method는 모든 평면(각도)에서 전단 응력과 법선 응력의 조합을 검토하여, 가장 큰 손상을 일으키는 임계면(Critical Plane)을 찾습니다. FEMFAT에서는 이 방법이 기본 옵션입니다.

$\tau_a$,max + $k \cdot \sigma_n$,max $\leq$ f (Findley Criterion)

실차 30만km 내구를 실제로 달릴 수 없으므로, 등가 가속 시험을 설계합니다. 핵심 원리: 벨기 블록, 급커브, 과속방지턱 등 가혹 노면만 조합하여 단위 시간당 손상(D/hour)을 극대화합니다.

가속 계수: $AF = \frac{D_{accel}}{D_{field}} = \frac{t_{field}}{t_{accel}} \rightarrow$ 통상 $AF = 5\text{~}20$

시험 시간: $t_{test} = \frac{t_{target}}{AF} \rightarrow$ 10만 km $\rightarrow$ 가속 $5000\text{~}20000$ km 등가

르노코리아 실무 Tip: Simpack 가상 내구 시험에서 하중 스펙트럼을 추출한 뒤 레인플로우 카운팅으로 부품 수명을 예측합니다.

가속 배수 계산: 실도로 1km당 D_road와 시험장 1km당 D_test를 비교. 예: 벨기 블록 D_test/D_road $\approx$ 1,000~1,200배. 따라서 벨기 블록 300시간 $\approx$ 실도로 30만km.

르노/닛산 얼라이언스에서는 CERAM(프랑스) 및 Hokkaido PG(일본) 시험장의 표준 내구 코스를 운영합니다.

실측 데이터는 유한한 시간(보통 수시간~수일)이지만, 설계 수명은 10~15년입니다. 측정되지 않은 극한 하중을 통계적으로 추정하는 것이 하중 외삽입니다.

방법: Rainflow matrix에서 높은 진폭 영역을 극값 통계(EVS: Extreme Value Statistics)로 외삽합니다. Gumbel, Weibull, GEV 분포가 주로 사용되며, 1/10,000 확률 하중까지 추정합니다.

외삽 오류는 수명 예측에 수십 배 영향을 줄 수 있어, 최소 2~3개 독립 데이터셋으로 교차 검증하는 것이 필수입니다.

디지털 트윈(Digital Twin)은 실차 센서 데이터를 실시간으로 MBS 모델에 피드백하여, 물리 시험 없이 부품의 피로 수명을 모니터링하는 기술입니다.

프로세스: ① 텔레매틱스로 실주행 가속도/GPS 수집 $\rightarrow$ ② SIMPACK 모델로 하중 재구성(가상 센서) $\rightarrow$ ③ Rainflow + Miner로 실시간 D 누적 $\rightarrow$ ④ 잔여 수명 예측 및 정비 알림.

르노에서는 이를 "Virtual Proving Ground" 프로젝트로 추진 중이며, 향후 OTA 기반 예지 정비(Predictive Maintenance)로 확장될 전망입니다. 이 분야가 바로 MBD + 머신러닝의 융합 영역입니다.
