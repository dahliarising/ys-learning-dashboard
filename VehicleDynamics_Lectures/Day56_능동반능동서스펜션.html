Active & Semi-Active Suspension

INTUITION
왜 능동 서스펜션인가?

일반적인 패시브 서스펜션은 고정된 스프링 강성과 댐핑을 가집니다. 이는 승차감과 핸들링 사이에 근본적인 상충(trade-off)을 만듭니다:

  • 부드러운 승차감을 원하면: 낮은 댐핑 필요 $\rightarrow$ 몸체 진동 증가 (불편)
  • 좋은 핸들링을 원하면: 높은 댐핑 필요 $\rightarrow$ 승차감 악화 (거친 느낌)

반능동 서스펜션(Semi-Active)은 CDC(Continuous Damping Control)으로 이 상충을 해결합니다. 실시간으로 댐핑력을 변화시켜, 주행 상황에 따라 최적의 감쇠 특성을 제공합니다.

핵심 개념: 능동/반능동 서스펜션은 전자 제어와 소프트 액추에이터를 통해 승차감과 핸들링을 동시에 개선합니다.
DERIVATION
4단계 유도: 스카이훅 제어 $\rightarrow$ CDC 구현
STEP 1: 스카이훅 댐퍼 이론

스카이훅 댐퍼는 이상적인 능동 댐퍼입니다. 차체 속도 ẋ_body에 비례하는 댐핑력을 발생시킵니다:

F_sky = c_sky · ẋ_body

이를 통해 차체의 상하 진동을 최소화합니다 (노면 입력과 무관).

STEP 2: CDC 가변 감쇠력

현실적으로, CDC 댐퍼는 두 가지 고정 감쇠값을 전자적으로 빠르게 전환합니다:

c(t) ∈ {c_soft, c_stiff}

스카이훅 제어 알고리즘이 ẋ_body 부호를 판단하여 댐핑 모드를 실시간 전환합니다 (1~100ms).

STEP 3: 에어스프링 강성 변화

능동 시스템에서는 에어스프링 압력을 조절하여 공진주파수를 변경합니다:

k_air = $\gamma$ · P_air / V_ref

$\gamma$ = 단열 계수, P_air = 공기압, V_ref = 기준 용적

STEP 4: 드라이브 모드 맵핑

최종적으로, 드라이브 모드(Eco, Normal, Sport)에 따라 CDC 패라미터와 에어스프링 압력을 맵핑합니다:

(c_soft, c_stiff, P_air) = f(mode, v, a_body)

차량 속도, 차체 가속도를 실시간 입력으로 최적 파라미터를 산출합니다.

📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① 스카이훅 댐퍼 이상 제어법

$$F_{skyhook} = -c_{sky} \dot{z}_s$$
$$\text{반능동 구현: } c_{actual} = \begin{cases} c_{sky} \frac{\dot{z}_s}{\dot{z}_s - \dot{z}_u} & \text{if } \dot{z}_s(\dot{z}_s-\dot{z}_u) > 0 \\\\ c_{min} & \text{otherwise} \end{cases}$$

스카이훅: 가상 관성 기준으로 차체 감쇠. 에너지 소산만 가능(반능동) $\rightarrow$ 입력 불가능 영역은 c_min 적용

② 그라운드훅 (타이어 접지력 최적화)

$$F_{groundhook} = -c_{ground} \dot{z}_u$$
$$\text{반능동: } c = \begin{cases} c_{ground} \frac{\dot{z}_u}{\dot{z}_s - \dot{z}_u} & \text{if } \dot{z}_u(\dot{z}_s-\dot{z}_u) < 0 \\\\ c_{min} & \text{otherwise} \end{cases}$$

그라운드훅: 타이어 동하중 변동 최소화 $\rightarrow$ 접지력 유지. 절충안: $\alpha$·스카이훅 + (1-$\alpha$)·그라운드훅

③ CDC (Continuous Damping Control) MR 댐퍼

$$F_{MR} = F_{viscous} + F_{yield}\text{sgn}(\dot{x})$$
$$F_{yield} = \alpha_1 I^2 + \alpha_2 I \quad (\text{전류 } I \text{에 의해 제어})$$

MR 유체: 자기장으로 점성 실시간 변경. 응답시간 < 10ms. 르노 다카르 랠리 차량에도 적용

④ 능동 서스펜션 에너지 소비

$$P_{active} = F_{actuator} \times \dot{x}_{actuator}$$
$$\text{평균 소비: } P_{avg} \approx 1\sim5 \;\text{kW (풀 액티브)}$$
$$\text{반능동 소비: } P_{avg} \approx 10\sim50 \;\text{W}$$

풀 액티브 대비 반능동은 에너지 효율 100배 이상. EV에서는 에너지 소비가 주행거리 직결 $\rightarrow$ 반능동 선호

1. CDC Damper Component
Suspension $\rightarrow$ Damper (Force) $\rightarrow$ Nonlinear Characteristics
입력: 상대 속도 $\Delta v$ = ẋ_body − ẋ_wheel, 제어신호 u(소프트/스티프) 출력: 댐핑력 F_damp = c(u) · $\Delta v$
2. 스카이훅 제어 알고리즘 (User Subroutine)
Body Accel Input $\rightarrow$ SKY_CONTROL $\rightarrow$ Mode Selection Output
조건식: IF (ẋ_body · F_sky_ideal) > 0 THEN c_soft ELSE c_stiff 샘플링 주기: 10ms
3. 에어스프링 압력 모듈
Suspension $\rightarrow$ Spring (Air) $\rightarrow$ Pressure Control
입력: 주행 모드, 차량 속도 출력: 역동적 강성 k(t) = f(P_air, V_acc)
4. 시뮬레이션 결과 비교
Output $\rightarrow$ Charts: Acceleration / Displacement / Force
지표: 차체 가속도 RMS, 응답시간, 피크값 제어
⚠️ 주의: 스카이훅 제어는 이상적이므로 현실 CDC는 제한된 액추에이터 대역폭(5~50Hz)을 고려해야 합니다.
EXAMPLE
스카이훅 알고리즘으로 바디 진동 감소 계산

실제 차량 예시: Mid-size Sedan

기본 스펙:

차량 질량 m = 1500 kg, 스프링 강성 k = 25 kN/m, 패시브 댐핑 c_passive = 5 kN·s/m

노면 입력: 랜덤 범프 (5cm 높이, 0.2s 지속)

패시브 시스템 응답:

공진 주파수 f_n = $\sqrt{25e3/1500}$/(2$\pi$) = 0.65 Hz
감쇠비 $\zeta$ = 5e3/(2·$\sqrt{25e3·1500}$) = 0.32 (부족감쇠)
차체 가속도 피크 $\approx$ 1.5 m/s² (초기)
정착 시간 ($\pm$5%) $\approx$ 2.5초

반능동 (스카이훅) 시스템 응답:

스카이훅 상수 c_sky = 8 kN·s/m
제어 알고리즘: 차체 속도 ẋ_body 부호에 따라 c_soft(3 kN·s/m) ↔ c_stiff(10 kN·s/m) 전환

스카이훅 제어 단계:

  • t=0~0.1s (상향 시작): ẋ_body > 0 $\rightarrow$ c_stiff = 10 kN·s/m 선택
  • t=0.1~0.2s (상향 최고점): ẋ_body $\approx$ 0 $\rightarrow$ c_soft = 3 kN·s/m 전환
  • t=0.2~0.3s (하향): ẋ_body < 0 → c_soft 유지
  • t>0.3s (정착): ẋ_body $\approx$ 0 $\rightarrow$ c_soft 유지 (승차감 우선)

결과:

차체 가속도 피크 $\approx$ 0.8 m/s² (패시브 대비 46% 감소)
정착 시간 $\approx$ 0.8초 (패시브 대비 68% 단축)
승차감 개선도 $\approx$ ISO 2631 기준 쾌적성 +30%

승차감 평가 기준:

차체 가속도 RMS < 0.5 m/s²: 쾌적 | 0.5~1.0: 보통 | > 1.0: 불쾌

반능동 예시 결과: RMS $\approx$ 0.35 m/s² $\rightarrow$ 매우 쾌적

DEEP DIVE
고급 주제

Groundhook 제어는 스카이훅과 상보적인 전략입니다. 노면(wheel) 속도를 기준으로 하는 제어로, 타이어 동적 부하(wheel hop) 감소를 목표로 합니다.

Groundhook: $F_{GH} = -c_{gnd} \cdot \dot{x}_u \rightarrow$ 타이어 접지력 최적화

ADD(Acceleration Driven Damper): $c = \begin{cases} c_{max} & \ddot{x}_s \cdot (\dot{x}_s - \dot{x}_u) > 0 \\ c_{min} & \text{else} \end{cases}$

조건식: IF (ẋ_wheel · F_ideal_wheel) > 0 THEN stiff ELSE soft

효과:

  • 타이어 접지압 변동 감소 $\rightarrow$ 접지력 안정화
  • 브레이킹 안정성 향상 (ABS 상호작용)
  • 고속 코너링에서 롤 거동 안정화

결론: 스카이훅 + Groundhook 조합 제어로 승차감 + 핸들링 + 안전성을 동시에 달성.

MR(Magnetorheological) 댐퍼는 전자제어 댐퍼의 한 단계 진화입니다. 강자성 입자가 포함된 오일이 전자기장에 반응하여 점도가 연속적으로 변합니다.

Bingham 모델: $F = c\dot{x} + F_y \cdot \text{sgn}(\dot{x})$,   $F_y = f(I_{coil}) \rightarrow$ 전류로 항복력 제어

응답시간: $\tau_{MR} \approx 1\text{~}5$ ms $\rightarrow$ 가변 댐퍼 중 가장 빠름

르노코리아 실무 Tip: MR 댐퍼의 Fail-Safe 모드(전류 0)에서도 최소 감쇠가 확보되어야 합니다. 이는 안전 규정 필수 요건입니다.

동작 원리:

  • 전자기 코일에 통전 $\rightarrow$ 자기장 발생
  • 강자성 입자 정렬 $\rightarrow$ 오일 점도 급증 (10배 이상)
  • 댐핑력 = 기본 마찰 + 자기장 의존 점성력

장점:

  • 응답시간 매우 빠름 (1~5ms)
  • 감쇠력 무한 단계 제어 (CDC의 2~3단 전환 vs 무한 연속)
  • 전력 소비 매우 적음 (낮은 전류)

현재: BMW M, Audi Q7 같은 고급 차종에서 채용 중.
