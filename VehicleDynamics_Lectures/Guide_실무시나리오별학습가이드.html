댐퍼 튜닝 (감쇠력 세팅)
TUNING

저속 험로 승차감 vs 고속 안정성 밸런스를 맞추기 위한 감쇠력 최적화. C/Cc 비율 조정, FRF 분석으로 실차 상관성 확보.

D03 감쇠비 정의 → D17 댐퍼 모델 → D09 쿼터카 고유진동수 → D21 트레이드오프 → D25 FRF 측정 → D31 FRF 상관성
💡 실무 팁 — 리바운드/컴프레션 비율(R/C ratio)은 보통 2.5~3.5:1. D17에서 바이리니어 모델 확인 후 D31에서 실차-시뮬 상관성을 잡는 흐름이 핵심.

RIDE

D16 스프링 특성 → D08 쿼터카 모델 → D10 하프카 모델 → D11 모드 해석 → D14 노면 스펙트럼

D23 푸리에 변환 → D24 PSD 정의 → D14 노면 스펙트럼 → D12 전달함수
과속방지턱 충격 저감
RIDE

과도 충격 입력에 대한 범프스톱/댐퍼 최적화. 과속방지턱, 포트홀 등 단발성 충격 저감 전략.

D05 강제 조화진동 → D07 공진 조건 → D13 전달률·방진 → D16 스프링 특성 → D17 댐퍼 모델
💡 실무 팁 — 범프스톱 간극(jounce clearance)이 부족하면 2차 충격 발생. 범프스톱 프로그레시브 강성 설계가 핵심.

D08 쿼터카 모델 → D09 고유진동수 → D16 스프링 특성 → D21 트레이드오프
💡 실무 팁 — 적차 시 ωn이 20% 이상 변화하면 별도 세팅 필요. 에어 스프링 또는 자동 레벨링 검토.

RIDE

D10 하프카 모델 → D11 모드 해석 → D14 노면 스펙트럼 → D21 트레이드오프
💡 실무 팁 — 피치 절점이 뒷좌석에 가까울수록 피치 체감 감소. 전후 스프링 레이트 비율로 절점 위치 제어.

💡 실무 팁 — 0.2 Hz 부근이 멀미 유발 피크. 피치 모드와 바운스 모드의 주파수 분리가 핵심 전략.
