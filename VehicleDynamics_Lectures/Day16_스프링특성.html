INTUITION

왜 이 수식이 필요한가?

💡 수식 유도 읽는 법 — 각 STEP의 파란색 라벨은 "왜 이 변환을 하는가"를 설명합니다. 수식만 보지 말고 동기 → 변환 → 의미 순서로 읽으세요. 눈으로 3번 읽는 것보다 손으로 1번 쓰는 것이 기억에 4배 효과적입니다.

서스펜션에서 스프링은 '에너지 저장 장치'다. 코일 스프링, 에어 스프링, 토션 바 — 형태는 달라도 F = k·x가 기본이다. 여러 스프링이 조합되면 직렬(부드러워짐)과 병렬(뻣뻣해짐)의 합성 법칙을 따른다. 현가장치에서 메인 스프링 + 범프 스톱은 직렬이 아닌 병렬이며, 이 구분이 실무 설계의 출발점이다.

DERIVATION

수식 한 줄씩 유도

Step 1: 병렬 스프링 합성
keq=k1+k2k_eq = k_1 + k_2ke​q=k1​+k2​

이 항의 의미: 병렬: 두 스프링이 같은 변위 x를 공유하고 힘이 합산된다. F_total = F₁ + F₂ = k₁x + k₂x = (k₁+k₂)x.

부호/변환 이유: 합성 강성은 항상 개별 강성보다 크다 → 시스템이 더 뻣뻣해진다.

이전 단계와의 연결: Day 01에서 단일 스프링 F = kx를 유도했다. 이를 두 개로 확장한다.

↓
Step 2: 직렬 스프링 합성
1/keq=1/k1+1/k2→keq=k1k2/(k1+k2)1/k_eq = 1/k_1 + 1/k_2 \rightarrow k_eq = k_1k_2/(k_1+k_2)1/ke​q=1/k1​+1/k2​→ke​q=k1​k2​/(k1​+k2​)

이 항의 의미: 직렬: 두 스프링에 같은 힘 F가 작용하고 변위가 합산된다. x_total = x₁ + x₂ = F/k₁ + F/k₂.

부호/변환 이유: 합성 강성은 항상 개별 강성보다 작다 → 시스템이 더 부드러워진다.

이전 단계와의 연결: 병렬과 반대 패턴이다. 전기회로의 저항 병렬 합성(1/R = 1/R₁ + 1/R₂)과 형태가 같다.

↓
Step 3: 비선형 스프링 (Progressive Rate)
F(x)=k1x+k3x3(하드닝 스프링:k3>0)F(x) = k_1x + k_3x^3 (\text{하드닝 스프링}: k_3 > 0)F(x)=k1​x+k3​x3(하드닝 스프링:k3​>0)

이 항의 의미: 코일 스프링의 피치가 변하거나 범프 스톱이 접촉하면 변위가 커질수록 강성이 증가한다.

부호/변환 이유: k₃ > 0이면 하드닝(hardening) — 큰 입력에서 강성 증가로 바닥 침(bottoming) 방지. k₃ < 0이면 소프트닝(softening).

이전 단계와의 연결: Day 05(강제진동)에서 선형 스프링만 다뤘다. 실차에서는 비선형이 필수다.

Step 4: Wheel Rate와 Spring Rate의 관계
kwheel=kspring⋅(MR)2k_wheel = k_spring \cdot (MR)^2kw​heel=ks​pring⋅(MR)2

이 항의 의미: Motion Ratio(MR)는 '스프링 변위 / 휠 변위'의 비율이다. MR < 1이면 스프링이 휠보다 적게 움직여서 휠 레이트가 스프링 레이트보다 작아진다.

부호/변환 이유: MR² 형태인 이유: 에너지 보존 ½kₛ·δ_s² = ½k_w·δ_w² → k_w = kₛ·(δ_s/δ_w)² = kₛ·MR².

이전 단계와의 연결: Day 09에서 쿼터카 고유진동수를 구할 때 wheel rate 개념이 잠시 등장했다. 여기서 정식으로 유도한다.

📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① 선형 스프링: 후크의 법칙

$$F = kx \quad \Rightarrow \quad k = \frac{dF}{dx} = \text{const}$$

이상적인 경우. 실제 스프링은 비선형성이 항상 존재

② 프로그레시브(Progressive) 스프링 모델

$$F(x) = k_1 x + k_3 x^3 \quad (k_3 > 0)$$
$$k_{eff}(x) = \frac{dF}{dx} = k_1 + 3k_3 x^2$$

물리적 의미: 변위가 커질수록 강성↑. 코일 스프링의 피치가 불균일하면 자연스럽게 프로그레시브 특성

③ 듀얼 레이트 스프링

$$k_{eff} = \begin{cases} k_1 & |x| \leq x_{transition} \\ k_1 + k_2 & |x| > x_{transition} \end{cases}$$

범프스톱 접촉 시 갑자기 강성↑ → 충격 완화를 위해 프로그레시브 범프스톱 적용

④ 에너지 저장과 위치에너지

$$U = \int_0^x F(x')\,dx' = \frac{1}{2}k_1 x^2 + \frac{1}{4}k_3 x^4$$
$$\text{등가 선형 강성: } k_{eq} = \frac{2U}{x_{max}^2}$$

SIMPACK 적용: 비선형 스프링은 Force Element → Spline Data로 F-x 곡선 직접 입력

심화 학습

에어 스프링의 강성은 압력·체적의 함수: k_air = n·P·A²/V (n=폴리트로픽 지수). 코일 스프링은 k = Gd⁴/(8D³N)으로 재질과 기하학에 의해 결정된다. 에어 스프링은 하중에 따라 압력을 조절해 차고를 일정하게 유지할 수 있어 프리미엄 차량에 사용된다. Simpack에서는 에어 스프링 전용 Force Element(압력-체적 커브)를 제공한다.

코일 스프링: $k = \frac{Gd^4}{8D^3 N_a}$ ($G$: 전단탄성계수, $d$: 선경, $D$: 코일 평균경, $N_a$: 유효 권수)

에어 스프링: $k_{air} = \frac{nP_0 A_{eff}^2}{V_0}$ ($n$: 폴리트로픽 지수, $P_0$: 초기 압력, $A_{eff}$: 유효 면적)

■ 선형 스프링의 한계

비선형 강성: $F = k_1 x + k_2 x^2$ → 등가 강성: $k_{eq}(x) = k_1 + 2k_2 x$

이중 감김(Dual Rate): $k = \begin{cases} k_1 & |x| < x_0 \\ k_1 + k_2 & |x| \geq x_0 \end{cases}$

선형: F = kx → 모든 하중에서 같은 강성

문제: 공차(1인승)에선 너무 딱딱, 만차(5인승)에선 너무 물렁

■ Progressive 스프링 설계

저하중: 소프트(1.5~2.0 N/mm) → 승차감 확보

고하중: 하드(3.0~4.0 N/mm) → 범프스톱 전에 충분한 지지력

구현: 피치 간격(pitch)이 불균일한 코일 스프링

■ 레퍼런스 차량 스프링 특성

메인 스프링: k₁ = 22 N/mm (0~60mm)

범프스톱 접촉 후: k₂ = 80+ N/mm (병렬 추가)

합성: k_eq = k₁ + k₂ = 102 N/mm

■ 범프스톱 (Bump Stop)

범프스톱: $F_{bs} = k_{bs}(x - x_{gap})^n$ ($n > 1$, 비선형) → 급격한 강성 증가

잔여 스트로크: $\Delta x = x_{max} - x_{static}$ → 최소 $\frac{2}{3} \cdot x_{max}$ 확보

르노코리아 실무 Tip: 범프스톱 간섭(Bump Stop Contact)은 승차감 급격 악화의 원인입니다. 정적 처짐 위치에서 범프스톱까지 거리를 반드시 확인하세요.

역할: 최대 압축 시 금속 접촉 방지

재질: 마이크로셀룰러 폴리우레탄 (비선형 탄성)

레퍼런스 차량: 잔여 스트로크 30mm에서 접촉, 풀범프에서 ~5kN 지지

■ 리바운드스톱 (Rebound Stop)

역할: 최대 인장 시 댐퍼 로드 보호

주로 댐퍼 내장형 (하이드로릭 범프스톱)

■ Simpack 입력
