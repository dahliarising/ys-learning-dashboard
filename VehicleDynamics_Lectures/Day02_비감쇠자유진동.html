INTUITION

왜 이 수식이 필요한가?

Day 01에서 유도한 mẍ + cẋ + kx = F(t) 에서, 가장 단순한 경우부터 풀어봅니다. 감쇠(c)도 없고 외력(F)도 없는 상황 — 즉, 스프링에 연결된 질량을 손으로 잡아당겼다 놓은 경우입니다.

K&C 시험에서 차체를 눌렀다가 놓으면 위아래로 출렁이다 멈추죠? 여기서 '얼마나 빠르게 출렁이는가'를 결정하는 것이 고유진동수 ωₙ 입니다. 이 값 하나로 서스펜션의 기본 성격이 결정됩니다.

정교화 질문: "왜 하필 √(k/m) 인가? k²/m 이나 k·m 이 아닌 이유는?" → 유도 과정에서 자연스럽게 답을 찾아보세요.

🔗 기억 훅: 농구공(높은 k, 작은 m) vs 비치볼(낮은 k, 큰 부피) — 농구공을 바닥에 튕기면 빠르게 통통(높은 ωₙ), 비치볼은 느릿느릿 둥둥(낮은 ωₙ). "단단+가벼움 = 빠른 진동"

DERIVATION

수식 한 줄씩 유도

STEP 1 — 비감쇠·무외력 조건 적용
m⋅x¨+k⋅x=0m\cdot \ddot{x} + k\cdot x = 0m⋅x¨+k⋅x=0

이 항의 의미: c=0(감쇠 없음), F(t)=0(외력 없음)을 Day 01 운동방정식에 대입하면 질량-스프링 만의 순수 자유진동 방정식이 됩니다.

부호/변환 이유: 두 항 모두 양(+)인 이유: 좌변으로 모든 시스템 특성을 모았기 때문입니다. 관성항(mẍ)과 복원항(kx)만 남았습니다.

이전 단계와의 연결: Day 01의 mẍ + cẋ + kx = F(t) 에서 cẋ 과 F(t) 를 제거한 것입니다.

↓
STEP 2 — 해의 가정 (지수함수)
x(t)=A⋅est→해를 가정x(t) = A\cdot e^{st} \rightarrow \text{해를 가정}x(t)=A⋅est→해를 가정

이 항의 의미: 상미분방정식의 표준 풀이법: 해를 x = Ae^(st) 형태로 가정합니다. A는 상수(초기조건에서 결정), s는 특성근입니다.

부호/변환 이유: e^(st)를 가정하는 이유: 미분해도 형태가 보존됩니다. ẋ = A·s·e^{st}, ẍ = A·s²·e^{st}.

이전 단계와의 연결: 2차 ODE의 표준 풀이 전략입니다. 대입 후 s에 대한 대수방정식(특성방정식)을 얻습니다.

↓
STEP 3 — 특성방정식 도출
ms2+k=0→s2=−k/m=−ωn2ms^2 + k = 0 \rightarrow s^2 = -k/m = -\omega_n^2ms2+k=0→s2=−k/m=−ωn2​

이 항의 의미: STEP 2를 STEP 1에 대입: m·A·s²·e^{st} + k·A·e^{st} = 0. A·e^{st} ≠ 0이므로 양변을 나누면 ms² + k = 0.

부호/변환 이유: s² = −k/m에서 우변이 음수. m > 0, k > 0이므로 s²은 반드시 음수 → s는 허수(imaginary)! 이것이 "진동"의 수학적 근거입니다.

핵심 통찰: 여기서 −k/m을 −ωₙ²으로 정의합니다. 즉 ωₙ ≡ √(k/m) — 이것이 고유진동수의 정의입니다!

↓
✓ 중간 체크포인트: 여기까지 따라왔다면 핵심을 잡은 것입니다. "특성방정식의 근이 순허수 → 진동 해". 잠시 눈을 감고 "왜 s²이 음수면 진동인가?"를 스스로 설명해 보세요. (답: e^(iωt) = cos(ωt) + i·sin(ωt) → 삼각함수 = 진동)
STEP 4 — 고유진동수 정의 + 일반해
s=±iωn→x(t)=C1⋅eiωnt+C2⋅e−iωnts = \pm i\omega_n \rightarrow x(t) = C_1\cdot e^{i\omega_nt} + C_2\cdot e^{-i\omega_nt}s=±iωn​→x(t)=C1​⋅eiωn​t+C2​⋅e−iωn​t

이 항의 의미: 두 켤레 허근 s₁ = +iωₙ, s₂ = −iωₙ. 2차 ODE이므로 두 독립 해의 선형결합이 일반해입니다.

부호/변환 이유: 오일러 공식 e^(±iθ) = cos θ ± i·sin θ를 적용하면 실수 형태로 변환됩니다.

↓
STEP 5 — 실수 형태 일반해
x(t)=A⋅cos⁡(ωnt)+B⋅sin⁡(ωnt)x(t) = A\cdot \cos (\omega_n t) + B\cdot \sin (\omega_n t)x(t)=A⋅cos(ωn​t)+B⋅sin(ωn​t)

이 항의 의미: 오일러 공식 적용 후 실수 상수 A, B로 재정의한 최종 일반해. 순수한 조화진동(harmonic oscillation)입니다.

부호/변환 이유: cos은 초기변위 성분, sin은 초기속도 성분. 감쇠가 없으므로 진폭이 영원히 유지됩니다.

↓
STEP 6 — 초기조건 적용 [NEW]
x(0)=x0→A=x0∣x˙(0)=v0→B=v0/ωnx(0) = x_0 \rightarrow A = x_0 | \dot{x}(0) = v_0 \rightarrow B = v_0 / \omega_nx(0)=x0​→A=x0​∣x˙(0)=v0​→B=v0​/ωn​

이 항의 의미: t=0에서 x(0) = A·cos(0) + B·sin(0) = A = x₀. 미분하면 ẋ(t) = −A·ωₙ·sin(ωₙ t) + B·ωₙ·cos(ωₙ t). t = 0 에서 ẋ(0) = B·ωₙ = v₀.

최종 특해:

x(t)=x0⋅cos⁡(ωnt)+(v0/ωn)⋅sin⁡(ωnt)x(t) = x_0\cdot \cos (\omega_n t) + (v_0/\omega_n)\cdot \sin (\omega_n t)x(t)=x0​⋅cos(ωn​t)+(v0​/ωn​)⋅sin(ωn​t)

차량 적용: K&C 시험에서 x₀ = 눌린 양(mm), v₀ = 0(정지 상태에서 놓음) → x(t) = x₀·cos(ωₙ t). 순수 코사인 진동!

진폭-위상 등가 표현:

x(t)=X⋅cos⁡(ωnt−φ)여기서X=x02+(v0/ωn2),φ=arctan⁡(v0/(x0⋅ωn))x(t) = X\cdot \cos (\omega_n t - \varphi) \text{여기서} X = \sqrt{x_0^2 + (v_0/\omega_n}^2), \varphi = \arctan (v_0 / (x_0\cdot \omega_n))x(t)=X⋅cos(ωn​t−φ)여기서X=x02​+(v0​/ωn​​2),φ=arctan(v0​/(x0​⋅ωn​))
✓ 주파수 변환 체크포인트: ωₙ [rad/s] ↔ fₙ [Hz] ↔ Tₙ [s] 변환을 꼭 기억하세요:
   fₙ = ωₙ / 2π  |  Tₙ = 1/fₙ = 2π/ωₙ  |  승용차 바운스: fₙ ≈ 1.0~1.5 Hz → Tₙ ≈ 0.7~1.0 s

🚗 실차 적용 예시 — 레퍼런스 C-SUV

코너 스프렁 매스 mₛ ≈ 320 kg, 스프링 강성 kₛ ≈ 22 N/mm, 타이어 강성 kₜ ≈ 200 N/mm

라이드 레이트: kᵣ = (22 × 200) / (22 + 200) = 19.8 N/mm = 19,800 N/m

고유진동수: ωₙ = √(19800 / 320) = √61.9 = 7.87 rad/s

Hz 변환: fₙ = 7.87 / 2π = 1.25 Hz → Tₙ = 0.80 s

✓ 1.25 Hz는 승용차 바운스 목표 범위(1.0~1.5 Hz) 내. 좋은 승차감 기반!

📏 승차감 설계 기준선

Janeway Criterion: fₙ < 1.0 Hz → "보트 같은" 느낌(멀미 유발), fₙ > 1.5 Hz → "딱딱한" 느낌. 1.0~1.3 Hz가 최적 대역.

프론트 vs 리어: 리어를 10~20% 높게 설정(f_rear / f_front ≈ 1.1~1.2) → 피칭 수렴을 빠르게 → "flat ride" 달성.

ISO 2631: 1~2 Hz 영역은 인체 전정기관(귀의 평형기관)의 민감 대역 — Day 12(인체 응답)에서 상세히 다룸.

DEEP DIVE

심화 학습

특성방정식 ms² + k = 0에서 s = ±iωₙ 이 나왔습니다. '허수'라니, 현실에서 무슨 의미일까요?

오일러 공식: 진동의 수학적 뿌리

핵심은 오일러 공식입니다:

eiθ=cos⁡θ+i⋅sin⁡θe^{i\theta} = \cos \theta + i\cdot \sin \thetaeiθ=cosθ+i⋅sinθ

일반해 x(t) = Ce^(st) 에 s = iωₙ 을 대입하면:

x(t)=Ceiωnt=C(cos⁡ωnt+i⋅sin⁡ωnt)x(t) = Ce^{i\omega_nt} = C(\cos \omega_nt + i\cdot \sin \omega_nt)x(t)=Ceiωn​t=C(cosωn​t+i⋅sinωn​t)

실수부만 취하면 → x(t) = A·cos(ωₙt) + B·sin(ωₙt), 즉 순수한 조화진동!

s-평면(복소평면)에서 바라보기

제어공학에서 사용하는 s-평면(복소평면)으로 시각화하면 직관적으로 이해할 수 있습니다:

실수축 (Re)
σ 값 → 진폭의 시간 변화
σ < 0: 진폭 감소 (안정)
σ > 0: 진폭 증가 (불안정)
σ = 0: 진폭 일정 (오늘!)
허수축 (Im)
ω 값 → 진동 주파수
ω 클수록 빠른 진동
ω = 0이면 진동 없음
±iωₙ: 허수축 위 = 비감쇠

즉 극점(pole)이 허수축 위에 있다 = 비감쇠 자유진동입니다. Day 04에서 감쇠가 추가되면 극점이 왼쪽(Re < 0)으로 이동하며, 이것이 부족감쇠 진동입니다.

실수부 σ에 따른 시스템 거동

s=σ±iωds = \sigma \pm i\omega_ds=σ±iωd​

● σ < 0: e^(σt) → 0 — 진동이 점점 줄어듦 (부족감쇠) — Day 04

● σ > 0: e^(σt) → ∞ — 진동이 점점 커짐 (불안정) — 차량에서 절대 불가!

● σ = 0: e^(0) = 1 — 영원한 진동 (비감쇠) — 오늘의 주제

● 실수근만 (ω=0): 진동 없이 지수적 복귀 (과감쇠) — Day 04

엔지니어 관점에서의 핵심 요약

수학적으로 복소수가 나타나면 두려워할 필요 없습니다. 복소수는 단지 "진폭 변화(σ)와 진동 주파수(ω)를 하나의 수로 표현하는 도구"일 뿐입니다. 서스펜션 엔지니어에게 s-평면은 시스템의 안정성과 진동 특성을 한눈에 파악하는 강력한 도구입니다.

실제 차량에서 ωₙ = √(k/m) 의 k는 스프링 강성(kₛ)이 아니라 라이드 레이트(kᵣ)입니다.

직렬 스프링의 원리

1/kr=1/ks+1/kt→kr=ks⋅kt/(ks+kt)1/k_r = 1/k_s + 1/k_t \rightarrow k_r = k_s \cdot k_t / (k_s + k_t)1/kr​=1/ks​+1/kt​→kr​=ks​⋅kt​/(ks​+kt​)

스프링과 타이어가 직렬(series) 연결이므로, 합성 강성은 항상 개별 강성 중 작은 쪽보다 더 작습니다. 이것은 저항의 병렬연결과 같은 원리입니다 (전기회로 비유!).

타이어 강성 kₜ 200 N/mm
라이드 레이트 kᵣ 19.82 N/mm
차이 −9.9% (kₛ 대비)

실무 Tip: 언제 구분이 중요한가?

✅ kᵣ ≈ kₛ 인 경우
- 일반 승용차 (kₜ >> kₛ)
- 개념 설계 단계
- 빠른 타겟 계산 시
⚠ 반드시 kᵣ 사용
- SUV/트럭 (kₛ가 높아 kₜ에 근접)
- 런플랫 타이어 (kₜ ↑↑↑)
- 정밀 튜닝 / 최종 사양 결정 시

비감쇠 자유진동이 영원히 지속되는 이유를 에너지 관점에서 이해하면, 감쇠 시스템(Day 04)으로의 확장이 자연스러워집니다.

Etotal=KE+PE=12mx˙2+12kx2=constE_total = KE + PE = \frac{1}{2}m\dot{x}^2 + \frac{1}{2}kx^2 = constEt​otal=KE+PE=21​mx˙2+21​kx2=const

x(t) = X·cos(ωₙt) 일 때, 각 에너지의 변화:

t = 0 (최대 변위): x = X (max), ẋ = 0 → KE = 0, PE = ½kX² = 전체 에너지가 PE에 저장

t = T/4 (평형점 통과): x = 0, ẋ = −Xωₙ (max) → PE = 0, KE = ½m(Xωₙ)² = 전체 에너지가 KE로 전환

검증: ½m(Xωₙ)² = ½mX²·(k/m) = ½kX² ✓ — 에너지 완벽 보존!

감쇠기(damper)가 추가되면 소산 에너지(dissipated energy)가 발생합니다:

Edissipated=∫cx˙2dt>0→Etotal감소→진폭 감소E_dissipated = \int c\dot{x}^2 dt > 0 \rightarrow E_total \text{감소} \rightarrow \text{진폭 감소}Ed​issipated=∫cx˙2dt>0→Et​otal감소→진폭 감소

감쇠기는 운동에너지를 열로 변환합니다. 이것이 서스펜션 댐퍼의 역할이며, 자동차가 범프를 넘은 후 빠르게 안정되는 이유입니다.

Simpack에서 에너지 확인

Simpack에서 비감쇠 시뮬레이션 후 KE와 PE를 y-Results로 플로팅하면, 두 에너지가 정확히 반대 위상(180°)으로 교번하고, 합은 일정해야 합니다. 만약 합이 서서히 감소한다면 → 수치 감쇠(numerical damping)의 증거!

설계 과정에서 "스프링을 10% 높이면 진동수가 얼마나 변하는가?"라는 질문이 빈번합니다. 감도 분석(sensitivity analysis)으로 체계적으로 답할 수 있습니다.

편미분으로 구하는 감도

fn=(1/2π)⋅k/mf_n = (1/2\pi)\cdot \sqrt{k/m}fn​=(1/2π)⋅k/m​

k에 대한 감도:

∂fn/∂k=(1/2π)⋅(1/2)⋅(1/km)=fn/(2k)\partialf_n/\partialk = (1/2\pi)\cdot (1/2)\cdot (1/\sqrt{km}) = f_n/(2k)∂fn​/∂k=(1/2π)⋅(1/2)⋅(1/km​)=fn​/(2k)

상대 감도로 변환하면:

Δfn/fn≈12⋅Δk/k\Deltaf_n/f_n \approx \frac{1}{2} \cdot \Deltak/kΔfn​/fn​≈21​⋅Δk/k

즉, 강성이 10% 증가하면 → 고유진동수는 약 5% 증가 (√관계이므로 절반).

실차 시나리오 분석

기본: kₛ=22, m=320kg fₙ = 1.32 Hz
승객 2명 추가 (+150kg) fₙ = 1.08 Hz (−18%)
스프링 20% 강화 fₙ = 1.45 Hz (+10%)
런플랫 타이어 (kₜ 2배) fₙ = 1.37 Hz (+4%)

질량 변화가 강성 변화보다 fₙ에 큰 영향을 줍니다. 이는 ωₙ = √(k/m) 에서 m이 분모에 있기 때문이며, 실제 차량에서 공차/만차 진동수 차이가 크다는 것을 의미합니다.

설계자의 딜레마

승차감을 위해 fₙ을 1.0~1.3 Hz로 낮추고 싶지만, 이는 정적 처짐(static deflection)을 키웁니다:

δst=mg/k=g/ωn2→fn=1Hz일 때δst≈25mm\delta_st = mg/k = g/\omega_n^2 \rightarrow f_n = 1 Hz\text{일 때} \delta_st \approx 25 mmδs​t=mg/k=g/ωn2​→fn​=1Hz일 때δs​t≈25mm

서스펜션 스트로크가 제한되어 있으므로, fₙ을 무한정 낮출 수 없습니다. 이 승차감 vs 패키징(스트로크)의 트레이드오프가 서스펜션 설계의 핵심 과제입니다.

이론에서 x(0) = x₀, ẋ(0) = v₀ 라고 했는데, 실제 차량에서 이 초기조건은 무엇을 의미할까요?

차량에서의 초기조건 시나리오

시나리오 1: 정적 하중 해제 (x₀ ≠ 0, v₀ = 0)
차를 잭업 후 내리면, 초기 변위 x₀ = mg/k 에서 시작하여 진동합니다.
→ x(t) = x₀ · cos(ωₙt) — 순수 코사인 파형
시나리오 2: 속도방지턱 (x₀ = 0, v₀ ≠ 0)
평탄한 도로에서 방지턱을 만나면 타이어가 수직 속도 v₀ 를 받습니다.
→ x(t) = (v₀/ωₙ) · sin(ωₙt) — 순수 사인 파형
시나리오 3: 복합 입력 (x₀ ≠ 0, v₀ ≠ 0)
도로 요철 위를 주행할 때 — 변위와 속도가 동시에 존재합니다.
→ x(t) = X · cos(ωₙt − φ) — 진폭-위상 표현 사용

진폭 X와 위상 φ 계산

X=x02+(v0/ωn2)X = \sqrt{x_0^2 + (v_0/\omega_n}^2)X=x02​+(v0​/ωn​​2)
φ=arctan⁡(v0/(ωn⋅x0))\varphi = \arctan (v_0 / (\omega_n \cdot x_0))φ=arctan(v0​/(ωn​⋅x0​))

수치 예: 방지턱 통과 시 x₀ = 5 mm, v₀ = 0.3 m/s, ωₙ = 8.3 rad/s 라면:

v₀/ωₙ = 0.3/8.3 = 36.1 mm

X = √(5² + 36.1²) = √(25 + 1303) = 36.4 mm

φ = arctan(36.1/5) = 82.1° — 거의 사인 파형에 가까움 (v₀ 지배적)

이 결과는 방지턱 통과 시 초기 속도가 초기 변위보다 훨씬 지배적이라는 것을 보여줍니다. Simpack에서 초기조건을 설정할 때 이 물리적 의미를 이해하고 있으면 결과 해석이 명확해집니다.

QUESTION 01 — 개념 이해

스프링 강성 k를 2배로 높이고 질량 m을 절반으로 줄이면, 고유진동수 ωₙ 은 어떻게 변하는가?

A. √2배 (약 1.41배)
B. 2배
C. 4배
D. 변화 없음
📝 단계별 풀이
Step 1 기본 공식 확인: ωₙ = √(k/m)
Step 2 새로운 조건 대입: k → 2k, m → m/2
Step 3 ωₙ' = √(2k / (m/2)) = √(4k/m) = 2·√(k/m) = 2 · ωₙ
💡 핵심 인사이트: ωₙ ∝ √(k/m) 이므로, k와 m이 각각 ωₙ에 제곱근 관계로 기여합니다. k↑ 2배와 m↓ ½배의 효과가 합쳐져 ωₙ² 기준으로 4배 → ωₙ 기준으로 2배!
QUESTION 02 — 실차 적용

코너 스프렁 매스 350 kg, 라이드 레이트 20 N/mm인 차량의 바운스 고유진동수 fₙ 은 약 얼마인가?

A. 0.85 Hz
B. 1.20 Hz
C. 1.75 Hz
D. 2.50 Hz
📝 단계별 풀이
Step 1 단위 변환: kᵣ = 20 N/mm = 20,000 N/m
Step 2 ωₙ = √(k/m) = √(20000/350) = √57.14
Step 3 ωₙ = 7.56 rad/s
Step 4 fₙ = ωₙ / 2π = 7.56 / 6.283 = 1.20 Hz ✓
💡 핵심 인사이트: 1.20 Hz는 Janeway Criterion의 승차감 목표 범위(1.0~1.3 Hz) 안에 있습니다. 실무에서 이 계산은 가장 기본적인 타겟 검증이므로 암기 수준으로 빠르게 할 수 있어야 합니다.
💡 암산 꿀팁: fₙ ≈ (1/2π)·√(k/m) 에서 1/2π ≈ 0.159. 즉 fₙ ≈ 0.159 × √(k/m).
QUESTION 03 — Simpack 검증

Simpack Time Integration에서 비감쇠 자유진동 결과가 시간이 지나면서 진폭이 서서히 줄어든다. 가장 가능성 높은 원인은?

A. k 값 입력 오류
B. 질량이 음수
C. 수치 감쇠(numerical damping) — 적분기 설정 확인 필요
D. 초기조건 미설정
📝 풀이 및 해설
분석 c = 0 (비감쇠)으로 설정했는데 진폭이 줄어든다? → 물리적 감쇠가 아닌 수치적 원인
원인 시간적분기(integrator)가 각 스텝에서 발생시키는 미세한 수치 감쇠(numerical damping)
검증법 Step size를 절반으로 줄이고 재실행 → 진폭 감소율이 줄어들면 수치감쇠 확정

QUESTION 04 — 설계 판단

"Flat ride" 조건을 위해 프론트와 리어의 바운스 고유진동수 관계는?

A. f_front = f_rear (동일)
B. f_front > f_rear (프론트가 높음)
C. f_rear > f_front (리어가 10~20% 높음)
D. 관계없음
📝 풀이 및 해설
배경 차량이 범프를 넘으면 프론트가 먼저, 리어가 나중에 가진됩니다.
원리 프론트 진동이 아직 남아있을 때 리어 진동이 시작 → 피칭(pitch) 발생
해법 f_rear를 10~20% 높게 설정 → 리어가 더 빨리 진동 → 뒤쪽이 먼저 정착 → 피칭 수렴 빠름
수치 목표: f_rear / f_front ≈ 1.1 ~ 1.2
💡 Maurice Olley의 Flat Ride 이론 (1934):
"리어 서스펜션의 진동수를 프론트보다 높게 만들면, 범프 통과 시 프론트와 리어의 진동이 빠르게 상쇄되어 피칭이 최소화된다." 이 원칙은 90년이 지난 지금도 모든 승용차 설계의 기본입니다. 대부분의 양산 차량이 이 원칙을 따르고 있습니다!
