💡 타이어는 어떻게 횡력을 만드는가?

타이어의 횡력(Lateral Force) 발생 메커니즘은 직관적이지 않습니다. 타이어가 진행 방향과 다른 각도로 회전하면, 접지면(Contact Patch)의 고무 요소들이 탄성 변형을 일으킵니다. 이 변형이 복원력을 만들어내고, 그 합력이 바로 횡력 Fy입니다.

슬립앵글(Slip Angle, $\alpha$)은 타이어 휠 평면의 방향과 실제 진행 방향 사이의 각도입니다. 브러시 모델(Brush Model)로 비유하면: 빗질할 때 빗을 비스듬히 밀면 털이 옆으로 휘는 것과 같습니다. 접지면 앞쪽은 아직 변형이 작고, 뒤쪽으로 갈수록 변형이 커져 결국 슬립이 시작됩니다.

Pacejka의 Magic Formula는 이 비선형 관계를 단 하나의 수식으로 모사합니다. 4개의 계수(B, C, D, E)로 거의 모든 타이어의 힘-슬립 특성을 표현할 수 있어, 차량 동역학 시뮬레이션의 산업 표준이 되었습니다.

STEP 01 — Magic Formula 기본 형태

Pacejka Magic Formula의 일반 형태는 다음과 같습니다:

Y(x) = D · sin(C · arctan(B·x − E·(B·x − arctan(B·x))))

여기서 Y는 출력(Fy 또는 Mz), x는 입력(슬립앵글 $\alpha$ 또는 슬립비 κ)입니다.

물리적 의미: arctan 함수의 중첩으로 S-커브를 만들어, 선형 영역 $\rightarrow$ 포화 영역 $\rightarrow$ 감소 영역(E에 의해)을 하나의 수식으로 표현합니다.

↓
STEP 02 — 4개 계수의 물리적 의미

B (Stiffness Factor): 초기 기울기를 결정합니다. Cornering Stiffness C$\alpha$와 관련: C$\alpha$ = B · C · D. B가 클수록 작은 슬립앵글에서 빠르게 횡력이 증가합니다.

C (Shape Factor): 곡선의 전체적 형상을 제어합니다. 순수 횡력은 C $\approx$ 1.3~1.65, 종방향은 C $\approx$ 1.5~1.8이 일반적입니다.

D (Peak Value): 횡력의 최대값으로, D = $\mu$ · Fz에 비례합니다. 하중이 커지면 최대 횡력도 증가하지만, 마찰계수는 감소합니다(하중 감도).

E (Curvature Factor): 피크 이후 곡선 형태를 조절합니다. E < 1이면 점진적 감소, E $\approx$ 0이면 피크 유지, E > 1이면 급격한 감소를 보입니다.

↓
STEP 03 — 하중 의존성 (Load Sensitivity)

실제 타이어에서 B, C, D, E는 모두 수직하중 Fz의 함수입니다:

D(Fz) = a₁·Fz² + a₂·Fz

특히 $\mu$-Fz 관계가 중요합니다: 하중이 증가하면 절대 횡력은 증가하지만 마찰계수($\mu$ = Fy/Fz)는 감소합니다. 이것이 바로 하중 이동이 총 그립을 감소시키는 근본 이유입니다.

$\mu$eff = (Fy,inner + Fy,outer) / (Fz,inner + Fz,outer) < $\mu$0
↓
STEP 04 — Combined Slip (복합 슬립)

실제 주행에서는 종슬립(κ)과 횡슬립($\alpha$)이 동시에 발생합니다. 이때 마찰원(Friction Circle) 제약이 적용됩니다:

$\sqrt{Fx^2 + Fy^2}$ $\leq \mu$ · Fz

종력이 사용되면 그만큼 횡력 여유가 줄어듭니다. Combined Slip 모델에서는 합성 슬립을 정의합니다:

$\sigma$ = $\sqrt{\kappa^2 + \tan^2\alpha}$

그리고 가중 함수(Weighting Function)로 종·횡력을 분배합니다:

Fy,combined = Fy,pure · Gyκ(κ) · cos(arctan(κ/tan$\alpha$))
↓
STEP 05 — 캠버 스러스트 추가

캠버각($\gamma$)에 의한 추가 횡력(Camber Thrust)을 MF에 포함합니다:

Fy,total = Fy($\alpha$) + SVy + Fy,camber($\gamma$)

여기서 SVy는 수직 오프셋(Ply Steer, Conicity), Fy,camber는 캠버 스러스트입니다. 캠버 스러스트 계수 C$\gamma$는 타이어 구조에 따라 200~500 N/deg 범위입니다.

✅ 체크포인트: Magic Formula의 4계수(B,C,D,E)가 각각 강성, 형상, 피크, 곡률을 결정하며, 이들은 모두 하중의 함수라는 점을 기억하세요.
📐 상세 유도과정 (Step-by-Step)

중간 과정 완전 전개

① Pacejka Magic Formula (횡력)

$$F_y = D \sin\bigl(C \arctan(B\alpha - E(B\alpha - \arctan(B\alpha)))\bigr)$$
$$D = \mu_{y} F_z, \quad C \approx 1.3\sim1.65, \quad B = \frac{K_{y\alpha}}{C \cdot D}$$

B: 강성 팩터, C: 형상 팩터, D: 피크값, E: 곡률 팩터. 4개 계수로 타이어 특성 완전 표현

② 복합 슬립 (Combined Slip)

$$\sigma_x = \frac{s}{1+s}, \quad \sigma_y = \frac{\tan\alpha}{1+s}$$
$$\sigma = \sqrt{\sigma_x^2 + \sigma_y^2}$$
$$F_x = \frac{\sigma_x}{\sigma} F(\sigma), \quad F_y = \frac{\sigma_y}{\sigma} F(\sigma)$$

마찰원(friction circle/ellipse) 개념: 종+횡 합력이 $\mu F_z$ 이내. 제동 중 조향 시 횡력 감소!

③ 하중 감도 (Load Sensitivity)

$$\mu_{eff}(F_z) = D/F_z = p_{Dy1} + p_{Dy2}(F_z - F_{z0})/F_{z0}$$
$$\frac{\partial \mu}{\partial F_z} < 0 \quad (\text{하중 증가 }\to\text{ 마찰계수 감소})$$

핵심: 하중 감도 때문에 하중이동 시 총 횡력 감소. 낮은 CG가 유리한 물리적 근거

④ 온도/마모 보정

$$F_y(T) = F_{y,ref} \times \lambda_T(T_{surface})$$
$$\lambda_T = 1 + p_{Ty1}(T - T_{ref}) + p_{Ty2}(T - T_{ref})^2$$
$$\text{최적 온도: } T_{opt} \approx 80\sim100^{\circ}C \;(\text{일반 승용 타이어})$$

MF 6.2 이상에서 온도 효과 포함. 레이싱에서 타이어 워밍업이 중요한 이유 = 점착력 온도 의존성

실전 예제

📝 예제: 205/55R16 Pacejka 피팅 — F&T 시험 데이터로부터 계수 추출

측정 조건: 수직하중 Fz = 4,500N, 캠버 0°, 슬립앵글 −12° ~ +12° 스위프

Step 1. 초기 기울기에서 Cornering Stiffness 추출:

C$\alpha$ = $\Delta F$y/$\Delta \alpha$ |$\alpha \rightarrow 0$ = 3,200N / 2.2° = 83,520 N/rad

Step 2. 피크값에서 D 결정:

D = Fy,max = 4,800N at $\alpha$ = 8.5° $\rightarrow \mu$ = 4800/4500 = 1.067

Step 3. Shape Factor C 추정 (경험적):

C = 1 + (1 − 2/$\pi$ · arcsin(Fy(20°)/D)) $\approx$ 1.45

Step 4. Stiffness Factor B 계산:

B = C$\alpha$ / (C · D) = 83,520 / (1.45 $\times$ 4800) = 12.0

Step 5. Curvature Factor E 최적화 (Least-squares fit):

E = −0.80 (MSE minimization, R² = 0.998)

🔧 검증: 피팅 품질 평가

R² $\geq$ 0.995: 우수 | 0.990~0.995: 양호 | < 0.990: 재피팅 필요

피크 오차 < 2%: 필수 | 초기 기울기 오차 < 3%: 핸들링에 직결

DEEP DIVE
심화 학습

MF 5.2 (PAC2002): 가장 널리 사용되는 버전으로, 순수 슬립과 복합 슬립을 별도의 가중 함수로 처리합니다. 계수 약 80~90개, 대부분의 상용 MBS 소프트웨어에서 지원합니다.

Magic Formula: $F_y = D\sin\{C\arctan[B\alpha - E(B\alpha - \arctan B\alpha)]\}$

$B$: Stiffness Factor, $C$: Shape, $D$: Peak, $E$: Curvature $\rightarrow B \cdot C \cdot D = C_\alpha$ (코너링 강성)

MF 6.2 (PAC2006+): 턴슬립(Turn Slip), 대변형(Large Camber), 파킹/저속 특성을 추가로 모델링합니다. 계수 약 120~150개로 늘어나며, 특히 EV의 저속 파킹 시뮬레이션에 유리합니다.

실무 선택 기준: 일반 핸들링 해석은 MF 5.2면 충분하며, 파킹/극저속 해석이 필요하면 MF 6.2를 사용합니다. 데이터 피팅 비용도 MF 6.2가 약 1.5배 더 높습니다.

타이어 횡력은 슬립앵글 변화에 즉시 반응하지 않습니다. 접지면 길이만큼의 구름 거리가 필요하며, 이를 Relaxation Length ($\sigma$y)라 합니다.

복합 슬립: $\sigma = \sqrt{\sigma_x^2 + \tan^2\alpha} \rightarrow$ 마찰원: $\sqrt{F_x^2 + F_y^2} \leq \mu F_z$

가중함수: $F_y' = F_{y0}(\alpha) \cdot G_{y\kappa}(\alpha, \kappa) \rightarrow$ 종방향 슬립에 의한 횡력 감소

르노코리아 실무 Tip: MF-Tyre 피팅 시 복합 슬립 영역 데이터가 부족하면 $G$ 함수가 부정확합니다. 시험 매트릭스에 복합 조건을 반드시 포함하세요.

1차 지연 모델: Fy(s) = Fy,ss($\alpha$) · (1 − e−s/$\sigma$y)

일반적으로 $\sigma$y = 0.3~0.6m (승용차), 레이싱 타이어는 0.15~0.3m 수준입니다. 고속 레인체인지나 장애물 회피에서 이 지연이 핸들링 응답성에 큰 영향을 미칩니다.

Relaxation Length의 하중 의존성: $\sigma$y = C$\alpha$ / Clat (Clat: 접지면 횡강성). 하중이 증가하면 C$\alpha$와 Clat 모두 증가하지만 비율이 달라 $\sigma$y도 변합니다.

타이어 온도는 그립에 직접적 영향을 미칩니다. 최적 온도(60~100°C)에서 $\mu$가 최대이며, 이보다 낮거나 높으면 그립이 감소합니다.

$M_z = -t_p \cdot F_y + M_{zr} \rightarrow t_p$: 뉴매틱 트레일, $M_{zr}$: 잔류 토크

뉴매틱 트레일: $t_p = \frac{D_t\cos[C_t\arctan(B_t\alpha)]}{\cos\alpha} \rightarrow$ 슬립각 증가 시 트레일 감소

Thermal Model 구성: 타이어를 트레드 표면, 트레드 벌크, 카카스, 내부 공기의 4-Layer로 모델링합니다. 각 레이어 간 열전도와 외부 대류/복사를 고려합니다.

D 계수의 온도 보정: D(T) = Dref · (1 + κT · (T − Tref)²) 형태의 포물선 모델이 일반적입니다.

Tire Warm-up 시뮬레이션: 콜드 스타트 후 3~5 랩(서킷) 동안의 그립 변화를 예측하여, 아웃랩 전략이나 타이어 예열 프로토콜을 최적화합니다.

하중 감도는 차량 동역학에서 가장 중요한 타이어 특성 중 하나입니다. 코너링 시 하중 이동이 발생하면:

• 바깥쪽 타이어: 하중 증가 $\rightarrow$ Fy 증가하지만 $\mu$ 감소

• 안쪽 타이어: 하중 감소 $\rightarrow$ Fy 감소하지만 $\mu$ 증가

결과적으로 총 횡력 = Fy,out + Fy,in은 하중 이동이 없을 때보다 항상 작습니다. 이것이 Anti-Roll Bar로 롤 강성 배분을 조절하여 전/후축 밸런스를 제어하는 근거입니다.
