TROUBLESHOOTING
================================================================================

ISSUE: Text looks blurry
SOLUTION: Your browser may need zoom adjustment. Try Ctrl + 0 to reset.

ISSUE: Pronunciation audio doesn't work
SOLUTION: Try a different browser. Web Speech API support varies.
         Make sure you're clicking the pronunciation cards.

ISSUE: Colors look different
SOLUTION: This could be your monitor or browser color settings.
         The design is tested on standard displays.

ISSUE: Text seems cut off
SOLUTION: Your window might be too narrow. Try resizing or zooming out.

ISSUE: Checkboxes on Day 07 don't work
SOLUTION: Make sure JavaScript is enabled in your browser.
         Most browsers have it enabled by default.

ISSUE: Page is slow or laggy
SOLUTION: Try closing other browser tabs.
         These are lightweight files, so performance should be fine.
