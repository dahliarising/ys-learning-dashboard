## Notes for Learners

- **Consistency is key**: Spend 25 minutes daily (except Day 07: 30 min review)
- **Active learning**: Don't just read—use pronunciation, quizzes, dialogues
- **Testing effect**: Quiz yourself before looking at answers
- **Spaced repetition**: Return to earlier days 1-3 times as indicated
- **Context matters**: These lessons are designed for automotive engineers
- **Korean integration**: Korean translations provided for every expression
