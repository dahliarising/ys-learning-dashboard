🧠 뇌과학 학습 전략 — 포모도로 25분 배분

3가지 주기화 모델을 시각적으로 비교하여 각 모델의 특성을 대비 기억법으로 저장합니다.

0~5분개념 이해 — 핵심 구조 파악 5~18분핵심 내용 + 다이어그램 정독 18~23분실전 프로토콜 체크 23~25분퀴즈 3문제

📅 주기화 — 왜 필요한가?

CONCEPT 01

한스 셀리에의 일반 적응 증후군 (GAS)

모든 훈련 자극에 대해 신체는 3단계를 거칩니다: ① 경보(Alarm) — 근통증, 피로 ② 저항(Resistance) — 적응, 성장 ③ 탈진(Exhaustion) — 과훈련, 퍼포먼스 저하. 주기화는 탈진 전에 자극을 변화시켜 저항 단계를 최대화하는 전략입니다.

과훈련 vs 과도달(Overreaching)
기능적 과도달: 1~2주 고볼륨 → 디로드 → 초과보상 (계획적)
비기능적 과도달: 4~8주 이상 무계획 고볼륨 → 회복에 수개월
과훈련 증후군: 만성 피로, HRV 저하, 면역 저하 → 회복에 수개월~1년
30대 후반 + 육아: 회복 능력↓ → 주기화가 더 중요

🔄 주기화 모델 비교

MODEL 01

선형 주기화 (Linear Periodization)

가장 전통적. 볼륨↓ + 강도↑로 점진적 변화. 4~8주 단위.

Phase 1 (2~4주): 근비대 — 3~4×10~12 @RPE7
Phase 2 (2~4주): 근력 — 4~5×5~6 @RPE8
Phase 3 (1~2주): 피킹 — 3~5×2~3 @RPE9

MODEL 02

비선형(물결) 주기화 (Undulating)

일별 또는 주별로 강도와 볼륨을 변동. 적응 방지 + 다양한 자극.

월: 근비대 (3×10 @65%)
수: 근력 (5×5 @80%)
금: 파워/스피드 (5×3 @85%+)

장점 적응 지연, 다양한 자극, 크로스핏과 호환성↑

MODEL 03

블록 주기화 (Block Periodization)

2~4주 블록마다 하나의 주요 목표에 집중:

블록 1 (축적): 고볼륨·중강도 → 근비대 + 작업용량
블록 2 (전환): 중볼륨·고강도 → 근력 전환
블록 3 (실현): 저볼륨·최대강도 → 피킹/PR

30대 후반을 위한 추천 모델
추천: 주간 비선형(DUP) + 블록의 혼합
· 4주 축적 블록 내에서 DUP 적용 (주 3회 다른 자극)
· 4~6주마다 1주 디로드 (볼륨 50% 감량)
· 연간 2~3회 피킹 블록 (PR 도전)
· 크로스핏 WOD = 자체적 비선형 자극 → 웨이트는 기본 근력/근비대에 집중

📊 디로드 — 전략적 회복

RECOVERY

디로드 주의 과학

디로드 = 계획적 볼륨/강도 감소 → 피로 해소 + 적응 완성. 주기: 4~6주마다 1주. 30대 후반은 4주에 1회가 안전합니다.

디로드 방법볼륨 변화강도 변화적합 상황
볼륨 디로드−40~50%유지가장 일반적
강도 디로드유지−10~15%관절 피로 시
완전 디로드−50%−20%과훈련 의심 시

⏱ 5분 루틴

주간 볼륨 체크

각 근육군 주간 총 세트수 집계
지난주 대비 볼륨 변화율 확인
RPE 평균 > 8.5이면 → 다음 주 5~10% 감량
