REVIEW 02

당신의 프로필에 맞는 영양 전략

30대 후반, 크로스핏 + 러닝, 어린 자녀 2명, 맞벌이 — 이 프로필에 최적화된 전략:

🎯 목표: 에너지 넘치는 모습 + 근육질 체형

칼로리 TDEE 기준 유지~소폭 서플러스 (+200~300kcal) → 린매스 증가
단백질 2.0g/kg = 약 150g/day → 4끼에 37~38g씩 분배
탄수화물 운동일 5g/kg (375g), 휴식일 3g/kg (225g)
지방 1.0g/kg = 75g/day → 호르몬 유지 필수
수분 2.5L 기본 + 운동 시 500~800ml/h 추가

맞벌이 + 육아를 위한 현실적 식사 패턴
07:00 아침 — 오트밀 + 계란 3개 + 바나나 (탄40g + 단30g)
10:30 간식 — 그릭요거트 + 견과류 (단20g + 지방15g)
12:30 점심 — 회사 식당 (가장 큰 끼니, 밥 + 고기 + 채소)
16:00 운동 전 — 바나나 + 프로틴바 (탄30g + 단20g)
17:00~18:00 운동
18:30 운동 후 — 프로틴 쉐이크 + 고구마 (단40g + 탄40g)
19:30 저녁 — 가족 식사 (단백질 위주, 탄수화물 줄임)
* 주말은 가족 식사 중심, IF 탄력적 적용

🧪 자가 실험 설계 — N=1 영양 프로토콜

EXPERIMENT

2주 영양 실험 프레임

과학적 자가 실험의 핵심: 한 번에 하나의 변수만 변경

Week 1~2 (베이스라인): 현재 식사 패턴 유지, 매일 에너지 로그 기록
Week 3~4 (실험 A): 단백질 섭취량 2.0g/kg로 조정
Week 5~6 (실험 B): 탄수화물 주기화 적용
Week 7~8 (실험 C): 식사 타이밍 최적화 or IF 도입

측정 지표와 판단 기준
주관적 지표: 에너지 로그 (1~10), 수면 질, 운동 RPE
객관적 지표: 체중(주 1회 동일 조건), InBody(월 1회), HRV
판단 기준: 에너지 평균 +1점 이상 → 채택, 변화 없음 → 다음 실험
급격한 변화보다 2주 평균의 트렌드를 봅니다
INTEGRATION

Part 3 → Part 4 연결 다리

영양은 운동의 연료이자 회복의 재료입니다. Part 4(웨이트 트레이닝)에서 배울 내용과 영양은 다음과 같이 연결됩니다:

운동 전 글리코겐 충전 → 세트 수행 능력 직결
운동 후 단백질 + 탄수화물 → 근합성 + 글리코겐 재충전
수면 중 성장호르몬 + 충분한 영양 → 실제 근육 성장의 시간
주간 주기 탄수화물 주기화 = 훈련 주기화의 영양 파트너
